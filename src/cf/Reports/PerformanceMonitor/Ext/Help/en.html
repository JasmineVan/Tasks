The report is used to analyze application performance.

To open the report, in the Administration - Support and service - Reports and data processors - Administrator reports section, click Performance assessment by key operations.

Performance analysis

  • Specify the Period to assess performance. To do this, set the Start date and Period end date (time is required).
  • Specify the Measurement frequency to manage the number of results to be displayed within the period. The field is required. If an operation did not run during a period, the related table cell will contain zero.
  • In the Profile field, select a key operation profile from the list. The field is required.
  • You can select a key operation in the Key operation name field.
  • In the User field, you can select an application user.
  • If necessary, select the Executed with error check box and choose a filter value, for example, No (by default) to select time measurements by correct operations only. Analysis of the correct key operation execution is particularly relevant for operations related to posting documents. This allows you to separate the correct measurements of document posting from incorrect ones, for example, associated with filling errors. Click Generate. The report consists of two sections: a diagram and a table.

  • To view key operation data, double-click the key operation name in the table.
