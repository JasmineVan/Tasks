The report provides you with information on document records. It can be useful to analyze nonstandard accounting situations. The report is not intended for standard analysis or correction of document records.

To open the context report, in a document or document list, in the Reports submenu, click Document records. The command is available for all documents that can be posted. If the document has records but cannot be posted, it is not considered in the report.

Register list

Upon posting a document, information on its records is written to various registers, for example, accumulation registers or bookkeeping registers. By default, the report displays information from all registers.

  • In the report, register descriptions are hyperlinks. Click a hyperlink to go to a register and view document records in detail.
  • Use the Display only field in the report header to restrict the list of document registers. Click . In the list, select registers using check boxes, and then click OK. After selecting values, the check box beside the field is selected.

Configuring report

Click Settings.
Using check boxes, you can do the following:

  • Customize the report appearance:
    • Mark negative values in red.
    • Highlight accumulation record kinds in colors.
  • Change register selection.
You can switch to the advanced mode using the radio button in the Kind field and change other report settings.

Report options

Click to select one of the following report options:
  • Horizontally. This option is useful for viewing document records if there are not many columns. You can see all details on records in one line.
  • Vertically. This option allows you to arrange data more compactly, which makes it easier to view registers with a large number of columns. For visibility, lines are numbered. In the tooltip, you will see names of columns in the selected line.
Warning! In that case, other commands are missing as you cannot save your options for context reports. Note this upon changing option settings of the report. When you close the report, the application does not suggest that you save the new report option.
