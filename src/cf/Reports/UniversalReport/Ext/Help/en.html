Use this object to generate a report on various data types of the application.

To open it, in Administration - Print forms, reports, and data processors section, click Universal report.

The report can show data of documents, lists, tasks, business processes, charts of calculation types and their tabular sections, accumulation registers, information registers, calculation registers, accounting registers and their virtual tables.

This can be helpful when you need to get data and setting up a special report for these aims is unavailable for some reasons.

Main report settings are displayed in the header.

Select data for the report

  • Select data type from the list, for example, Accumulation register.
  • You may have to specify additional information to generate the report depending on the data type. For example, you must select Period for the accumulation register.
  • Select the data type from the list. For example, the accumulation register type - Demo: Invoice for payment turnovers.
  • Specify the data to be displayed. For example, for the Demo: Invoice for payment turnovers accumulation register you can include Invoice for payment turnovers or Turnovers in the report.

Generating report

  • Use Settings button to change report settings.
  • Then click Run report.
