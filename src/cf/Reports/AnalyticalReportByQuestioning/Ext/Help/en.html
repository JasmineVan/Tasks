This report is used to analyze how the respondents answer the questionnaires.

To open the report, click Analytical questionnaire report in the Survey section.

The report displays a variety of survey results: answers to simple questions and questions with aggregated indicators; answers to table questions; comparison of answers to the same question asked in different surveys.

For the purpose of basic answer analysis, a set of predefined report options is available. Custom report options can also be created if necessary.

To display the list of available report options, click More – Select report option:

  • View answers to simple questions. This report option displays the respondents' answers to the simple questions (as opposed to table questions). The report data can be grouped by survey, by respondent, or by question. Filters by survey, by respondent, or by question are also available.
  • View answers to table questions (flat view). This report option displays the answers to the table questions. The report data can be grouped by respondent or by question. Filters by survey, by respondent, or by question are also available. To view the respondent data, double-click the respondent. To view a question answered by the respondent, double-click the corresponding line.
  • View answers to table questions (table view). This report option displays the answers to the table questions in a slightly different form: the data is displayed in a separate table for each respondent.
  • Simple questions with aggregated indicators. This report option displays simple questions with numeric answers. For each question, the number of respondents that answered, min/max/average answer value, and total sum of the answer values are displayed. Filters by survey or by question are available.
  • Number of answers to simple questions. This report option displays the number of answers to simple questions. Filters by survey or by question are available.
  • Comparison of numbers of answers to simple questions in surveys. This report option displays the number of answers to simple questions broken down by survey (surveys in columns). Filters by survey or by question are available.
  • Table questions with aggregated indicators. If aggregated numeric answers are used for table questions, this report option displays the number of respondents that answered each question, min/max/average answer value, and total sum of the answer values. Filters by survey or by question are available.
  • Comparison of table questions with aggregated indicators in surveys. If aggregated answers are used for table questions in several surveys, this report option displays comparison of the table questions.
  • Number of answers to table questions. This report option displays the number of answers to all table questions. Filters by question or by other criteria are available.
