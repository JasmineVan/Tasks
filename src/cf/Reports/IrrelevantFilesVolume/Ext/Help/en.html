The report is designed to get information on the volume of data on the disk used by unnecessary files (obsolete, or added by mistake).

The number of files is shown for each owner—application object the files are linked to.

You can open the report from the File cleanup settings window where you can set criteria for file estimation and a storage period for each object.

Click Settings to use features of a finer records filter of the report.

Filtering by owner

  • Files can be filtered by owner, for example, by folder. To do so, double-click the owner in the report.
  • The report displays results of filtering by owner.
  • You can refine your filtering criteria by selecting different fields from the list.
