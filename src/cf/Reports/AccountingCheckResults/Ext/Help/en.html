This report is used to review and eliminate issues identified in documents, registers, and other kinds of data, and to review the accounting check results.

The report displays the following information:

  • Check kind. Information about the check, and the list of objects that have issues.
  • Summary column.

The check information is highlighted in color. This section includes:

  • Issue description and severity
  • Possible causes
  • Elimination recommendations
  • Solution. A check may include a hyperlink starting an automated troubleshooting wizard.

Next, the list of the objects with issues identified during the check is displayed. The list contains the following columns:

  • Information about the object with issues
  • Issue identified
  • Check date

Filter by employee responsible

  • Click Settings.
  • In the Settings window, select the Show employee responsible check box for Basic mode.
  • This enables grouping the checked objects by employee responsible in the report. The number of objects with issues is displayed for each employee responsible.

Viewing objects

  • To view an object, double-click it. While viewing an object, you can eliminate the issues the check identified.
