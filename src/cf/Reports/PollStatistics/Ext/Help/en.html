This report is used to display statistical data on the survey results.

To open the report, click Survey analysis in the Survey section.

To generate this report for a specific survey, select the survey from the list in the Survey field. The report contains summary information on respondents' responses that can be broken down by a variety of parameters.

  • Select the report in the Report kind field:
    • Response analysis
    • Respondent analysis

  • Click Generate.

Response analysis

  • For numerical responses, the following statistical information is available: minimum, maximum, and average values; total sum (marked with ). The total sum is only available for questions that have the Aggregate the sum in reports check box enabled.
  • The first number is the number of respondents that provided a response to the question.
  • The following statistical information is displayed for the question, separated by a slash: /Minimum/Maximum/Average.
  • Detailed information (which respondent provided which response to which question) is also available in the report.
    • To view the detailed information for a question, double-click the corresponding report item.
    • Select a Report option:
      • Grouping by respondent (default)
      • Grouping by responses

Respondent analysis

  • The Survey analysis report also provides the survey information broken down by the participating respondents. The report information is grouped by respondents that already filled or did not yet fill their questionnaires.
