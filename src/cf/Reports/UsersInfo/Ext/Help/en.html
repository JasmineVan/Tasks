This report is used to view information on all users and external users, including authorization settings ().

It can be helpful when you need to perform mass analysis of user authorization settings (authorization names, authentication types, etc).

Filtering users

  • Click to open. Click the corresponding check boxes to select users by specific criteria.
  • The following user filters are available:
    • Invalid;
    • To be deleted;
    • No matching infobase user;
    • Can sign in to application;
    • Standard authentication;
    • OS authentication;
    • OpenID authentication.

Report options

  • Click to open the list of available report options. Three report options are available:
    • —accessed from the Users list;
    • —accessed from the External users list;
    • —common report on all users, containing the field that can have the "Yes" value or remain blank and that is used for selecting or filtering external users.
