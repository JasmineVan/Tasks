The report provides statistics on stored applied object versions.

To open the report, execute the Number and size of stored object versions command in the versioning settings.

If the disk space for storing application data is limited, you can use the report to control and compare the size of stored object versions and reduce the number of stored versions.

The report displays the following information:

  • Size of stored versions by object type chart;
  • Number of stored versions by object types chart;
  • Number and size of stored object versions table - numeric data on the number and size of stored object versions. The report includes the following columns:
    • Object type - the applied object;
    • Number - number of object versions;
    • Data size (MB) - volume utilized by object versions.
