This form is designed to set rebuild aggregate usage parameters and receive optimal aggregates list.

  • Max relative size. This field displays maximum sum of aggregates size as a percentage of the records table. If the parameter is not specified or is equal to 0, aggregate size is not restricted.
  • Min rebuilding effect. This field displays the percentage by which the old network during the rebuild will be increased. If the new network increases the effect by Min rebuilding effect, %, the method actually rebuilds the network. If the parameter is not specified or is zero, there is no effect requirement. This parameter is not available for a list of optimal aggregates.
