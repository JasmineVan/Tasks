Full-text data search allows to search through contents of both card fields and attached files.

You can run full-text search directly from the Home page of the application.

Full-text search provides such features as:

  • Latinization (conversion of non-Latin characters to Latin),
  • Substitution (using Latin characters instead of identically-looking non-Latin letters),
  • And fuzzy search (finding strings approximately matching the search string).

To search data in the application, enter the word or its beginning. Click Find. You can search by one or several words. You can use such search operators as AND, OR, NOT.

You can enter data URL in the search input field. This will open the related data card instead of performing data search. You can receive an URL to any infobase object using the command in the each window title of the 1C:Enterprise 8.

Index update

In File mode infobase, full-text search index is not updated automatically. You need to refresh it manually on a regular basis. If the full-text search index is not relevant, i.e. there is data that is not yet indexed, in the infobase file mode the Update index button becomes available.

Examples

Search example Result
"Sales 00016422" Returns document "Goods and services sales" numbered 00016422.
"Order 120000" Returns transfer orders for cash amounts 120000 and 120000.50
"Johnson Acacia" Returns all documents prepared by Mr. Johnson for Acacia Ltd. counterparty
"Sales Acacia" Returns all documents "Goods and services sales" related to Acacia Ltd. counterparty
"Acacia 07/05" Returns all documents related to Acacia Ltd. prepared on July 5th
"Sneakers" It will return all mentions of goods like "sneakers" in the right order:
• White sneakers (Goods)
• Black sneakers (Goods)
• Goods and services receipt # 00086822
• Goods and services sales # 00016422

 

Recommendations

  • If your search returns too many results, specify the search query by adding to it any clarification (date, amount, name of the counterparty).
  • If your search returns too few results, expand your search by removing some data from the query.
  • To perform exact search, use:
  • Quotation marks to search the exact phrase, for example: "mini vacuum cleaner".
  • OR to search for any one of two words. Example: Acacia OR Uniconstructors search returns all references to Acacia Ltd. or Uniconstructors Inc. counterparties.
  • Brackets to group search terms. Example: Johnson (Acacia OR Uniconstructors) search returns all documents prepared by Mr. Johnson for Acacia Ltd. or Uniconstructors Inc. counterparty.
  • To search giving consideration to synonyms, use "!", for example, if you search "number!", the application will find all documents that contain words "number" or "figure".
  • Use hash sign (#) for fuzzy search. Example: Uniconstruct#3 search returns all documents containing words that differ from Uniconstruct by 3 or less letters.
  • Use an asterisk (*) to search by a word beginning. Example: Unico* search returns all documents containing words that start with Unico: Uniconstructors, Unicorn, and so on.
