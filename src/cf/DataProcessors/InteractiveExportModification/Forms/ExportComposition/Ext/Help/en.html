This report is used to access statistics on sent data content during data synchronization.

You can access the report via the Sent data content report link from the Interactive synchronization wizard. The report is generated automatically.

The report header includes the name of the application participating in data synchronization, as well as report generation date and time.

The report includes the following columns:

  • Data - list of applied objects (lists and documents) participating in data synchronization.
  • Selected:
    • By general rules;
    • Additional - selected on the Interactive synchronization wizard page;
  • Total number of objects in the infobase;
  • Will not be sent - number of items in the selected objects that will not be sent;
  • Will be sent - number of items in the selected objects that will be sent.

Total number of sent items is calculated in the report.

The sent data may also include additional objects required for data integrity purposes.

At the bottom of the list, more data is displayed:

Data synchronization rules - detailed description of synchronization rules set for this infobase;

Additional data added - the list of data added via a related Interactive synchronization wizard page.
