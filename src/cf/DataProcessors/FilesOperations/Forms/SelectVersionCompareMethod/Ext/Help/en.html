Selecting an application for comparing file versions. In the application, you can compare versions of files in the HTML, TXT, RTF, DOC, DOCX, and ODT format.

The form allows you to select a text editor to use for comparing file versions to each other. It opens when you click Show difference in the file version list if a default text editor for comparison is not specified.

Selecting a method for version comparison

  • In the Map file versions with drop-down list, select Microsoft Word or OpenOffice.org Writer.
  • Your selection will be saved in the application personal settings.

Selecting default editor

  • In the application personal settings (the File operations tab), in the Map file versions with field, select a text editor for version comparison: Microsoft Word or OpenOffice.org Writer. This setting is not set automatically by default.
