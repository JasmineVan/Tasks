In this list, you will see the files that you have locked for editing. You can edit files in the file list or in the list of attached files of application objects.

It opens when you click Files to edit.

The list displays:

  • File. Name of the file.
  • Placement. Folder in which the file or the object the file is attached to is located.
  • Size (KB).

Available commands:

  • View. Open the file for viewing.
  • End editing. Save the file version and make the file available to other users.
  • Open card. Open the information about the file.
