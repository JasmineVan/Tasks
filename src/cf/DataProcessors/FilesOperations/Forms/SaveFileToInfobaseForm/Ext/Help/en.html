This form is used to save a file version.

You will see the form after finishing file editing by clicking Save changes or Finish editing or when creating file versions in lists: Files, Attached files, Files to edit, list of locked files when closing the application, and Files in the main working directory.

Saving versions

  • You will see the name of a file being saved under the window header.
  • In the Comment to version field, you can type a short comment describing file changes you made.
  • If the Create new version check box is cleared, new version is written over the previous one when updating data in the storage. We recommend that you create versions to be able to revert a file to a previous version if the file is corrupted when editing. By default, the check box is selected. You can change this setting using the Store versions check box in the file card or when downloading files or folders.
  • For confirmation, click Save.

You can view the file version list using the command on the navigation panel of the file card.
