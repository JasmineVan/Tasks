The main working directory is a special directory on your computer in which the application places files you are viewing or editing. The list is designed to manage files in the working directory.

You can open the list from the personal application settings. On the File operations tab, click Main working directory setup - Open file list.

The list includes the following columns:

  • File is in use. The check box is selected if the file is being edited.
  • File. File format shown as an icon and name.
  • Size (KB).
  • Change date.

To add fields to the list, click More - Change form.

Available actions

  • To remove unnecessary files from the working directory, click Remove. Files in which you did not make any changes will be removed.
  • To save changes in the file and unlock it for other users, click End editing.
  • To release files you locked, click Cancel editing.
  • To view a folder that contains a file you are editing, click Open file directory.
  • To Open a file profile, choose this command in the context menu, or in the More menu.
