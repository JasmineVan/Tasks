In a built-in text file editor, you can view or edit a file, change its encoding, open a tree of processes and tasks in which the file is used, compare the file with the initial version during the editing, open it in the standard application if required.

To open it, in the Files list, click Open or Edit file in case you have the corresponding setting enabled in the application personal settings.

By default, in the built-in editor, you can open text files with the following extensions: TXT, INI, XML. You can add other extensions to the list in the application personal settings. On the File operations tab, in the File opening settings table, you can specify Extensions of files to edit in the built-in text editor (separated by spaces). You can also specify an Opening method: With standard application or With embedded editor (default value).

Editing files

  • If a file is opened for viewing, the Write and close and Write buttons are not available. To modify the file, click Edit.
  • The built-in text file editor works the same way as an external application. A file is copied to a local computer and marked as locked in the infobase. Until you click More - End editing, the created file version will not be available to other users.
  • After you finish editing the file, click More - End editing to remove the lock. Depending on the settings, the command also saves the file version and updates the file in the storage. By default, a new file version is created. To change the setting, clear the corresponding check box when you create a version.

Viewing file cards

  • To view information about the file, click More - Open card.

Comparing versions

  • To compare file versions, click More - Show difference. If an application for version comparison was not selected in the personal settings earlier (the File operations tab), you will be prompted to select it. You can select Microsoft Word or OpenOffice.org Writer.

Saving files

  • To save the file to the specified directory on the hard drive or a network drive, click More - Save as.

Opening files in a standard application

  • To edit the file in an external application (Windows notepad), click More - Open with a standard application.
  • If you want text files with specific extensions to open in an external application by default, make changes to the application personal settings. To do that:
    • In the application personal settings, go to the File operations tab.
    • Change File opening settings. In the Opening method column, select With standard application.

Changing encoding of text files

  • Text files can be saved in various encodings. The following encodings are typically used for files containing texts in Russian:
    • IBM866 (Cyrillic DOS).
    • KOI8-R (Cyrillic KOI8-R).
    • Windows-1251 (Cyrillic Windows)
  • If you see a set of meaningless symbols instead of the text when you open the text file, click More - Change encoding and select another encoding from the list.
