Viewing file version information

When you click Finish editing in a file list, the application recommends that you save the file version and specify information on this version. By default, this option is enabled. We recommend that you save versions as in case of fatal errors leading to file damage, you can revert the file to the previous uncorrupted version. You can edit a comment to the version and change its author if necessary.

File version information

  • Owner. A description of an owner file. It is filled in automatically when you add a version and it is not to be edited.
  • Description. A version description. It is filled in automatically when you add a version and it is not to be edited.
  • Parent version. To subordinate a version to another one, select the version from the list. For visibility, version history in the application is displayed as a tree.
  • Size (bytes). A file version size in bytes.
  • Extension. A file version extension and format. Next to this information, you can see a picture showing the file version format. The picture is defined by the application automatically.
  • Comment to version. Enter additional information on the file version. You can also enter it when you save the version after finishing file editing.
  • Author. A user who added the file version. You can select another author from the user list if necessary.
  • Creation data. A date and time when the file version was added to the infobase.
  • Version number. Numbers are assigned to versions automatically when you change a file. The latest saved version has the highest number.
  • Change date. A date when the file version was changed (local time).
  • Encoding. The By default value is selected automatically.

Viewing file versions

  • To open the selected file version in a viewer or editor, click View. You can also open the file version by double-clicking the row or using the command in the More actions menu.

Saving file versions to hard drive

  • To save a file version to a hard drive or another device, click Save as. Specify a directory where to save the file.
