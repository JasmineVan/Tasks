This form is used to select a file opening mode.

To open a file in a file list, do one of the following:

  • Press Enter. You will be prompted to select a file opening mode if no other settings are made.
  • Double-click the file. This option is similar to the previous one.
  • Click Open file.
  • Click Edit or press F4.

File opening options

  • Using the radio button, select a file opening mode:
    • Read-only. The file will be opened in read-only mode in an external viewer application. It will not be locked for editing.
    • Edit. The file will be opened in edit mode in an external editor. It will be locked for editing until you close it.
  • To confirm your selection, click Open file.

Disabling opening mode selection

  • If you do not want to select an opening mode in the future, select the Do not ask again check box after you select a file opening option.
  • If the In embedded editor opening option is set for text files in personal application settings, you do not need to select a file opening mode. The editor opens immediately, and you can switch between opening modes in the editor.
  • If the file was locked for editing, you do not need to select a file opening mode. The file opens in edit mode.
  • You can enable the mode selection in personal application settings, on the File operations tab.
  • In personal application settings (on the File operations tab), you can select an Action on file selection:
    • Open file (set by default).
      • Select the Select opening mode when choosing a file check box. By default, the check box is selected.
    • Open card of the file. In this case, upon pressing Enter or double-clicking the file, the file card opens. A file opening mode is not selected.
