This form allows you to import files from a disk on your computer.

To open it, right-click the file list or click More, select Import files, and then select one or several files in the Windows Explorer. In the Explorer, select one or several files, and then click Open.

You will see a list of files to import. You can change the entire list by clicking Select and selecting other files in the Windows Explorer. However, you cannot add new files to the existing list as old files will be removed once you select new ones.

Fill in the information required to import the files:

  • The application automatically fills in the Folder field with the folder in the file list to which the files will be imported. You can select another folder if necessary.
  • To save disk space, select the Delete files after adding check box (it is cleared by default).
  • Add Description for created files.
  • Select the Store versions check box (recommended). It is selected by default.
  • You can Select encoding from the list for text files.
  • To confirm, click Import files.
You can also use the drag-and-drop method to import files from the hard drive or a network drive. Use the mouse to drag the files from the Windows Explorer to the required folder in the application file list.
