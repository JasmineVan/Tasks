The application allows you to simultaneously import several selected files or folders with files from the hard drive on your computer. To do so, perform required actions in the file list.

If one or several files exceed the maximum allowed size, the application shows a warning message with a list of files that cannot be imported.

Click Continue import:

  • To import other files from the list with allowed size.
  • To get a report on errors with a list of files that cannot be imported.

Click Cancel to go back to the application without continuing further attempts. If you planned to import several files and other files met the conditions, they will not be imported to the application.

The maximum allowed file size can be increased in the application settings. To do so, in the Administration - File operation settings, specify the Maximum file size. The default value is 50 MB.
