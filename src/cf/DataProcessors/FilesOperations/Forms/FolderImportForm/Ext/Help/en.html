The form is designed to import folders from the hard drive on your computer.

You can open it from the file list by right-clicking Import folder in the context menu after you select a folder in the Windows Explorer. In the Explorer, specify a path to the folder, and then click Select folder.

Specify information required for folder import:

  • The Directory on hard drive field is filled in automatically. You can select another folder if necessary.
  • In the Placement field, you can see a folder you selected in the file list. It is inserted automatically. Folder from the hard drive on your computer will be imported here. You can select another folder if necessary.
  • To save disk space, select the Delete files after adding check box (it is cleared by default).
  • Add Description for created files.
  • Select the Store versions check box (recommended). It is selected by default.
  • You can Select encoding from the list for text files.
  • To confirm, click Import folder. If a folder selected for import is empty, you can see a message: "No files to add".
You can also use the drag-and-drop method to import a folder from the hard or network drive. All you should do is just drag a folder with the mouse from the Windows Explorer to the folder of the application file list.
