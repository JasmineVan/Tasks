This list allows you to manage application files.

The list displays:

  • Indicates that a file was signed with a digital signature or was encrypted.
  • Description. Full name and description of the file. An icon next to the name specifies file type.
  • Placement. Folder of the file list or an application object the file is attached to.
  • Author. User who added the file.
  • Creation date. Date and time the file was added to the infobase.
  • Size (KB). File size.
  • Extension. File extension.
  • Edited by. User currently editing the file.
  • Version. File version number.
  • Edited. User who added a file version.
  • Volume. Volume on the hard drive the files are stored in.
  • Path to file on volume.

Available commands:

  • View. Open the file for viewing in an external application.
  • Cancel editing. Make the file available for editing by other users.
  • You can Open card of the file by clicking the corresponding command in the More menu.
  • To save the file to the specified directory on the hard drive or a network drive, click More - Save as.
