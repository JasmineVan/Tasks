When you try to close the application, in this list, you will see the files that you have locked for editing.

If you do not want the list of locked files to appear when you close the application, clear the Show locked files on exit check box. You can also do that on the File operations tab of the application personal settings.

The list displays:

  • File format as an icon.
  • File name.
  • Description. Detailed information about the file (from the file card).

Available commands:

  • View. Open the file for editing in an external application. You can also open the file by double-clicking it or by pressing Enter.
  • End editing. Save the file version and make the file available to other users.
  • Open card. Open the information about the file.
