The application allows you to store files received from the scanner. To use a scanner in the application, you should configure required scan settings.

Before configuring the settings, connect a scanner to the computer. Scan parameters are available depending on the features of connected scanner.

Scanner name

  • In the Scanner field, select a scanner from the list of available devices connected to the computer. If there is only one scanner, it is selected by default.

Main

The parameters depend on features of the selected scanner connected to the application. Some of them might be unavailable.

  • Select scanner Resolution (DPI). A list of available resolutions depend on the selected scanner.
  • Select Chromaticity of images received from the scanner (Monochrome, Grayscale, or Color). It depends on scanner features.
  • Select one of the image Rotation options after scanning. This function may be unavailable for some scanner models.
  • Select Paper size that is placed into the document feeder. This function may be unavailable for some scanner models.
  • Select the Scan both sides check box. Both sides of paper will be scanned. This function may be unavailable for some scanner models.

Scanner dialog box

  • Show scanner dialog box before scanning. If the check box is selected, a dialog box with advanced scan settings appears. For example, you can select an area on the piece of paper for scanning. By default, the check box is selected.

Converting to PDF

  • Select the check box to Use ImageMagick to convert to PDF. Scanned images can be converted to PDF using ImageMagick. You should have a free software ImageMagick installed on the client computer with connected scanner. Use Win32 dynamic at 8 bits-per-pixel and Ghostscript.
  • Specify a Path to the application of conversion to PDF. This is a path to the executable ImageMagick file (by default it is a convert.exe). If this name conflicts with other applications, click Select, and then specify a full path as "C:\Program Files\ImageMagick-6.6.3-Q8\convert.exe".

For single-page images

  • Format. The application allows you to select one of supported file formats (BMP, JPG, GIF, PNG, TIF, and PDF) to add a file to the infobase. If the Use ImageMagick to convert to PDF check box is selected, PDF format is also available in the list. In this case, you can select PDF for storing, and other formats for scanning.
    • Quality. If you select JPG, you can configure its quality. 100 is the maximum quality and size, 1 is the minimum quality and size. The default value is 100.
    • Compression. You can set up compression for the TIF format. Possible options: LZW, RLE, CCITT3, CCITT4, and Without compression. Default: Without compression.
    • Type (for scanning). This parameter allows you to select one of the supported formats (BMP, JPG, GIF, PNG, and TIF) for images received from the scanner. The option is available if you select PDF for single-page image. In this case, PDF is used to store the file.

For multi-page images

The parameter becomes available for changing if the Use ImageMagick to convert to PDF check box is selected. TIF is selected by default.

  • Format. The application allows you to select one of supported file formats (TIF and PDF) to add a multi-page file to the infobase. It is used if you select the Accept all as one file option for several scanned images when using Scanning wizard.

Viewing the "Scanned file numbers" list

  • To view the necessary list, click Open scanned file numbers.

Note: if you use Windows Server 2008 operating system, install the Desktop Experience component to be able to use scanner.
