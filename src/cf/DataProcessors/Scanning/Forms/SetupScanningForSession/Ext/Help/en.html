The form is designed to set settings for the current scanning session. These settings are not saved. To open the form, click Settings in the Scanning wizard.

Depending on the scanner features and long-term scan settings, some parameters can be hidden. For example, if conversion to PDF is disabled, you cannot change the multi-page image format. TIF is set as default.

Main

Parameters become available if your scanner has matching device capabilities.

  • Resolution. Select scanning resolution (DPI). A list of available resolutions depend on the selected scanner.
  • Chromaticity. Select chromaticity of images received from the scanner (Monochrome, Grayscale, or Color).
  • Image rotation. Select one of the image rotation options after scanning. This function may be unavailable for some scanner models.
  • Paper size. Select paper size inserted in the document feeder tray. This function may be unavailable for some scanner models.
  • Scan both sides. Both sides of the document are scanned. This function may be unavailable for some scanner models.

For single-page images

  • Format. The application allows you to select one of supported file formats (BMP, JPG, GIF, PNG, TIF, and PDF) to add a file to the infobase.
    • Scanned image type. This parameter allows you to select one of the supported formats (BMP, JPG, GIF, PNG, and TIF) for images received from the scanner. The option is available if you select PDF for single-page image.
    • Quality. If you select JPG, you can configure its quality. 100 is the maximum quality and size, 1 is the minimum quality and size. The default value is 100.
    • Compression. You can set up compression for the TIF format. Possible options: LZW, RLE, CCITT3, CCITT4, and Without compression. Default: Without compression.

For multi-page images

The parameter becomes available for selection if conversion to PDF is enabled in the scan settings.

  • Format. The application allows you to select one of supported file formats (TIF or PDF) to add a file to the infobase. The parameter is applied if the Accept all as one file check box is selected.

Scanner dialog box

  • Show scanner dialog box. If the check box is selected, a dialog box with advanced scan settings appears before scanning. For example, you can select an area on the piece of paper for scanning.
