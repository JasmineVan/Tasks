Used for infobase backup configuring during program update.

It is recommended to create a infobase backup before starting the program update. The backup is used to automatically restore the infobase in case of emergency.

Use Create temporary IB backup and perform rollback in case of error from Search and install updates window.

To create backup select parameters:

  • Do not create backup;
  • Create temporary backup. In this case, you can check Restore infobase check box to be able to roll back if an error occurs during the program update;
  • Create backup and save in a specified directory. Specify the path to create a backup in a directory field, for example, C:\Backup.

Click OK to return to installing the update after setting the backup parameters.
