Use this window to clear user settings.

You can use this option when invalid settings impact the user experience. You can clear any settings, including personalized report options.

You can clear user settings in section Administration—Users and rights settings.

Clearing user settings

  • Under Clear settings for, select users:
    • Selected users. Click Select.
      • Select the Users or External users list.
      • In the Select users list, select users and click Select.
      • Once you select the users, the link text changes to a user name or the number of selected users.
    • All users. If you select this option, you can only clear All settings.
  • Under Clear the following, select the settings:
    • All settings. If you select multiple users or all users, only clearing all settings is available.
    • Selected settings. For a single user, you can select specific settings to clear.
      • Click Select.
      • In the Select user settings window, select settings.
      • Once you select the settings, the link text changes to a setting name or the number of selected settings.
  • Click Clear and close to clear the settings and close the window, or click Clear to clear the settings without closing the window.
