Use this window to select one or several users for copying settings to them or clearing their settings.

The list consists of two parts: the left section of the list displays groups, and the right section of the list displays the users or external users included in the groups. By default, the All users or All external users group is selected.

To view all users included in nested groups, select the Show nested group users check box at the bottom of the list.

Selecting users

  • Select or clear the check boxes. You can also:
    • Click to Select all users.
    • Click to Clear all users.
    • Select the users by clicking them while holding down Ctrl or Shift, then click More actions—Add check marks to selected users.
    • Select the previously selected users and click More actions—Remove check marks from selected users.
  • To copy or clear settings, click Select.

Viewing active users

  • Before you copy application layout settings to specific users, end all their sessions. Otherwise, the settings might not be applied correctly.
  • Click Active users to view the list of current user sessions.
