Use this window to select user settings that you want to copy or clear.

The list consists of three tabs:

  • Interface. Settings that affect the desktop layout, application windows layout, and the command interface.
  • Report settings. Settings that affect user-defined report options. The list is grouped by report options.
  • Other settings. Personal user settings, printing settings for spreadsheet documents, quick access settings for additional reports and data processors, the Favorites section settings, and other settings not listed on the other tabs.

Selecting settings

  • Select or clear the check boxes. You can also:
    • Click to Select all settings.
    • Click to Clear all settings.
  • Click Select to copy and clear settings.

Viewing settings

  • To view report and interface settings of the current user, double-click the user, or use the More actions—Open command.
