Use this list to manage user settings.

In this list you can view, clear, or copy report user settings, application appearance settings, and other settings.

Click in the section to open the list of settings.

  • In this case, the User field is displayed in the list. By default, the User field cannot be blank and is filled in by the value Administrator. If there are external users in the application, you can manage their settings.

To open the settings for every user/external user, you can click Settings on the navigation panel.

  • In this case, the User field will not be displayed.

To locate the specific setting, type the name or the part of the name in the Find field.

The list consists of three tabs:

  • Interface. Settings that affect the desktop layout, application windows layout, and the command interface.
  • Report settings. Settings that affect user-defined report options. The list is grouped by report options.
  • —user's personal settings, spreadsheet document printing settings, settings for quick access to additional reports and data processors, Favorites section settings, and other settings.

Copy settings

Select the user in the User field. You can use the following commands:

  • —this command copies the selected settings of the selected user and applies them to one or multiple users.

    • On the tabs of the list, click the settings you want to copy and press Ctrl and Shift;
    • In the list, select the check marks for the users you want to apply the settings to and click . The settings will be copied.
  • —operates in the same way as the previous command, but allows the following groups of settings:
    • Report settings.
    • Interface settings.
    • Report and interface settings.
    • —this command copies all user settings.
  • —this command copies user settings of the selected user to apply them to another user:
    • select user whose settings you want to copy;
    • select settings you want to copy. This action is similar to the command in the section.

Restrictions

Some of the user settings cannot be copied even if you select All settings.

  • When you copy the application appearance settings to any users, these users must terminate their application sessions. Otherwise, the settings might not be applied correctly. To check if the users are currently working with the application, click in the list.
  • Before copying report option settings, make sure that recipient users have sufficient rights for working with these reports.
  • Personalized report options cannot be copied between users.

If any selected settings are not copied, the application displays the warning. Click Show report to view the list of settings that were skipped.
