Use this window to copy settings between users. You can copy settings between users included in a single list: Users or External users.

It is convenient when you add new users to a group that already contains users.

You can copy user settings in section Administration—Users and rights settings.

Copying user settings

  • Under Copy settings from, select the source user or external user:
    • Click Select and select the Users or External users list.
    • In the Select user or Select external user list, select a source user or external user and click Select, or double-click the user.
  • Under Copy to, select destination users. Select one of the following:
    • Selected users. To select users, click Select.
      • In the Select source and destination users window, select users or external users.
      • Once you select the users, the link text changes to a user name or the number of selected users.
    • All users. If you select this option, proceed to the next step.
  • Under Copy the following, select the settings.
    • All settings. If you select this option, you can proceed to copying the settings.
    • Selected settings. If you select this option, to select the settings, click Select.
      • In the Select user settings window, select settings.
      • Once you select the users, the link text changes to a setting name or the number of selected settings.
  • Click Copy and close to copy the settings and close the window, or click Copy to copy the settings without closing the window.

Restrictions

  • Some of the user settings cannot be copied even if you select All settings.
  • When you copy the application appearance settings to any users, these users must terminate their application sessions. Otherwise, the settings might not be applied correctly.
  • Before copying report option settings, make sure that recipient users have sufficient rights for working with these reports.
  • Personalized report options cannot be copied between users.

If some of the settings are not copied, the application displays a warning message. Click Show report to view the settings that were not copied.
