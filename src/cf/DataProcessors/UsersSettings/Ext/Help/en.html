Use this window to manage user settings.

User settings management commands are located on the Users and rights settings panel of the Administration section.

Only administrators are allowed to manage user settings.

Available actions

  • Copy settings. Copy user settings between users. You can use this option to configure multiple new users in a group that already contains users.
  • Clear settings. Clear the settings of all users or the selected users. You can use this option when invalid settings impact the user experience.
  • User settings. Open the user settings management list.
