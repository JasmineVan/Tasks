The form is used to filter data during the interactive data synchronization.

Filtering data for export

  • The filter period is used for documents. In the For period field, click Select and specify the period. You can specify dates manually or select from the calendar. Click the Show standard periods link to quickly pick a period.
  • Click Add condition.
  • Select a Field for filtering.
  • Select a Comparison kind.
    • Equal, Not equal, In group, and Not in group comparison types allow selecting only one value. Use Equal or In group comparison types to filter only records that completely match the specified value.
    • In list, In group from list, Not in list, and Not in group from list comparison types allow selecting multiple values. If lists for value selection include groups, In group from list and Not in group from list comparison types are available.
    • Use In list or In group from list comparison types to filter items containing a value from the list.
    • Not equal, Not in group, Not in list, and Not in group from list comparison types negate relative comparison types. These comparison types filter all items except the specified ones.
    • Filled and Not filled comparison types do not require value selection. Use the Filled comparison type to filter documents that lack prices of goods.
  • Specify Value.
    • You can type the value or select it from a list depending on the comparison type.
    • To specify a document field of a list item as a value, click More - Use field as value. Select a field from the suggested list.

Grouping conditions

  • If necessary, you can set multiple filter conditions and group them with And, Or, and Not logical operations using the Group conditions button. Select conditions for grouping and click the button. All selected conditions will be added to the group.
  • To cancel condition grouping, select the conditions and execute the More - Ungroup command.
  • You can also create groups for conditions using the More - Add new group command and then drag the required conditions into the group or create new conditions in the group.
