Use this window to view properties of application events registered in the event log.

It displays the details of each application event (more detailed than event log data):

  • User. The full name of a user.
  • Date, time. The exact time of the event.
  • Application. For example, Thin client.
  • Computer. The network name of a computer.
  • Data group:
    • Metadata. An application object name and type, such as list or document type.
    • Data. An application object. For example, a specific list item or a specific document.

    • Presentation. An application object.

  • Event group:
    • Event. An event type.
    • Comment. A brief description of the event.
  • Session group:
    • Number. A session number.
    • Working server.
    • Primary IP port.
    • Secondary IP port.
  • Transaction group:
    • ID.
    • Transaction status. For instance, Committed.

You can hide some of the event log columns.
