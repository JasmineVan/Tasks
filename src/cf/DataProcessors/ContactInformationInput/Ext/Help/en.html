As soon as you have completed setting up the contact information kinds used by the company or organization, and loaded the address classifier, you are free to proceed to enter contact information into the application lists or documents.

You can enter contact information:

  • As free-format text lines
  • By filling in template fields.

Data input via templates ensures data uniformity, resulting in easier readability and navigation.

Using templates to enter phone and fax numbers also expedites data input and reduces the input error rate.

When entering contact information, you can have access both to the predefined fields and to the fields manually added to the Contact information kinds list during the application operation.

If the Allow multiple value entry check box is selected when you set up one or multiple contact information kinds, the Add button is available in the input window. Clicking this button will call up a list of contact information kinds you can add to the template. For example, you may need to add multiple fields for email addresses, webpages, etc. for a specific contact.

When editing contact information, you can use the following commands available via context menu accessed by right-clicking:

  • Select (F4). Enter contact information by template. It is identical to theSelect button. Templates can be used to enter addresses, phone numbers, or fax numbers.
  • Open (Ctrl+Shift+F4). View the entered contact information (for example, if an address is too long and cannot be fully displayed, use this command to read the full address).
  • Cut (Ctrl+X).
  • Copy (Ctrl+C).
  • Paste (Ctrl+V).
  • Clear (Shift+F4). This will also clear fields used for address or phone input by template. In the Address country field, the USA value is filled in automatically.
  • Select all (Ctrl+A).
  • Enter comment. Use this command to enter or edit comments without opening the field edit window. Comments are available for all contact information kinds, except the Other text field, as you can enter all the necessary information in this field. Use the command, then type a comment, and click OK to continue working in the application. If the comment field is not blank, the comment is displayed below the contact information field.
  • Address on Yandex.Maps and Address on Google Maps. These are commands that allow you to see an object address on Yandex or Google maps respectively. You can also find these commands in the More actions menu in the address entry card.
  • Change history. Use this command to view changes in contact information.
