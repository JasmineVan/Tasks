Use this window to enter phone (fax) numbers using a template.

You can access it from the tabular section of a document or list items containing contact information by clicking Select.

To enter a phone or fax number, fill in the following fields:

  • Country code. "+" character will be appended automatically.
  • Area code. This code will be bracketed automatically.
    • To enter new phone numbers quickly in the phone number input template, the values entered previously will be automatically filled in the fields with country and area codes. Additionally, for each area code, 10 previously entered codes are stored, which you can select from the drop-down list.
  • Phone number. There are no format restrictions.
  • Extension. "ext" will be added automatically before the extension digits.
  • Comment. Enter additional information about a phone here, if any.

Clearing phone (fax) numbers

  • To delete a wrong phone or fax number and type the correct number, click More actions – Clear phone number.
