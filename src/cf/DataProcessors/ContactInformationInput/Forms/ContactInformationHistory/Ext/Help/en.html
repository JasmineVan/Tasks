The form is designed to store history of contact information, for example, addresses.

You might need it, for example, if the legal address of the company was changed, and you want to print documents with correct legal address as of the document entry date.

Changing address

  • To backdate the date the address is valid from and the address itself, right-click Change history in the context menu, or in the More menu of the Address input window. The Address history window appears.
  • Select a line with address, and then click Change. After that, the Select button becomes available. Click this button to open an address entry template.
  • In the template window, you can see a message Address is still valid for valid addresses.
