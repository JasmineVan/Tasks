This form is used to input contact information according to a template. Address classifiers can be utilized to further streamline the procedure.

It is accessible by clicking Select from the Address field in the lists items.

Autocomletion of domestic address

  • Autocomletion of whole address is available when entering (only when using web service). Type in a search word or its part in the City, locality field.
  • Pay attention to the color difference of addresses:
    • By municipal division - standard color of text.
    • By administrative and territorial division - grey.

Enter address

  • In the Country field, select the address country. Addresses of the countries that are not members of Eurasian Economic Union (EEU) are edited in a common field, as they are not broken into standard subfields like "Street" or "Apartment". "USA" is the default country. This field can be inaccessible depending on the settings. To select a different country, click Select. You will be able to select a country from the Classification of countries of the world (OKSM) list.
  • In the Postal code field, enter the postal code. If the address can be found in the address classifier, postal code will be filled in automatically. You can also fill in an address based on the zip code, provided that the state address classifier is loaded.
  • The City, locality field is used for entering state, city, and so on. If necessary, you can enter detailed information of some part of the address, by clicking Select. In the opened template, enter data of locality.
  • The Street field is used for selecting (clink Select and find the required street) or entering address street.
  • The House, Block, Apartment fields allow you to enter the matching address parts. You can change these fields by selecting additional values from dropdown list.
  • You can also Add additional fields to enter other predefined address parts, such as Construction, Letter, Facility, Area, Office, Box, Wareroom, Room, P.O. box.
  • to enter house number, click , in the Select house window a list of all available houses and constructions of the entered address. Additional parts of address, for example, block, are also selected from the list.
  • The Address presentation filed is not available when filling the address by template. It will be filled automatically as you fill address fields.

Address input examples:

  • To enter "1515 Connecticut Ave, Office 202, Washington DC, 20008, USA":
    • Select "Washington, DC" in the City, locality field.
    • Select "Connecticut Ave" in the Street field.
    • Type in "1515" in the House field.
    • Block -
    • If you need to enter letter, Add field, select from the dropdown list the Letter value, and enter the letter in the field.
    • Instead of the Apartment field, select the Office field from the dropdown list and type in "202".
  • To enter an address with a different set of fields, for example, "45 East 52nd Street, Unit 2, Room 111, New York, NY, 10022, USA":
    • In the City, locality field select "New York".
    • In the Street field select "52nd Street".
    • Instead of the House field, select the Building field from the dropdown list and type in "45 East".
    • Block -
    • Click Add, select the Unit field from the dropdown list, and type in "2".

Address validation

Click Check filling to validate the address you have entered. The following criteria are validated:

  • the relevance of the FIAS version with an offer to update data;
  • address completeness;
  • absence of prohibited characters;
  • length of address attributes;
  • compliance with address classifier. In Client/server mode infobase, compliance of addresses with the address classifier is performed on the server side. When address validation is complete, the "Address is correct" message or list of errors will be displayed. Validation is not performed for foreign addresses outside of the U.S. Validation is not performed for free-format addresses.
  • If the address is not valid, the application searches for numbering options (not house, but building; he presence of blocks, units; letters in the house number).
  • The search is conducted in the historical data of an address, for example, a street was renamed.

Change history

  • In some lists, you can view the change history of the address using the More - Change history command.

Address on Yandex.Maps, Address on Google Maps

  • These commands allow you to view any address entered in the application on maps.

Selection of the type of territorial division

  • Selecting the check box before filling in the address, you can select:
    • Municipal division.
    • Administrative and territorial division.

Enter free-format address

  • Select the Address in free format check box of the More menu in order to knowingly enter an invalid address (for example, a supplier address on an incoming receipt is invalid, but you need to temporarily save it for printing, but you cannot use it later). The invalid address will be further used for storage, so this option is only used as a last resort.
  • Write the address. The standard address validation procedure will not be performed.
  • Click OK to confirm the address. It will be saved as a free-format address, to be viewed without any changes.

Filling address by postal code

  • If you know zip code for an address and have the address classifier for the related state loaded, you can find the address by using this zip code. To do it, execute the More - Fill by postal code command. This will display a list of addresses related to the postal code you have specified. Select the required address from list and fill the remaining fields manually.
  • If the zip code is specified, information about which is missing in the application, the application displays a message.

Clear address

  • You can clear an incorrectly entered address by using the command of the More actions - Clear address menu.

Import classifier

  • Execute the More - Import classifier command. Once the address classifier for the selected state has been updated, you can continue editing the address.
