This form displays the list of users currently working with the application.

The list includes users working with the infobase via thin client, web client, or Designer mode, as well as such technical users as COM connection, WS connection, background job, etc.

To open the list, click the relevant link in Administration - Support and Maintenance.

The list displays the following:

  • User. Username of the user. External users are displayed as well.
  • Application. Application type, for example, Thin client. Connection mode is specified as well. Sessions created in other connection modes (COM connection, web services, etc) are displayed too.
  • Session started. Session start date and time.
  • Computer. Network name of the computer used to sign in to the application.
  • Session. Session number, data separation.

Total number of active users is displayed at the bottom line of the list.

The line corresponding to the current session used to open the active users list is marked in bold.

For your convenience, you can sort the list by any column.

Filters

  • Use the Show field to select a filter by a predetermined value from the list. All sessions is selected by default.

Viewing user data

  • To view user info, double-click the required line.
  • You can also select a line and click More actions - Open.

Viewing event log

  • To review work history for a selected user, click Event log to go to the event log filtered by the selected user.

Refreshing lists

  • The active users list is not updated in real time. To view active users that were added to the list after it was opened, click Refresh current list or press F5.

Terminating sessions (Client/server mode)

  • In Client/server mode, the Terminate session command is available. Use this command to forcibly terminate the selected user sessions.
  • Prior to using this command, set the application administration parameters (customization window opens automatically).
