Allows you to import data from a file into documents and lists.

Importing data from a file into documents and lists

You can open it from documents and list items by clicking Import from file.

By clicking Import data from file, administrators can import data into an arbitrary list of the application.

Select how data will be imported using the button:

  • Filling a table in the application
  • Import from external file

Table filling

  • Click Table filling (enabled by default) to navigate to entering data into a table and later import it into a document directly in the application.
  • To fill in the table, you can copy data to the table from an external file via clipboard.
  • There are required columns in the table. They are listed above the table.
  • Click Edit template to open a list of table fields for input and use the check boxes to change its structure. Note that the fields in bold in the table are required and cannot be changed.
    • Select the required columns using the check boxes. You can set names of the columns, their display, and order to simplify import.
  • After filling in the table, click Next to go to data mapping.

Import from external file

  • By clicking From external file, you can import data from a previously prepared file saved on your computer, or save a template and fill in the data later.
  • To save a table for entering data in a file of the selected format, click Save template. Note that there are required columns in this file. Specify a path to the file on your computer. The application assigns a name to the file by default.
  • You can fill in external files to import them to the documents. Fill in the file according to the recommended rules (the required fields are highlighted in bold in the table).
  • To import data from an external file, click Import table from file. Specify a path to the file on your computer.

Data mapping

  • After importing the data, the application maps data automatically. If data cannot be mapped automatically, data for mapping is displayed automatically.
  • To view mapped or unmapped data, use a filter using the Show field. Unmapped data is highlighted in red. You can Resolve conflict using the corresponding button.
  • To manually map the data, open the corresponding application list, find the appropriate value in it or enter a new one, then click Select.
  • Optional fields can also be mapped in addition to the required ones. When importing, you can map the field and include it in the document using the Select from list button. If the item you need is not in the list, you can Create it by clicking the relevant button.
  • Once the data is mapped either automatically or manually, click Import data. After that, the data will be imported into a document or a list. You will only have to fill in the missing fields.

Data import to arbitrary list of the application (for administrator only)

  • The administrator can import data from file to an arbitrary list using the relevant command.
  • In this case, the administrator can select one of the application lists to import data. You can import contact information, additional attributes, and information to the lists.
    • After selecting a list to import data, click Next. Use the buttons to choose how the data will be imported:
      • The application offers a table for filling.
      • Also you can import data from an external file.

Mapping data (for administrator only)

  • After filling in the table or importing data from file, click Next. You need to map data.
  • You can set up data mapping:
    • Using the radio button, you can choose what to do with Mapped items: Update or Skip (by default).
    • Using the radio button, you can choose what to do with Unmapped items: Create (by default) or Skip.
  • The data will be mapped automatically, if no match is found, a table is displayed prompting you to map the data.
    • You can automatically create unmapped data in some lists. Such data is labeled as <New>. To automatically create data in the list, click Create using the radio button (selected by default).
    • Once the data is mapped either automatically or manually, click Import data.
  • After that, the data will be imported into a document or a list. You will only have to fill in the missing fields. Data import results are displayed.
    • Using the Show field, you can display All, New, Updated, and Skipped items.
    • You can Save to file and correct several selected lines, and then import them again.
    • You can change the entered items in the list if you need to. To change several selected items, click Change attributes.
    • Select the Open catalog by clicking Finish check box if necessary.
  • If all data in the table is correct, click Finish to confirm. If the check box is selected, a list with the entered items opens.
