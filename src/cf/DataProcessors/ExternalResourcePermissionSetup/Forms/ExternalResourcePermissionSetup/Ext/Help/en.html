If the 1C:Enterprise server cluster is using for this infobase the security profiles that limit access to resources exterior to it, for the correct functioning of the application you need to edit the list of permissions in the security profile.

The list displays external resources permitted to be used.

To confirm, click Apply changes.

Moreover, you need to ensure that:

  • Access to the external resources is not blocked by any software installed on the server (such as antiviruses, traffic control utilities, and so on).
  • All necessary rights were provided to operating system users on behalf of which the working servers of the 1C:Enterprise cluster are started.
