Administration section offers two tools for altering work parameters, performing scheduled jobs and controlling application state.

To provide for efficient work of the application, you need to set it correctly.

Frequently used settings can be added to a command interface of the Administration section. Depending on a setting type, you can do this via the navigation panel or the actions panel settings.

See also:

  • Service subsection offers the administrator various means of maintaining operability of the application;
  • Common settings subsection contains general settings of the application;
  • User rights settings subsection contains commands related to access management and user rights;
  • Internet support and services subsection offers commands related to internet support of the application, import of classifiers, and application version update;
  • Agenda subsection contains commands for setting email, notes, notifications, and business-processes;
  • Data sync subsection is used for setting data synchronization with other applications and managing multi-user use of the distributed infobase;
  • File management subsection offers settings for import, files storage, and file processing.
  • Print forms, reports, and data processors subsection allows to set print forms, chose between report presentation options, and set additional reports and data processors.
