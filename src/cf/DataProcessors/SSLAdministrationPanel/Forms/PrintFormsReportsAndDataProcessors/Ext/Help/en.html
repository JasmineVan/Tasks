In this panel you can set up print forms, report options, report distribution, additional reports and data processors. The following commands are available:

  • Click Print form templates to go to the list of templates to customize print forms generated from the application.
  • Use the "Print" submenu to set the Print submenu commands.
  • Reports. Go to the list of report options to set their accessibility and placement in application sections.
  • Report bulk email. Go to the list of report mailings to generate reports automatically and deliver generated reports to recipients.
  • Universal report is designed to receive reports on selected application data (on lists, documents, registers, etc.)
  • Extensions. Enable extensions of sections, reports and data processors in addition to those built in the application.
  • Add-ins. Go to the add-ins list. Using this application, the administrator can control the list of add-ins that are used at workplaces. In the list the administrator can connect add-ins from a file on the computer or update them.
  • Additional reports and data processors. Enables additional reports and data processors, and after that the following command becomes available:
  • Additional reports and data processors. Go to the connection of external printed forms, reports and data processors in addition to those built in the program.
