This panel is used to set up work with online user support services, to check for available updates, etc.

Online support settings

  • Username is a name saved in the application and used to sign in to the online user support system. To enter online user support system, use the same username and password as the ones you use for portal.1c.ru and portal.1c.eu. If you do not have a username and a password, register or contact a 1C partner you have an ITS contract with.
  • Connect - enter a username and a password to access all online support services. The username and password entered will be stored in the application.
  • Disconnect - delete a username and a password used to access online support services from the application.
  • Contact tech support - go to 1C:ITS Portal to send a message to technical support service. Technical details of the application will be added to your message automatically.
  • 1C:ITS Portal monitor - information about available support for the application, the application version, etc.

    Monitoring center

    This subsection contains settings for collection, analysis, and storage of technological information and anonymized statistics of configuration use. Reports on application use are sent to developers automatically to help them identify what issues must be addressed before the others. Collected info is depersonalized and contains only statistics. Preparation and sending of reports does not slow down application operation.

    Use radio button to select:

  • Allow automatic sending of application usage data to 1C
    		•  (by default this option is selected; it is recommended to keep the radio button in this position).
  • Monitoring center settings
    		•  follow this link to allow sending reports on errors and set if you want the application to ask the user for permission to send the report each time there is an error.
  • Allow sending data to a third-party application
    		•  enter the Service address.
  • Prohibit information sending
    		•  it is not recommended to select this option since statistics is important for developers to be able to analyze application operation; sending statistics to developers does not slow down ongoing operations in the application.
