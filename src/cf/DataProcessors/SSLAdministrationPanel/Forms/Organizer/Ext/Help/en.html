Use this panel to set up email, notes, reminders and business processes.

Mail

In this application you can keep records of interactions with external contacts as well as between application users.

  • Email client. Select this check box to use email client, built in the application, for interactions using emails. You cannot use all types of interactions without the email client. After you enable it, the following commands become available for setting up interactions:
    • Outgoing email in HTML format. Select this check box to edit outgoing emails in HTML format.
    • Calls, SMS, meetings and scheduled interactions.Select this check box to register calls, SMS, meetings and scheduled interactions in the application.
    • The Reviewed and Under Control flags. Select this check box to mark the consideration of incoming emails, meetings, phone calls and scheduled interactions, as well as to control the outgoing email and SMS message. Use this check box to organize collaboration in the application.
    • Prohibit to display unsafe emails content. Enable the check box not to display unsafe items in case of their presence in email. Incoming email can contain items that allow the sender to monitor operations with the email secretly, particularly, viewing time and some device parameters where the email is read. Unsafe items include images imported from the Internet or default scripts. Such items are disabled automatically when viewing an email.
  • Configure system account. Click here to configure the main email account to send notifications from the application.
  • Email accounts. Administration of all email accounts.

Send text messages

  • Configure SMS sending. Click here to configure parameters of sending SMS from the application.

Notes, reminders, questionnaires, message templates

  • Notes. Select this check box to use notes. They are arbitrary textual information for which it is impossible to pick up any more suitable catalog or document, and which should not be available to anyone else. Notes can be arbitrary or on a subject, referring, for example, to a catalog item, document, or other applied object.
  • Reminders. Select this check box to enter reminders that activate on the specified time. Enter the time and subject, about which the program will display a list of reminders at the specified time. Reminders can be arbitrary or on a subject, referring, for example, to a catalog item, document, or other applied object.
  • Polls. Select this check box to activate the corresponding section of the application, in which you can prepare surveys, create polls, conduct surveys among respondents, as well as analyze survey results.

  • Message templates. Clock this link to work with message template list. Using the application you can develop message templates for email and SMS messages, their storage and management in the list of templates. You can send letters and SMS messages generated on the base of lists or documents and by prepared message templates.

Business processes and tasks.

The application provides opportunities for solving tasks of enterprise or company management automation. In the application you can picture the process of enterprise management as a set of business processes consisting of tasks, each of them assigned to a specific performer.

  • Business processes and tasks. Select this check box to activate the Business processes and tasks section of the application, in which you can use starting business processes and work with tasks. A number of other commands also becomes available, using which you can start business processes and use the following tasks:
    • Roles and task assignees. Role settings to address tasks to assignee groups.
    • Subordinate business processes and tasks. Select this check box to keep records of business processes which you can create within other business process tasks.
    • Change started business processes. Select this check box to make changes to tasks of started business processes.
    • Date of task start. Select this check box to allow users to fill in the Start date field and sort your task list by it.
    • Date and time in task deadlines. Select this check box to enter task deadlines accurate to the minute.
    • Notify of overdue tasks by email. Select this check box to notify performers of overdue tasks according to schedule. After that the
      • Configure schedule command becomes available. Execute the command to configure schedule of notification bulk email.
    • Notify assignees of new tasks by email. Execute the command to configure schedule of notification bulk email. After that the
      • Configure schedule command becomes available. Execute the command to configure schedule of notification bulk email.
