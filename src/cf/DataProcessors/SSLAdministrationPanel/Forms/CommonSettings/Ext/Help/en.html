In this panel you can configure the main parameters of the application.

  • Application title. Enter the text that will be displayed in the window title of the application.
  • Time zone is the time zone where records are kept in the application. All dates of documents and other application data are stored in this time zone.
  • Current session time. Using this command, you can find out how much time of the client differs from the server time with correction for time zone.

Infobase publishing

  • The Online address and Local address application settings are designed to generate links to infobase objects depending on a type of user connection to the infobase.
  • For example, if you include such address into bulk email, then by clicking it is possible to go to the object form in the application itself.

Server infobase parameters

  • Internet access parameters are settings of proxy server parameters at the 1C:Enterprise server.
  • Use of security profiles. Use security profiles to control the use of external resources (in relation to the 1C:Enterprise server cluster) by the application.

Contact information

  • Contact information kinds. Add and set contact information kinds of different applied objects, for example partners, counterparties, employees, etc.

Additional attributes and information

  • Additional attributes and info. Select this check box to use additional properties for catalogs and documents.
  • Attributes and info with common value list. Select this check box to allow adding attributes that have common value list with other attributes (the Add - From other set - Back up by example command).
  • Common attributes and info. Select this check box to allow adding attributes to several applied objects simultaneously (the Add - From other set - Make the attribute common and add to a set command).
  • Additional attributes. Set up additional attributes for catalogs and documents.
  • Additional info. Set up additional info for catalogs and documents.

Change history

The application allows to store list change history, to view any version of given object, and to compare two versions of an object.

  • Store change history. Select this check box to store the application change history. After it the following command becomes available:
  • Storage settings. Configure storage and clear versions of documents and catalogs.

Data search

  • Full-text data search. Select this check box and the following command will become available:
  • Set. Setting full-text search and maintaining a full-text search index up to date.

Digital signature and encryption

  • Digital signature. Select this check box to allow usage of digital signatures and verify the absence of changes to the file since the digital signature was generated.
  • Encryption. Select this check box to allow file encryption.
  • Settings of digital signature and encryption. Click here to set parameters of cryptographic tools attachment: digital signature and encryption.
