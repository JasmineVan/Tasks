Bulk attribute editing

Bulk editing of attributes can be performed in two cases:

  • by executing the Edit selected command in most lists of the program.
  • only for an administrator: by executing the Bulk attribute editing command in the Administration - Service - Data editing program section or by executing the Additional processing - Bulk attribute editing command in case of external data processing.

Editing selected list items 

Editing values of attributes, as well as additional attributes and data, for selected items of the list.

Entering new attribute values for selected list items

  • The fields of selected list items available for editing are displayed in the Attribute column. Can edit several attributes at once.
  • To edit values of attributes, additional attributes, and data, fill in the New value in the rows. For some attributes, you can use formula editor. To open it, click .
  • Double-click the row to edit it. You can:
    • enter a value to search in the list;
    • execute the Show all command (F8 button) and select the value from the list;
    • execute the Create command to enter a new value to a list;
  • When entering a new value, the check mark is being selected automatically.
  • Click Edit attributes.

Clearing values of rows for the selected items in the list

  • To clear the attribute value, select the check box in the row and do not fill in the New value column.
  • Click Edit attributes.

Editing contains errors

  • The list of unedited items of the list is displayed in a separate window. For each item, there is a comment to the occurred error.
  • Click Open to view items that failed to be edited.
  • Click Retry editing to retry editing the items of the list with the same parameters.
  • To correct errors, you may need to edit values of attributes of the listed items.

Selecting previously saved settings for attribute changing

  • Execute the Previously edited attributes command,
  • Select the setting from the drop-down list. The list stores 20 most recent settings of attribute changes.

Changing blocked attributes

  • The list may contain attributes that are uneditable by default. Such attributes are displayed in gray.
  • Uneditable attributes are "key": editing them can affect other lists and documents of the program. For this reason they are locked.
  • When trying to enter new values, the program offers to unlock them. In case there is indisputable need for editing them, confirm your choice by clicking Yes.

Aspects of editing posted documents

  • Once the previously posted document is edited, it needs to be reposted.

Setting attribute editing parameters

  • The program uses default settings to edit attributes. Click Additional parameters to set the settings.

Bulk attribute editing (for administrators)

  • Administrators can edit attributes of all catalogs and documents in the program, including tabular sections. You can also select the items more precisely and edit their attributes (Selecting items to edit table).

Selecting items to edit

  • In the Change field, select the document or the application catalog to change elements in. By default, you can edit All items of a catalog and all documents of a selected kind.
  • To view the items selected for editing, click All items.
  • Use the Add selection filter button to select one or more items possessing a certain characteristic. While using a filter, changes are being made only in the selected items.
  • Once a filter is set, the link displays how many items are selected for editing.
  • Once a catalog or a document kind is selected, you can view attributes available for editing, including the service ones, in the Attributes available for editing table. To show service attributes, click Additional parameters and select the check box.

Enter new values

  • Once the attributes are selected, click the Enter new values button and execute standard commands to edit them.

Execute custom algorithm

  • This option is only available when using a program locally on a pc. It is unavailable while working in an online application.
  • You can use an arbitrary algorithm to edit records. Yoou may need it in the following cases:
    • Correcting irrelevant data in part on flexible condition;
    • Insufficient standard features for mass data transformation;
    • Editing or deleting other objects with consideration for current object's data.
  • Once the attributes are selected, click Execute an arbitrary algorithm.
  • In the Algorithm field, enter the name of the algorithm. The algorithm will be saved.
  • The left part of the window displays the algorithm code, and the right part – the attributes of the object being changed.
  • To enter the algorithm code quickly, simply drag attributes of the object from the attributes list to the algorithm field.
  • Use the Algorithms button to view the list of algorithms previously used for an object and use them. The program stores 20 latest executed algorithms for every object.
  • Use the Object is recorded radio button to set the way the object is to be recorded:
    • Automatically;
    • In the algorithm code - in this case the algorithm must contain the row that enables recording of an object.
  • Use the Execution mode radio button to select the Safe or Unsafe execution mode. Unsafe execution mode is unavailable in a basic version.
  • To start an algorithm, click Execute.
  • If an algorithm contains an error, you can learn about it in the execution report.
