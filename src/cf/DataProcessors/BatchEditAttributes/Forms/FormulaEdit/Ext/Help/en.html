Edit formula

The form allows the user to quickly create formulas using items from the Operators and functions panel. To open the form, select the attributes to be edited and click .

Compose the formula and click Save and close.

The form consists of three areas:

  • Available attributes to select the attributes to be used in the formula;
  • Operators and functions to select the operators and functions to be used in the formula. Operators and functions are grouped by kinds for convenience;
  • The formula itself in the bottom area.

Check formula

  • After composing the formula, click Check formula.

See also:

  • Bulk attribute editing.

Tip. To check whether the formula is valid, click Check formula.

Tip. The Operators and functions list contains all operators and functions that can be used in the formula.
