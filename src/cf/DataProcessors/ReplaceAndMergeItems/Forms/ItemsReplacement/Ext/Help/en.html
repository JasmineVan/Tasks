This form is used to replace one or several items with the selected item.

This option is most useful when you delete items. To delete an item permanently, you must delete all references to it.

To open the form, click Replace and delete in the application lists. You can also click Replace everywhere where found in the Deletion of marked objects window.

You can automatically replace references to the items being deleted with references to the selected item. All the specified items will be marked for deletion or deleted if the user has administrative rights.

All items of the list are displayed, including those marked for deletion.

Viewing list items

  • In the context menu, or in the More menu, click Change to view the selected item.

Operations with documents

  • Filter by period is available for documents. If there are a lot of documents in the list, you can use this filter. To do this, click Set period in the context menu, or in the More menu.
  • More - Post and Cancel posting commands are also available for documents.

Replace items

  • Select an item with which you want to replace the selected items.
  • Infobar at the bottom of the list displays attributes of the item used for replacement. If you have administrative rights, you will be provided with an additional option. Click the link, and then specify that the items to be replaced are:
    • marked for deletion.
    • deleted permanently.
  • Click Replace to replace all references.

If any errors are encountered during the replacement procedure, a list of items that failed to be processed will be displayed. At the bottom of the list, you can see a description of failure causes for the selected item. In the list, you can perform the following actions:

  • Open item for viewing by clicking Change, or double-clicking the item.
  • Retry item replacement by clicking Retry replacement.
  • Click Back, and then select another item for replacement.
  • Click Close to go back to the application.
