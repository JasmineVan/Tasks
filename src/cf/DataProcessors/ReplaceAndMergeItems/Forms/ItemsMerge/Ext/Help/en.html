This form is used to find and merge duplicate data.

The form can be accessed from application lists by clicking More - Merge selected. You can merge several items by selecting them.

A list of items to be merged displays:

  • Description.
  • Code.
  • Usage location. This field displays a number of occurrences for the item, or Not used message.

Merge items

  • The first item in the list is marked with icon for merging by default.
  • In the list, select a "correct" item that you want to keep using in the application, and then click Set as original. After that, this item will be marked with the icon as a merge target.
  • At the bottom of the list, you can see a number of items to be merged and the item that remains in the application. All other items will be marked for deletion. This is a reference that can be changed by administrator.
  • Click Merge.
  • After merge, only the target item remains in the application. All other selected items will be replaced with the target item and marked for deletion. User with administrative rights can permanently delete these items at once.
  • If errors occurred during processing items, you can see a list of unprocessed items with description of failure causes. For example, merge will fail if the items belong to different list groups with different accounting.

  • To continue merging, fix the cause of the failure by changing items in the list.

Changing item composition in the list

  • If the list does not contain the items you want, you can Add new item. In the list, you can Select an existing item, or add a new one.
  • If the list contains more than 2 items, you can Delete unnecessary items from the merge list via the More command.

Viewing items

  • Select an item, and then click Open.

Viewing references to list items

  • To view the Usage locations report for all list items, click the All usage locations link at the bottom of the list.

Configuring item deletion (for administrators only)

  • If you have administrative rights, you can permanently delete the items marked for deletion by changing the reference at the bottom of the list from marked for deletion to deleted permanently.
