The form is designed to transfer files stored in the infobase to the file storage volumes.

To transfer files, do the following:

  • In the application settings, in the Administration - File operation settings section, select the Store files in volumes on the hard disk check box.
  • In the File storage volumes list, add a number of volumes to store files of the infobase.

Check the following information:

  • Number of files in the infobase.
  • Size of files in the infobase (MB).

Based on this data, you can decide how many volumes you need to store files.
