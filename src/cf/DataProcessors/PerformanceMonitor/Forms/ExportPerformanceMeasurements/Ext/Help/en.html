The form is used to configure manual export of performance measurements for analyzing them in other applications.

The application runs performance tests. Performance measurements are stored in the application and can be exported to other applications for analysis.

Use buttons to fill in the Start date and End date of performance measurement export.

In the Profile field, select a key operation profile.

In the Export directory field, select a path to save data.

Specify the Archive name to save measurements.
