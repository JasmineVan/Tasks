Use this window to filter response time threshold of a key operation automatically on the basis of subjective user assessment and existing data about key operation execution time.

To open it, click Response time threshold selection from the Key operations list.

You need to get user assessment of key operation work quality, for example, poor or unacceptable.

Then you need to map quality assessment with APDEX value according to the table.

APDEX scale

Value

Estimation

from

to

0.00 0.50 unacceptable
0.50 0.70 poor
0.70 0.85 frustrated
0.85 0.94 good
0.94 1.00 excellent


Specify the Period for which measurements will be obtained for further analysis, the Key operation for which the response time threshold is being searched, and the Current APDEX according to the user assessment. When choosing an APDEX value from the table, it is recommended to shoose minimum suitable value.

After you click Select response time threshold, the Response time threshold field will display estimated response time threshold. The Estimated APDEX field will display APDEX value that is obtained with such response time threshold.

The application monitors performance, which requires setting a response time threshold for the key operation. Results are output to the report.
