The application features standard tools for performance data collection and analysis in accordance with Apdex (Application Performance Index, international standard for software performance evaluation).

Before you can proceed with performance evaluation, you need to fill in the Key operations list.

Software performance evaluation based on Apdex methodology includes the following major stages:

  • filling in key operation profiles (in the profile, specify the priority and target time value for each key operation predefined in the application);
  • gathering information on execution time for each key operation;
  • generating the Performance assessment by key operations report based on the collected data.

To view the Performance assessment by key operations report, in the Administration - Support and service - Reports and processors - Administrator reports, click the relevant link.

To enable performance evaluation, select the Performance assessment check box. Then the application will take performance measurements while running specified key operations, for example, posting a document.

  • Measurement results will be processed in the Performance monitor report.

Performance monitor

Export performance measurement data

  • The measured performance data can be exported to external files in background. This can be useful, for example, when you need to run performance analysis for an external application.
  • To configure data export, in the Administration - Support and service - Performance monitor section, click Settings.
  • The generated external files, as well as files with application settings used for performance testing, are compressed to ZIP archive automatically.
  • To export performance measurement data manually, in the Administration - Support and service - Performance monitor section, click Measurement export. You can open or save a file with performance measurement data. To save the file, specify the path to the local directory.

Import performance measurements

  • You can analyze other application performance measurement data using import from a file.
  • To import a file, in the Administration - Support and service - Performance monitor section, click Measurement import.
  • In the Import file, use the button to specify the path to the local file and click the Import button.
