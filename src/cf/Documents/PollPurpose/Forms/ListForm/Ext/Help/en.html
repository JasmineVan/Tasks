This list is used to assign surveys to respondents.

Before the respondent can start filling the questionnaire according to the template, you need to perform several preliminary actions. First, prepare a survey, define beginning and end dates for it, select respondents, and specify a questionnaire template to be used for generating the questionnaire for the respondents.

To open it, click Survey assignment in the Survey section.

The list includes the following columns:

  • Number. The number is assigned automatically.
  • Date. Date and time the Survey assignment document was created.
  • Description. Full description of the document.
  • Questionnaire template. Description of the questionnaire template to be used for the survey.
  • Start date. Beginning date of the survey.
  • End date. Ending date of the survey.

Assigning a new survey

  • Select Create and fill in the required fields.

Creating a new survey

  • To create a new survey with the specified parameters, click More – Post. This allows the respondents to fill questionnaires for the survey.

Cancel posting

  • To cancel a survey, click More – Cancel posting. After that, the questionnaire for this survey will not be available for the respondents.

Filter by period

  • Click More actions - Set date interval to set a required interval for viewing documents.
