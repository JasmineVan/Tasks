This document is used to assign a survey to respondents.

Before the respondent can start filling the questionnaire according to the template, you need to perform several preliminary actions. First, prepare a survey, define beginning and end dates for it, select respondents, and specify a questionnaire template to be used for generating the questionnaire for the respondents.

Assigning a survey

Fill in the following fields:

  • Number. Assigned automatically.
  • Date. Date and time of document creation. Assigned automatically but can be edited.
  • Description. The name of the questionnaire as it will be displayed in the respondents' questionnaire lists. The field is required.
  • Specify the beginning and ending dates of the survey period in Valid from... to... fields. These fields are optional. If the survey period is not specified, the questionnaire is displayed in the list of available questionnaires until the respondent provides answers to its questions.
  • Questionnaire template. A questionnaire template to be used for the survey (selected from the Questionnaire templates list). The field is required.
  • Choose the Respondent type. Respondents are external users related to specific application objects, such as partner list. In the Respondent type field, specify the type of external users to participate in the survey. To do this, choose one of the application lists.
  • Then, add external users to participate in the survey from the selected list to the Respondents list.
  • If you want all users of the selected type (for example, all partners) to participate in the survey, select the Free-form survey check box. In this event, the Respondents list is not editable.
  • Type in a Comment if you want to provide additional information about the survey.
  • If you want the questionnaire to be displayed in the questionnaire archive list, select the Show in questionnaire archive check box. This list displays questionnaires the respondent filled earlier. To view the questionnaire archive list, the respondent needs to execute the Questionnaire archive command.
  • If you want to allow the respondents to fill the questionnaire in stages, select the Presaving check box. When this check box is selected, the respondents can Save a partially filled questionnaire. The respondents can resume filling the questionnaire whenever convenient. The questionnaire is not considered to be completed until the respondent clicks Complete and close. If the Presaving check box is not selected, Save command is not available and the respondent must fill the questionnaire in one stage.
  • Click Post (or Post and close) to create a new survey with the specified parameters. This allows the respondents to fill questionnaires for the survey.
