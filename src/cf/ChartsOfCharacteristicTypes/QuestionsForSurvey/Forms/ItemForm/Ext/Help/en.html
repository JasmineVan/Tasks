Use this window to enter a question.

An important step in preparing questionnaires is question formulation.

This question is called an elementary question because it can be used as an element of more complex structures.

Entering a question

  • In the Description field, enter a short question presentation to be displayed on picking questions to questionnaire templates.
  • To place a question in one of the groups, select a group from the list in the Question group field.
  • In the Wording field, you can write a detailed question text to be displayed in the relevant field of the questionnaire template. The text is automatically inserted from the Description field by default.
  • The content and forms of respondent answers depend on the Answer type field value, which can be one of the following values:
    • String. In this case, the respondent needs to type an answer as a text string.
      • In the String length field, set a limit on the number of string characters. The field is required.
    • Text. The respondent enters an answer in the text field up to 1,024 characters. A long answer can be entered in a separate window.
    • Number. When answering this question type, the respondent enters a number.
      • In the Length field, specify a quantity of number signs. The field is required.
      • In the Accuracy field, set a number of decimals. Its default value is 0.
      • You can specify Min value and Max value of the number. If the respondent enters a number that is outside the limit specified in the Min value and Max value fields, the warning about entering incorrect data appears.
      • Select the Aggregate the sum in reports check box so that the Survey analysis report includes the total sum for all questionnaires having the answer to this question filled.
    • Date. When answering this question type, the respondent enters a date.
    • Boolean. The respondent can select one of the following values: Yes or No.
    • Select one response from the provided options. With this answer type, you can ask the respondent to select one of the response options using the radio buttons. The author can specify possible response options in the list.
      • Using the Appearance field, you can determine what response option selection will look like in the questionnaire template:
        • Radio button
        • Toggle switch.
    • Select several responses from the provided options. The respondent can choose any number of response options using the check boxes. The author can specify possible response options in the list.
      • If the Requires open ended response check box is selected for a response option in the list, the respondent will be prompted to give a detailed response in the text field under the response options, hence commenting on their choice.
    • In addition to the listed response types, it is also possible to select one of the application lists as a response type. In this case, the respondent will select an item of the application list as an answer to a question.
  • Selecting the Ask a respondent to comment on the response and explain the comment check box during question preparation will give the respondent an opportunity to clarify their response in the text field next to the response when filling out the questionnaire.
    • If the check box is selected, you need to enter a text in the field below to be displayed before the text field of respondent explanation. The field is required.
    • The Ask a respondent to comment on the response and explain the comment check box is unavailable if you select the following response types: Text and Select several responses from the provided options. A detailed response can be given in a response of the Text type. For a response of the Select several responses from the provided options type, the field with the comment can be displayed for each response option if needed.
