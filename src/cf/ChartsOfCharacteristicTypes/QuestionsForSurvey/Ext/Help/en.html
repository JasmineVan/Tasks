Preparing questionnaire questions.

To open the list, click Questions for questionnaire in the Survey section.

An important step in preparing questionnaires is question formulation.

In the general list, there are questions that you can add to any questionnaire. These questions are called elementary questions because they can be used as elements of more complex structures.

The list includes the following columns:

  • Code, which is populated automatically.
  • Description, which is a brief description of the question.
  • Wording, which is a full wording of the question.

Entering a new question

  • Select Create and fill in the required fields.

Grouping questions

  • You can group questions in the list, for example, by questionnaire subjects.
  • Click Create group and enter Group description. You can subordinate this group to one of the existing groups. For this, select a group from the list in the Parent field.
  • Enter the required quantity of groups. Drag questions to matching groups or enter questions in the selected group.
