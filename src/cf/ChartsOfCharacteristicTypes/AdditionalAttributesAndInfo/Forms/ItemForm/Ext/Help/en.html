This form is used to configure Additional attributes and info.

To open it, click Add or Create in the Additional attributes or Additional info list.

Assigning new additional attributes or info to catalogs or documents

  • Depending on settings, one of the following commands for entering new attributes or info is available:
    • Select a set matching this catalog or document, and then click Create. By default, an additional attribute or info can be added only to one set.
    • Select a set matching this catalog or document, and then click Add - New. It is similar to the Create command. Using this command, an additional attribute or info can be added only to one set.

Fill in the following required fields:

  • Specify a short Description of a property that will be displayed in application object windows.
  • To prevent the additional attribute from being blank when filling catalogs or documents, select the Required check box. This check box is not available for additional info.
  • Fill in the fields on Main tab:
    • Using the check box, select the Value type of an additional attribute or info.
    • To specify format for additional attributes of the Number, Date, or Boolean types, go to Format string wizard by clicking Default format (not available for additional info).
      • To change the default settings, set a value in the Language (Country) field.
      • Select parameters for setting formats.
      • Made parameter changes are displayed in the Example line.
      • To confirm the selected parameters, click OK.
      • After the format is specified, the link changes to Format is set.
      • For more information on formats, see below.
    • To place links to external resources or other application objects (for example, files) on the object (document, list) card, select the Display as a hyperlink check box. It is available for additional string attributes and reference objects.
    • Fill in the Tooltip field to explain how to fill in a specific additional attribute or info. Text entered in this field will be displayed as a popup upon hovering the mouse pointer over the additional attribute or info field.
    • For additional attributes only: Click the link in the Visible field to customize additional attribute visibility. Additional attribute visibility can depend on a condition (for example, on filling of another additional attribute). After the condition is set, it is displayed in the link. The always value is displayed by default.
    • For additional attributes only: Click the link in the Available field to set up additional attribute availability for filling. Additional attribute visibility can depend on a condition (for example, on filling of another additional attribute). After the condition is set, it is displayed in the link. The always value is displayed by default.
    • For additional attributes only: To prevent the additional attribute from being blank when filling lists or documents, select the Required check box. It can depend on a condition (for example, on filling of another additional attribute). To set a condition, click the link. The link has the always value by default.
  • To display an additional attribute or info description in the bottom part of the selection list upon picking an additional attribute or info to sets, fill in the Comment field. Fill in this field on the Comment tab.

Adding additional attributes or info from another set

  • In the application, you can use previously entered additional attributes and info for other sets.
  • This method is most useful for additional attributes and info of the Additional value and Additional value (hierarchy) types. If appropriate settings are made (at least one of the check boxes in the Administration - Common settings section is selected), the Add button becomes available.
  • Select an additional attribute or info set, click Add, and in the drop-down list, click From other set.
  • Wizard of adding an additional attribute or info opens.
    • In the left part of the first wizard page, you can see the list of previously entered attributes or info.
    • In the right part, values and other parameters of the selected attribute are displayed.
  • Select an additional attribute or info on the first wizard page. Note that you cannot create an additional attribute or info in this list.
    • Click Next to choose an option of adding an additional attribute or info.
    • You can also add an attribute from one set to another one by dragging it. If appropriate settings are made, the second page of the Add additional attribute or info wizard opens.
  • On the second wizard page, using the radio button, choose an option of adding an additional attribute or info:
    • Copy attribute to create the same additional attribute or info in the set.
    • Copy attribute by sample to create an additional attribute or info with a common value list. The Attributes and info with common value list check box in the Administration - Common settings section is to be selected.
    • Add common attribute to set to make an additional attribute or info common. In this case, not the value list becomes common but the attribute itself. The Common attributes and info check box in the Administration - Common settings section is to be selected.

Copying additional attributes or info

  • In the application, you can copy additional attributes or info.
  • In this case, a copy is created for a set of additional attributes or info. This copy is an independent additional attribute or info with its own value list, and it is not linked with the original based on which it is created.
  • Depending on settings, there are several options:
    • In the wizard of adding new additional attributes and info.
      • Select a set of additional attributes in the list, and then click Add - From other set.
      • On the first wizard page, select an attribute to copy in the attribute list, and then click Next.
      • On the second wizard page, using the radio button, select Copy attribute, and then click Next.
      • Fill in the rest of fields. After you copy the attribute, the value list is available for editing.
    • By dragging.
      • You can drag an additional attribute or info from one set to another one.
      • If a list contains the Add button, the second page of the Add additional attribute or info wizard opens.
      • If a list contains only the Create button, the card of the additional attribute or info to copy opens. Fill in the rest of fields.
    • By clicking Copy and Paste.
      • Select an additional attribute or info, and then click Copy. Select a set to which you want to add this additional attribute or info. Click Paste. If appropriate settings are made, the second page of the Add additional attribute or info wizard opens. If the selected additional attribute or info is already included in a set, the application suggests that you make it common.
      • The Copy and Paste commands are also available in the context menu, which opens by right-clicking an attribute, or in the More actions menu.

Adding additional attributes or info by sample

  • To be able to use this option, select the Attributes and info with common value list check box in Administration - Common settings.
  • You can use previously entered attributes and info from other sets and add attributes with a common value list for several attributes of the same type (for example, the Door color attribute by sample of the Color attribute). In this case, a value list of the sample attribute is used. It is convenient to configure a value list for several attributes of the same type at once.
  • Select a set from the list.
  • Click Add - From other set.
  • The list of all additional attributes or info already existing in the application opens. Select an additional attribute or info from the list of ones already existing in the application. Click Next.
  • Using the radio button, select Copy attribute by sample, and then click Next.
  • Type and all values of the selected additional attribute or info will be copied to the new item.
  • Fill in all required fields. By default, the attribute becomes common to the source set. In this case, you cannot enter new values. You will see a message informing you that the value list is common to another attribute.
  • After the additional attribute or info is saved, using the link, you can view the sample attribute and add values if necessary.
  • You can make an attribute created by sample that has a value list common to another attribute independent.
    • Click Change. The Change additional attribute or info settings window opens. In this window, click Attribute with its own value list and save your changes. The value list will be copied and links in documents will be replaced. After that, you will be able to enter the value list.
    • Warning: Use this conversion for already existing attributes (used for entering in documents) with caution. Attribute conversion is irreversible. The document update procedure can be performed automatically, but this is time-consuming and error-prone. We recommend that you modify the existing documents by manually deleting the old reference values and adding the new ones instead.

Adding common additional attributes or info

  • A common attribute is included in multiple sets. This option is suitable when not only a value list but also the attribute itself is to be common. In lists and reports on catalog and document items, common attributes are grouped and shown as a single attribute in one column or field. For example, using common attribute Color, you can filter all yellow clothes and shoes.
  • To be able to use this option, select the Additional attributes and info check box in Administration - Common settings.
  • To add a common additional attribute or info to a set, select the set, and then click Add - From other set.
  • Select a common additional attribute or information from the suggested list. Select Next.
  • The second wizard page opens. Using the radio button, select Make attribute common and add it to set, and then click Next. The common additional attribute opens.
  • You can also add a common additional attribute or info by dragging it or by clicking Copy and Paste.
  • To add a common attribute to multiple sets, repeat these actions for each set.
  • When a common additional attribute (common additional data) is included in one or more sets, the Common attribute (common data) is included in sets link will be displayed at the bottom of its window with the list of sets.
  • To convert a common attribute to an independent attribute with its own value list, click Change. You will find the button next to the link.
    • The Change additional attribute or info settings window opens. In this window, click Attribute with its own value list and save your changes. The value list will be copied and links in documents will be replaced.
    • Warning! Reconfiguration of the additional attribute (data) is irreversible and may take an excessive amount of time depending on the number of documents where the attribute is used. We recommend that you create a new attribute with a specific value list and perform a transition between these attributes manually (by clearing old attribute values and filling in new values).

Additional attribute or info value types

Additional attribute or info value types vary depending on usage logic. The following additional attribute or info value types are available:

  • String. Determine the string Length, select the Fixed or Variable option (if the entered string is less than the expected length, the number of characters reserved for this value will be automatically reduced to the actual). In this case, the maximum string length is 1024 characters. If very long strings are expected, select the Unlimited check box.
    • You can make the field with multiple lines for attributes of the Line type. To use the multiline input field, select Multiline input field check box and specify number of lines.
  • Select Length and Precision of the number. The default number length is 15, and default precision is 3. These values may vary depending on the input data. Keep in mind that number Length includes both integer and fractional parts, as well as the decimal point. Select Positive only check box if necessary.
    • When entering additional attributes of the Number type, you can specify the number format more precisely.
      • The entered numbers will be rounded to conform to values of Length and Precision fields in Format line wizard.
      • You can select Shift.
      • You can change the Decimal separator (it is a comma by default), and specify the Group separator (it is space by default) and the Grouping.
      • You can select Presentation of zero, for example, to remind the users that they should fill in the field.
      • You can also select Presentation of negative numbers.
      • You can select Display leading zeroes check box, if necessary. Then, if the actual number length is less than the specified Length, the remaining leading digits will be filled in with zeroes.
    • Warning! Some format settings (for example, blank number presentation, long date and so on) are not applied automatically in many places and are supported only where the developer has made it possible (for example, in printed forms).
      • They are inapplicable in object cards.
  • Date. Select date components: Date, Time or Date and Time.
    • To specify date format for additional attributes, click Default format link. You can select dd.mm.yyyy or yyyy-MM-dd formats as well as the Local date format (for example, 22 May 2013), set the Blank date presentation (a blank field will display a warning, for example, Date is missing).
      • Warning! Some format settings (for example, blank number presentation, long date and so on) are not applied automatically in many places and are supported only where the developer has made it possible (for example, in printed forms).
      • They are inapplicable in object cards.
  • Composite data type. This value type allows to finally select data type when entering data into a catalog Select Composite data type check box, then select check boxes for all data types allowed for the attribute.
    • For example, user may be unaware of a specific date, such as the date of company registration. In this case, users should be allowed to choose between Date and String data types. If the exact date is known, then it will be possible to select the Date type and enter, if it is unknown, then you can enter the date in the form of the line, for example "Beginning of the 19th century.".
    • Each data type included in the composite data type can be customized as usual, including date format, string parameters, and more.
    • When entering an additional attribute value of composite data type, you have to specify the data type first.
  • Boolean. Usually these are values like True or False, Yes or No.
    • You can modify the format of these values in Format line wizard for additional attributes. You can specify Representation of boolean False and Representation of boolean True. In this case, the users will be prompted to choose between values specified by the user, instead of generic Yes and No.
      • Warning! Selection of such additional attributes with the Equal and Not equal conditions is not supported in lists and reports.
  • Additional value data type is frequently used, and the input of information is reduced to the selection of values from the list. This is the default data type. Both standard and grouped value lists are used. Additional attributes (data) of Additional value type can be used to describe any properties of applied objects.
  • Additional value (hierarchy) structure is widely used. It is also called item hierarchy. Items of such list have identical properties, but they can have subordinate items. The hierarchy can be very complex.
  • User. In the application you can use values of the Users list (and other lists) as additional attributes (data).

Special features of additional data, such as Additional value or Additional value (hierarchy)

  • Besides standard fields, the following fields are available:
    • To Use weight coefficients for values, select the check box. This field can later be used for analysis in reports. You need to think through the usage logic if this field yourself. This field is optional.
    • Value form title. Enter the item input windows (hierarchical item input windows) header.
    • Value selection form title. Enter the value selection windows (hierarchical value selection windows) header. We recommend that you specify personalized titles for additional attribute (data) input and value selection windows, thus facilitating user input.
  • After selecting data type and filling in Name field, click Save. Then, value list will become available. To enter values, click Create. Enter the required fields.
  • If you did not save the entered data before clicking Create, a warning will be displayed. To proceed with entering values, click Save.

Changing format of additional attributes and data

  • After saving, no further changes can be made to value types or format of additional attributes and data. Instead, you will have to delete the existing additional attributes or data, and then create them again, with proper value types.
