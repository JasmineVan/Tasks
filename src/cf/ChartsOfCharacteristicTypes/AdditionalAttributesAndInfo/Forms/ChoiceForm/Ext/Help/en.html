The list is used to select additional attributes or info.

Opens from the Additional attributes or Additional info list by clicking Add - By sample or Add - Common.

Adding additional attributes or info by sample

You can use previously entered attributes and info from other sets and add attributes with a common value list for several attributes of the same type at once (for example, the Door color attribute by sample of the Color attribute). In this case, a value list of the sample attribute is used. It is convenient to configure a value list for several attributes of the same type at once.

  • Select an already existing additional attribute or info.
  • Click Select
  • You can also double-click the item.
  • Type and all values of the selected additional attribute or info will be copied to the new item.

Adding common additional attributes or info

A common attribute is included in multiple sets. This option is suitable when not only a value list but also the attribute itself is to be common. In lists and reports on catalog and document items, common attributes are grouped and shown as a single attribute in one column or field. For example, using common attribute Color, you can filter all yellow clothes and shoes.

  • Select an already existing additional attribute or info. If the attribute you need is not available in the list, click Create. Fill in the required fields.
  • Click Select
  • You can also double-click the item.
