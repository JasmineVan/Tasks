This setting is used to specify properties of an additional attribute depending on values of other attributes.

This setting is unavailable for common additional attributes.

To enter new conditions, click Add condition.

The table for entering conditions consists of the following columns:

  • Select an Attribute from the list by clicking .
  • Select a Comparison kind.
  • Select a Value from the list or enter it (depending on the selected attribute, the way of specifying a value can vary).

If the condition is not suitable, remove it from the table by clicking Delete condition.
