Additional attributes and info are used in property sets.
All additional attributes and info of objects can be displayed in reports and lists.

When choosing an additional property, follow the guidelines below:

  • Additional attributes are an integral part of an object with properties. They are stored in the tabular section of the object and are available to users who have an access to the object with properties. Additional attributes are edited in the object window similarly to other attributes.
  • Additional info is used to store object info that is not an integral part of the object. Generally, additional object info is available for viewing and editing to users who have no access to the object with properties itself. Additional info is edited in a dedicated window accessible from the window of the object with properties.

Upon choosing the Additional value or Additional value (hierarchy) value type, you can set a list of possible values or values with hierarchy of an additional attribute or info. To do this, click Create in the window of the additional attribute or info.
