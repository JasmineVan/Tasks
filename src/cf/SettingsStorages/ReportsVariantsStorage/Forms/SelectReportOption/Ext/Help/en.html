Use this object to select report options.

To open it, in the report option form, click More - Other - Select option.

You can select the following:

  • standard predefined report options provided by the application. Standard options are general by default and are listed with the icon ;
  • general report options shared by other users. They are listed with the icon . You can choose options of all authors;
  • personal report options that were saved earlier. Report options created by the current user are highlighted in color.

The list may contain three groups of report options:

  • report options in use (not a separate group, located at the beginning of the list);
  • Hidden in report panels - this group includes predefined general reports with disabled Visible in report panels by default check box;
  • Marked for deletion. The group is always at the bottom of the list. If you select an option that is marked for deletion, the application will display a warning.

The list includes the following columns:

  • Name - full name of the report option displayed in lists.
  • Picture Available to all users:
    • is visible - the report option is available to all users,
    • is not visible - the report option is available only to the author.
  • Author - full name of the user who is the author of the report option.
  • At the bottom of the list, you can see Description, which contains detailed information on the selected report option.

Select report options

  • To open a report option, select it in the list and click Select.
  • If you select a report option that is marked for deletion, the application will display a warning.
  • You can also double-click the item.

Filter

  • If the report option list is large, you can filter it by the author to find quickly the required report option.
    • In the Author field, select the report option author from the user list.

Configure report options

  • Select a report option, click More - Change, and edit desired fields.

Delete report option

  • To delete a user report option, in the More menu, click Mark for deletion. You can also use the Del key.
  • To Remove deletion mark, execute this command again.
  • This feature is unavailable for predefined report options.

Note for administrator or person responsible for reports:

  • Personal report options of other authors are usually hidden from users. However, you can rarely need to convert a personal report option into a public one or assign it to another user in this list.
  • To see all report options without exceptions, in the More menu, select the Show other authors' personal report options check box.
