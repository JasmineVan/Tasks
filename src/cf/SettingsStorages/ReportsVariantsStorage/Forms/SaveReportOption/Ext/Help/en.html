Use this object to save a report option.

When a report is configured and generated, you can save it in a new report option. To do that, click the Report options - Save report option button. You can also use a similar command in the More menu.

Existing options list

  • Upon saving a report option, the list of all existing options of this report is displayed (both predefined and custom ones).
    • The list includes:
      • standard predefined report options provided by the application. These report options are common by default;
      • common report options shared by other users;
      • personal report options that were saved earlier.
    • The list also includes report options that are marked for deletion.
  • In the list, you can Edit existing report options by right-clicking the matching command of the shortcut menu. For a predefined report option, you can change description or placement settings. For a custom option, you can also change the name (provided that you have the access rights).
  • To delete an existing custom report option, mark it and select Del or right-click the Mark for deletion command of the shortcut menu. Predefined report options cannot be deleted.

Report option name

  • To save a new report option, enter the Name that is not used for another option of this report. The application checks whether the report option name is unique when saving the option.
  • If you overwrite an option, you can save settings in the current option: by default, a report option with the executed Save option command is selected. You can select any other personal report option from the list to overwrite it.
    • For report options that cannot be overwritten, including predefined options, the application adds to the name the "copy" word by default. Such report options are displayed in gray in the list of existing options.
    • If a replaced report option was marked for deletion, the application will show warning. Click Yes to confirm.
    • The application administrators or persons responsible for reports can overwrite common report options of other authors.

Availability

  • An administrator of a report option can restrict the report option's availability by switching the Available radio button to the Author only position.

Reset settings

  • To reset custom report options, select the Reset settings check box.
  • If the check box is selected and the report is available to All users, the settings will be updated for all users of the report option. It also means that filter values specified by other users will be reset.

Save report options

You can save a report option in two ways:

  • Specify main parameters and click Save. Default settings will be used. The new report option will be placed to the same section of the report list as the source option.
  • Click Next to change settings of the report option placement.

Visibility in report panels

  • If you select the All users availability, you can specify visibility for other users.
  • If you want to make a report option visible in report panels of all users, select the Visible in report panels by default check box.

Report option layout in panels

  • To place a report option to the application sections or groups, in the Section, group column select one or more check boxes. When placing to a section, the report option will be displayed on the top of the section report panel and will not be placed to groups.
  • In the Importance column, select a method to highlight the report option in a group:
    • Important - the report option will be marked in bold.
    • See also - the report option will be displayed in a separate group at the bottom of the report panel.
  • Enter Description that will be displayed under the report name in the report panel and in report lists
  • Click Save to save a report option with selected settings.
