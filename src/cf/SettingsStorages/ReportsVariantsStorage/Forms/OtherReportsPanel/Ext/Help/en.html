The panel is used to go to similar report options.

To open the panel, in submenu Report options, click Other reports.

The list displays similar report options as references if there are any in the application.

The list of other report options of this section opens up. To go to the required report option, click its link.

Close this window after going to report check box is selected by default not to interrupt user sessions.
