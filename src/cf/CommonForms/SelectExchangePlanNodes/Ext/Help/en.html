Use this list to select exchange plan nodes (infobases).

Use the Select command or the Add command to access this list from other application objects—for example, when entering a Data import restriction date.

The list includes the following columns:

  • Application. Applications available for data synchronization.
  • Infobase. Infobases available for data synchronization.

Select exchange plan node

  • Click a node and click Select.

Selecting multiple exchange plan nodes

  • Select multiple nodes and click Select.
