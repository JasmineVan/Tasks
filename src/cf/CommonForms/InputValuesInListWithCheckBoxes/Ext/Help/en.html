Use this object to enter multiple values to the filter list.

To open it, in the Report settings window, select Filters tab and double-click in the Value column for In the list, Not in the list, In group from list, Not in group from list operations.

Select values

  • To add multiple values to the list, click Select. Select multiple values from the list.
  • To select values one by one, in More menu, execute Add command.
  • To add list items to the filter value list, select required check boxes (all check boxes are selected by default).

Select values from clipboard

  • Click , select values, click Add to the list.
