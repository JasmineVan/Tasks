Additional proxy server parameters are set up for mechanisms that interact with web services.

If required, specify proxy server addresses for specific protocols and the addresses of Internet servers that must be accessed directly.

By default, the Use proxy server by default for all protocols check box is selected.

Main parameters (server Address and Port) will also be used as additional parameters by default.

Setting up additional parameters

  • To specify proxy server addresses for specific protocols, clear the One proxy server for all protocols check box.
  • In the corresponding field (HTTP, HTTPS, or FTP), correct the name of the proxy server that will be used for each protocol.
  • Specify Port if required.
  • In the Bypass proxy server for addresses list, click Add to specify addresses that will be accessed directly.
  • To confirm the parameters you entered, click OK.
