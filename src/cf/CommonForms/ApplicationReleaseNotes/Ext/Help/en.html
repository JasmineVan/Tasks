The release notes include the list of changes for the latest application version, and also the change history for the earlier versions.

They are displayed after the application is updated. You can view them later by clicking Release notes in the Information panel of the Quick menu section.

Available actions

  • To save the release notes to a file, click Save as and select a file. To print the release notes, click Print.
  • To facilitate text navigation, you can collapse and Expand all groups.
  • You can search the release notes for specific text.

Infobase update progress details

  • If data processing is still running after an application update, an Additional data processing procedures are running warning is displayed for all users. At this time the application functionality is temporarily limited, as migration to the new version is not yet completed.

Administrative features

  • Administrators can view the list of additional actions that are recommended after the mandatory infobase update part is completed. To execute these actions, click the hyperlinks in the release notes.
  • To monitor the progress of additional data processing, click the Additional data processing procedures are running hyperlink.
  • To view the events that appeared in the event log during the update, click the Technical details of the update hyperlink.
