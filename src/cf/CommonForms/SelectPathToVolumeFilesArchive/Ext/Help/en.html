This form is used to specify path to the initial image file archive (in case of volume-based file storage).

If the file storage is based on disk volumes, an archive containing the volume-stored files will be automatically created together with the initial image.

When creating a subordinate node infobase from the initial image in the target application, you will need to get the files from this archive. To do that, use the Place initial image files command. All further data synchronization iterations will use the volume-based files automatically.

In one of the fields, specify Path to initial image file archive:

  • For 1C:Enterprise servers under Microsoft Windows, specify path to the archive by clicking .

or

  • For 1C:Enterprise servers under Linux, type in the archive path.

Click Place.
