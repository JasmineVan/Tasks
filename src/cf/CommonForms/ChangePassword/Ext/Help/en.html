Use this window to set or change the password.

To ensure data security, do not leave this field blank.

Entering a password

  • For security reasons, enter the Previous password before setting the new one.
  • Enter the New password.
  • In the Repeat password field, enter the new password again.
  • For security reasons, password characters are hidden. To show the password characters, select the Show new password check box.
  • Click Set password.

Automatic password generation

  • If you want the password to be generated automatically, click Generate password.
    • The application will suggest a new password. Remember it or copy it to the clipboard, then enter or paste it to the New password field.

Recommendations on creating passwords

A strong password:

  • Is not human-readable.
  • Contains at least 8 characters.
  • Includes:
    • Uppercase Latin letters.
    • Lowercase Latin letters.
    • numbers
    • Special characters (such as underscore and parentheses).

Passwords that contain numbers only, human-readable words, or periodic character sequences, as well as passwords identical to usernames are not recommended.

  • Examples of strong passwords:
    “nj7{jhjibq*Gfhjkm
    F5”njnGhkmNj;t{HI
  • Examples of weak passwords:
    Smith
    qwerty
    12345678
    123123123
