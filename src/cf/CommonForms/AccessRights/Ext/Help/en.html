This list is used to view and edit contents of the user (external user) access groups.

To open this list, execute the Access rights command located on the user (external user) navigation panel.

User's personal access rights are defined by roles assigned to this user via access groups and profiles.

An access group defines a set of actions with the application data the members of this group are allowed to perform. Generally, access groups correspond to user job functions or duties. A user can be a member of multiple access groups at the same time, these groups collectively defining their personal access rights settings.

The following users can manage access groups:

  • Administrators (for all application users).
  • People responsible for the lists of users and external users (for all list members, except for access groups with administrative rights).
  • People responsible for access group member lists (only for users that belong to their access groups).

The list includes the following tabs:

  • Access groups are the commands used to manage access rights settings for a specific user, and a list of access groups containing this user.
    • The list displays:
      • Description is an access group description;
      • Profile is an access group profile description;
      • Person responsible. The user responsible for the access group.
      • Details. The details of the access group selected in the list is displayed in the bottom of the list.
  • Allowed actions (roles). View all actions allowed to the user (external user). The list of roles assigned to a user includes all roles assigned to access groups containing this user.
    • The role list can be sorted in alphabetic order or grouped by subsystem. To switch between these views, click By subsystem.

Access rights settings

  • To add a user into the participants of any existing access group, click Include in group.
    • Select an access group from the list and click Select. Or double-click the access group.
  • To delete the selected access group from the user access group list, click Exclude from group.
  • To change settings of the selected access group, click Change group.

Access rights report

  • To view the full list of user access rights, including the full list of applied objects and access rights per object granted to this user (user group), User access rights report can be generated.
  • Click Access rights report.
