To view a report, specify its parameters (if available) and click Run report. In the client/server mode, the application can generate reports in background.

Generation mode

  • To generate the report automatically upon opening without clicking the Run report button, on the More menu, turn on Run report automatically.

Report options

  • The application may include multiple report options performing a similar analysis. To view these options, click the Report options button.
    • Select the required report option from the dropdown list. The current report option is marked by a selected check box.
    • Use the Other reports command to view the list of other report options.

Settings

  • Click the Settings button to go to configuring report parameters. Some settings (filters) may be sometimes displayed in the report header.
  • To restore default settings, click More - Other - Standard settings.
  • To save current report settings, click More - Other - Save settings.
  • To use previously saved report settings, click More - Other - Select settings.

Changing filter conditions

  • You can view and Change filter conditions for each field if necessary by executing the appropriate command in the More menu.

Other reports

The window contains the list of similar reports.

  • The list includes report names and descriptions.
  • The report list is grouped. There are groups displaying similar reports of the same application section.
  • To go to another report option, click its title.
  • You can Close this window after going to report by selecting the appropriate check box.

Printing reports

  • To print a generated report, click Print button.
  • To check a report and change page parameters, click Preview.
  • To configure report printing, click More - Page setup.

Sending reports by email

By clicking the button Send report by email, you can perform the following actions:

  • To Send now, do the following:
    • select one or more check boxes against the report formats that must be used for sending.
    • Select the Archive attachments (.zip) check box if necessary.
    • Click Continue. The report will be attached to an email.
  • Create report bulk email;
  • Include in the existing report bulk email. To do that, select the required bulk email from the bulk email list.

Select report options

  • To load a previously saved report option, use the More - Other - Select report option command.
    • To go to the required report option, in the report option list, click the Select button.

Edit report options

  • Click More - Other - Edit report option. You can add fields, change report parameters and settings.
  • The changes can be saved to another report option.

Save report options

  • Click Report options - Save report option, specify Name and other parameters.
  • You can also use the Save report option command of the More menu.

Reset settings

  • To reset custom settings, click More - Other - Standard settings.

Other available actions

  • To search in a report, use the Find command.
  • Click Collapse all groups or Expand all groups buttons to hide or show report grouping.
  • To save report result in a file, click , specify the file name and path.
  • Right-click to open the context menu and perform the following:
    • Open the value of the selected cell;
    • Filter records by the selected field value;
    • Order:
      • by the selected field value (by default);
      • Advanced - you can select Field to order the report and order Type (Ascending or Descending);
    • Group:
      • by the selected field value (by default);
      • Advanced - you can select Field for grouping and grouping Type: No hierarchy, Hierarchy, and Only hierarchy;
    • Apply appearance:
      • configure the Conditional report appearance item for the selected field;
      • Advanced - you can configure appearance for any report fields.
