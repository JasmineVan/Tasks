It is designed for full-text search.

You can run search directly from the Home page of the application.

Full-text search

  • To search data in the application, enter the word or its beginning, click Find.
  • Search phrase is saved and at next searches the application displays a list of previously entered search phrases.
  • You can search by one or several words. You can use such search operators as AND, OR, NOT, etc. in the search string. Also you can use other search operators and run search by a part of the word.
  • Search result window displays objects that contain search phrase or word, and short description of this object. The search phrase or word is highlighted. You can open the object from the search result window by selecting it.
  • Use the and buttons to scroll through the search result pages.

Search area restriction

  • If there is a lot of data found, you can restrict Search area to increase the accuracy of search results.
  • For example, you can search for data that relates only to key performance measurement operations.
  • You can customize search area, clicking the link, located below the search string. The link has the Everywhere value by default.
  • Click the link to change settings.

Search errors and how to correct them

  • If data is not found, the application returns an error message. Ways to correct an error:
    • Check the search expression. Modify it if necessary, correct errors according to the application recommendations.
    • You may have selected the wrong Search area. Try to remove or change it using the appropriate link.
    • Perhaps the full-text search index has not been updated for a long time. Control its state, click Update index, if necessary.
  • After correcting the error, run the search again.
