Use this form to add a digital signature from file to items of specific infobase lists. This is convenient, for example, if the signature file was obtained later than the signed data was.

To open this form from the file list (list of attached files for an object), or the Attached files list, click Digital signature and encryption - Add digital signature from file. Or, use the corresponding command in the More menu of the file profile (attached file profile).

The list includes the following columns:

  • Digital signature file - path to the file that contains the digital signature
  • Comment - additional information related to the digital signature to be added

Adding a digital signature from file

  • Select one or several files containing digital signatures.
  • Click Add, then click to select path to the signature file on a local or network disk.
  • Enter Comment for the signature.

Deleting a digital signature

  • Click More - Delete to Delete the digital signature file from the list.
