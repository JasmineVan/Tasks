This form is used to configure working directory parameters.

A working directory is a folder on your computer where files received from a viewer or editor are temporarily stored.

To open the form, in personal application settings, on the File operations tab, click Working directory setup.

Working directory location

  • In the Location field, the path to the main working directory on your computer is specified.
  • It is set automatically by the application, but you can change it manually if required. To do this, select another folder on your computer. When specifying another path to the main working directory, select an empty directory that does not contain any files or subdirectories.
  • To return to initial application settings, click Restore default value.

Max size

  • In the Max size (MB) field, set a limit for total size of all files in the main working directory. By default, it is 100 MB. If it is set equal to 0, the file size is not limited. The size is specified in megabytes (MB).

Application request for file deletion

  • If the Confirm on deleting from the working directory check box is selected, you will be asked for confirmation before deleting a file. By default, the check box is cleared.

Deleting files after editing

  • If the Delete files on finishing editing check box is selected, a file is deleted from your computer after being transferred to an application. Use this setting to save space on your hard drive or to increase security. By default, the check box is cleared.

Current state

  • The application shows the Current state of the main working directory: the number of Files stored and their Total size (MB).

Viewing file list in main working directory

  • To view the list of files currently stored in the main working directory, click Open file list.

Clearing main working directory

  • To delete all files stored in the main working directory, except for files you locked for editing, click Clear working directory. To confirm clearing, click Yes.
