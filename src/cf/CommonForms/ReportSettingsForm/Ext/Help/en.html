Use this object to set up reports.

To open it, in the report form, click Settings.

You can use the View button to select one of the following settings modes:

  • Simple - you can only edit existing parameters by switching check boxes to add or remove the parameters;
  • Extended - you can manage settings: add parameters, change fields and report structure. When you finished configuring in the advanced mode and saved the settings, you can enable and disable them in the simple mode by selecting and clearing check boxes.

Depending on the report complexity, a settings window consists of several tabs:

  • Filters. Use the tab to enter conditions for filtering values in a report.
  • Fields and sortings. Use check boxes to add and remove fields in a report. You can add, remove, or disable fields for sorting the report.
  • Appearance - manage the report appearance;
  • Structure - manage the report structure.

Filters

  • You can use various parameters and filters when generating reports.
  • On the tab, you can change filter parameters. To enable or disable filtering by a parameter (for example, Period), select or clear the appropriate check box.
  • In the Simple mode, you can Change filter conditions in each field by right-clicking the field and selecting the appropriate command in the context menu.
  • There can be the Select link for selecting values. Select the specific value from the list.
  • To select values from the clipboard, click button.
  • In the Advanced mode, you can configure all available report parameters.
  • You can Add a filter by clicking the Add button.
  • To change report filter settings, use the table with the following columns:
    • Field - fields to filter by;
    • Condition - select operation for filtering for each field;
      • Operations Equal, Not equal, In group, and Not in group are used for selection of only one value (single value selection).
      • Operations In list, In group from list, Not in list, and Not in group from list are used for selection of multiple values (multiple value selection).
      • Filled and Not filled operations do not require value selection.
    • Value - specify one or more values for each field for filtering depending on the operation.
      • To enter a value, double-click the field.
      • To enter a parameter value, click Select. Select check boxes next to the parameters you need to filter by. You can use all parameters for filtering or only some of them by enabling or disabling check boxes.
      • To select multiple values, click the Select button. Open the list, select multiple values from the list. Enable check boxes next to the parameters required for filtering. Enter parameter values for filtering or select values from the list.
    • Placement - use it to configure the visibility of a parameter or a filter in the report window. To change a filter's visibility, double-click to select one of the following values:
      • In report header - a parameter or a filter is displayed in a report header and in the report settings header;
      • Only check box in report header - only a selection check box is displayed in a report header and in the report settings header. Use this check box to enable or disable the filter. If you choose this placement, it is highly recommended that you set a completely clear filter title describing the filter value because the filter value will not be displayed. Applicable only for filters;
      • In report settings - a parameter or a filter is displayed in the settings form. This is the default value;
      • Only check box in report settings - only a selection check box is displayed in the report settings. Use this check box to enable or disable the filter. If you choose this placement, it is highly recommended that you set a completely clear filter title describing the filter value because the filter value will not be displayed. Applicable only for filters;
      • Do not show - a parameter or a filter is displayed only in the advanced mode of the report settings.
    • Title - enter your title for a filter.
  • You can group filters. To do that, select multiple filter conditions, right-click to enter the context menu and execute the Group filters command or execute this command in the More menu. You can combine filters to And, Or, and Not groups. To cancel grouping, execute the Ungroup command.
  • For more information on reports and filtering operations, see 1C:Enterprise 8.3. User Manual.

Fields and sortings

  • The Fields and sortings tab consists of two sections:
    • In the right section, you can use check boxes to remove or add fields to a report. This setting changes a report view.
    • In the left section, you can do the following:
      • use and buttons to add or remove fields for sorting.
      • use button to Remove all sortings.
      • click a link to change the sort order and use a check box to enable or disable sorting.
      • if there are multiple sortings, use and buttons to increase or decrease sorting priority.
  • In the advanced mode you can do the following:
    • in the right section, use the Add button to add other fields of documents and lists to a report;
    • in the left section, use the appropriate button to Add sorting by a field. Select a field for sorting from the list suggested by the application.

Appearance

  • On the tab, in the simple mode, you can change some report appearance parameters. For example, you can enable or disable Background color for all fields by selecting or clearing the appropriate check box.
  • In the advanced mode you can do the following:
  • Use the Add button to add a new Conditional report appearance item.

Structure

  • You can change settings on this tab only in the Advanced mode.
  • On the tab, you can add grouped fields to a report. Select matching buttons on the navigaton panel to add, change fields, or swap them.
  • To change report grouping, select the button. On the matching tabs, you can set up the following parameters for each report structure item:
    • Grouping content.
      • On the tab, click the Add button to select a field for grouping. Select the Grouping type from the list: No hierarchy, Hierarchy, Only hierarchy.
      • You can use other tabs of the Customize grouping window in the way you use the same tabs of the report option settings.
    • Fields and sortings;
    • Filters;
    • Conditional appearance.

Changing filter conditions

  • Use button to view and Change filter conditions for each field if necessary.

Getting report data

  • Specify the required parameters and click Close and generate. Last settings will be saved automatically. If incorrect settings were used, you can change them and even reset settings using the commands of the report form.
