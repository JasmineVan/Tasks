The form is designed to configure the 1C:Connect application start button on the home page. 1C:Connect is an application used to connect with an online support specialist. Settings are stored for each user with reference to the workplace.

To open it, in the Administration - Online support and services - Online support section, click Configure online support.

If you launch a base in a web client and in a thin client, you must configure it in each client separately.

In the first step, download the executive file from the company's website. Clicking the Download application link starts downloading the application in the default browser. Unpack the archive to a directory at your convenience.

You must also register on the 1C:Connect website. Use the Register in online support service (1C:Connect) link.

In the second step, specify the following settings:

  • Select the Display application start button to contact online support specialist on the home page check box to display the start button on the home page.
  • Specify a Path to the executable application file. If the file is not found by the specified path during start, the application will prompt for the path again.

In the third step, specify the following settings:

  • Select the Insert username and password when the application starts check box to insert automatically the previously entered login and password at the application startup. In this case, the authorization window will be skipped.
  • Enter Username/password (authorization data of a user on the 1C:Connect website). If the data is not specified or is not valid, the application will prompt for username and password again. You can enter username and password only if it is allowed to insert the data at the application startup.

Click Save to apply changes.

To get help, follow links to the company's website provided for your convenience. To find out system requirements and the list of required settings, click Technical requirements.
