This form is used to write and send text messages.

You can open it from profiles of application objects (partners, companies, and others) by clicking Send text message.

To be able to send text messages, conclude a contract with one of mobile operators (MTS or Beeline). In Text message sending settings, specify received registration data.

Writing messages

  • Type a Message text. The maximum text length of one message is 140 characters (160 Latin characters or 70 Cyrillic characters). Providers can support sending of long text messages (multiple text messages glued together). The maximum length of such text messages is defined by the operator, for example, for Beeline, it is 480 Latin characters and for MTS - 1000 Latin characters. Note that some phone models cannot glue long text messages.
  • Send in transliteration - select the check box to be able to use this feature.
  • In the Message length in characters field, the application counts the number of characters in the message.
  • Select the Sender name check box to add a sender name to the message. Fill in the field with Latin characters.

Checking text message sending settings

  • To make sure that settings are correct, click Configure text message sending.

Send text messages

  • If all settings are correct, to send a message, click Send. The application will inform you of a sending status.
