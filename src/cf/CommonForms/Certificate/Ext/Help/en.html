Use this form to view the basic information about the digital signature and encryption key certificate.

Information about certificate

  • Owner - personal information of the certificate's owner
  • Issued by - information about the certification authority that issued the certificate
  • Expiration date - expiration date of the certificate
  • Purpose - certificate purposes and codes:
    • Signing - this check box is selected if the certificate is for signing purposes
    • Encryption - this check box is selected if the certificate is for encryption purposes
  • Subject - list of properties of the certificate subject, in common recognizable names
  • Issuer - list of properties of the certificate issuer, in common recognizable names
  • Internal content - list and values of properties, as specified in the certificate data

Exporting a certificate

  • To save a certificate on your computer, click Save to file and select a folder to save the certificate file to.

Checking a certificate

  • To check validity of a certificate, click Check.
