Use this window to select attachment format for sending print forms by email.

To open this window, click Send by email.

  • Select the check boxes for one or more formats.
  • It is recommended that you pack print forms into .zip archives before sending them by email.
  • If you select the Transliterate file names check box, the names of report files will contain only Latin letters and digits. This ensures compatibility between different operating systems. For example, the Отчет.mxl file will be saved as Otchet.mxl.
  • Click Select to save your settings.
