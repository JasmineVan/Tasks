This form is used to configure parameters for sending text messages.

To open the form, click Configure text message sending in Administration - Organizer - Send text messages.

With text messages, you can do the following:

  • Inform technical specialists of issues in 1C databases.
  • Automatically confirm payments and other transactions made by customers.
  • Inform customers of a birthday, next payment date, and other events.
  • Bulk notify customers of goods, services, promotions, and special offers.

To be able to send text messages, conclude a service agreement with a provider. You can see service details on provider websites:

  • Corporate SMS services (Beeline).
  • MTS Communicator (MTS).

Configure text message sending

  • In the Provider field, select the service provider the agreement is concluded with.
  • Specify Username and Password. In the fields, specify identification data you received upon concluding the service agreement.
  • You can specify a Sender name in text message sending settings. Only Latin letters are supported. In this case, recipients will see the specified name instead of the sender number.
