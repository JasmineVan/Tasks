This list is used to view and enter values of additional data. Generally, values of additional data are entered in a separate window, as opposed to values of additional attributes.

Opens with the button Additional data from program object windows, for which additional data was set.

The list includes the following columns:

  • Description is additional data field description.
  • The Value field is used to view or enter additional data values.

Entering values

  • To switch to additional data value input/edit mode, double click the corresponding field.
  • The way of entering a value depends on additional data type (for more information on types of additional attributes and data click here).
  • Additional data can have a choice only from two values, for example Yes or No. In this case, new values cannot be entered.
  • If a list of values is available for input, click Show all (or press F4). Then select a value from the list. Value list can be with hierarchy or without hierarchy, depending on settings. New values can be added to these lists if necessary.
  • Depending on the settings, values may also be selectable from the application catalogs.
  • Depending on the additional data settings, you may be required to Select type of data.

Changing additional information content

  • If you have access rights, you can set up new additional information or change some settings of existing additional information. To do this, execute the More - Change additional data content command.

Deleting additional data

  • Additional data marked for deletion is depicted with the sign. In order to permanently remove additional information, you may have to Clear the values. Double click a value to switch to edit mode.
