Use this list to assign a user or an external user to groups.

To open the list of available groups, use the Groups command available on the user (or external user) navigation panel.

Adding users to groups

  • Select the check boxes next to the groups.
  • Click Save and close.

Removing users from all groups

This feature is convenient when a user was added to multiple groups by mistake. Then you can quickly remove a user or an external user from all groups and add it to appropriate groups.

  • Click Remove from all groups.
