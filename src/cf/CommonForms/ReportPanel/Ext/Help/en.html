The panel displays report options available in a section or in one of the section's groups.

You can open the reports panel by the matching command from the application sections, for example, in the Administration-Maintenance-Reports and Processing section there is the Administrator reports command.

The report panel provides a place to work with reports and groups report options by command interface groups. You can use the report panel in two modes:

  • in the view mode;
  • in the setup mode.

View mode

  • When the report panel is opened for the first time, a notification will be displayed. It informs that descriptions will be displayed together with reports. The application allows you to choose whether you would like to leave descriptions enabled or to disable them. You can also make this choice in the setup mode.
  • Report panel groups match groups of the application section where the report is located.
    • The report panel can also include other grouping types depending on the specific report list.
    • There is the See also group. It usually consists of report options that are rarely used in the application.
    • Important reports are marked in bold in the report panel.
    • You can set up these groups in the all reports list. To go to the list, execute All reports.
  • To go to the report panel setup mode, select Setting or F2.

Searching for reports

The search is available both in the view mode and in the setup mode.

  • You can search for reports using a word (a part of a word) or several words. The search is performed in:
    • a report name,
    • a report option description;
    • report option field presentations;
    • presentations of report option filters and parameters;
    • names of report options' custom settings;
    • names of sections and groups of the application and the report panel.
  • In the Search field, enter the required value and select Find. The search is performed first in the current section, and then additionally in all other sections of the application (these results are displayed under the Reports found in other sections header, the found reports are grouped by application sections, each section header is a link that is designed to quickly go to the panel reports of another section).
  • Search settings and search words are saved.

Creating reports

  • Click the report name.

Canceling the report panel settings

  • If reports were posted incorrectly, to roll back to the report panel default settings, click More - Clear my settings.

All reports

  • To go to the list of all reports, click All reports. In the list, you can set up report options and their location in all sections of the application.

Setup mode

  • To switch to this mode, click Settings or press F2.
  • Select the Report descriptions check box to set up displaying report option descriptions.
  • Use check boxes to control visibility of a report option in a group.
  • Right-click a report option to access the context menu and move this report option to quick access or to open it for editing the name, description, or location settings:
    • To set up quick access, right-click a report option and select Move to quick access;
    • To cancel quick access to a report option, right-click this report option and select Remove from quick access.
    • To open a report option for editing its name, description, or location settings, right-click this report option and select Edit.
  • To save changes and switch the report panel to the view mode, click Settings or press F2.
