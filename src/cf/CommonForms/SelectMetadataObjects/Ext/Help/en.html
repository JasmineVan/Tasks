Use this list to select application objects.

You can open it from other application objects using the Select command or the corresponding hyperlink.

The list of objects is a tree grouped by the object type.

Selecting application objects

To select objects, select the check boxes next to them.

  • To select all objects of a certain type, select the check box next to the object group. For instance, to select all catalogs, select the Catalogs check box.
  • To select all objects, select the root-level check box.
