This form is used to view, edit, or print the generated print forms, to send print forms by email, and to save print forms to file.

You can access this form from any document or application document list by executing the Print command

The following functionality is available:

  • Print documents is a common print form for all documents.
  • Printing the document with digital signature visualization stamp;
  • Printing batches;
  • Customizing batches.

Output to printer

  • Click Print to print the specified number of copies of the generated print form.

Number of copies

  • Specify the number of Copies. The selected number of copies will be saved. When printing document batches, the number of copies depends on the batch settings.

Edit print form

  • Click Edit to enter the print form edit mode. This will lock the print form and make it unavailable for editing by other users. Click Edit again to exit the edit mode.

Saving print forms

  • Click Save to save the print form to a folder on a local computer, in a specified format, or to attach it to a document. Use the radio button to select the method of saving the document and the required formats for saving.

Sending by email

You can Send the generated print forms as email attachments.

  • Prior to sending a print form, you need to specify one or more formats for it using the check boxes.
  • We recommend that you compress the print forms into .zip format before sending them by email. Click Select to save your settings.
  • Fill in the To and Subject fields, type in the email text. Click Send.
  • When sending a document batch by email, each print form will be attached separately. When printing a print form batch for multiple documents from the list, we recommend that you break down the documents by counterparty. This is helpful in cases when you need to send a document batch to a specific counterparty; for example, if the batch includes two print forms, the first file will contain all invoices, and the second file will contain all bills.

Quick access to documents

  • You can select several documents to print from the document list.
  • In this case, multiple print forms will be provided. For quick access to a specific print form, click Go to document (or press F7).
  • Select a document from the list and click OK.

Edit template

  • Click More actions - Change template to edit template of the print form. This will lock the print form. When the editing is complete, the print form will be updated automatically.

Go to list of print form templates

  • To open the list of templates, click More actions - Go to list of print form templates.

Printing the document with digital signature visualization stamp

To add the digital signature visualization stamp to the print form of the selected document:

  • Select a document in the list of documents and click Print.
  • To attach print form to the document, click Save and select the saving format - Spreadsheet(.mxl).
  • In the document command bar click the Attached files hyperlink, select the added spreadsheet document and sign it with the digital signature.
    • select the Digital signature and encryption - Sign in the context menu, then select the needed digital signature certificate in the Certificate field, enter the password, and click Sign.
    • in the web client at request for cryptography operation extension click Yes;
  • When signing is completed, stay in the same spreadsheet document and click Print - With digital signature stamp.

Print set

  • You can print out document batches. Batch print forms are displayed on separate pages; to view a specific print form, select it in the print form list on the left. You can change the number of Batch copies.

Customizing batches

You can print out document batches selectively.

  • The number of copies selection panel, located in the left section of the Customize batch window, includes the following columns:
    • - select check boxes in this column to specify the batch documents for printing;
    • Print form
    • number of Copies.
  • To customize printing of document, click More actions - Batch setup. Use the drop-down menu commands to Select or Clear all check boxes, change print form order within the batch (Move up/down), or Reset batch settings.
  • Select a batch print form to view. Select the check box for printing this form. Specify the number of copies for each print form. to save your settings.
  • You can also change the number of Batch copies.
  • For your viewing convenience, you can disable the number of copies setup panel by clicking Show/hide number of copies settings.

Aggregate values of selected cells

  • Select the cells with numbers that you need.
  • Click to calculate Aggregate values of selected cells.
