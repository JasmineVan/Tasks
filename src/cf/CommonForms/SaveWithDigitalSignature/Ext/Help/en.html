Use this list to select digital signatures to be saved together with a file.

This list is opened when you execute the Digital signatures and encryption - Save with digital signature command, for example, from the file list, attached file list, or file profile (attached file profile), provided that On saving data to the file with digital signature is set to "Prompt" for the current user. This command is available in the navigation panel, the More menu, and the context menu.

The list includes the following columns:

  • Save - a check box
  • Signature author - presentation of the person the certificate is issued to
  • Signature date
  • Comment - detailed information on the digital signature

Select check boxes for digital signatures to be saved with the file and click Select.

You can save a certificate together with digital signature. To do this, select the Save certificate with signature check box.

To disable this window and always save all digital signatures automatically, select the Do not ask again check box.
