Use this window to save print forms. You can save print forms in the required formats on a computer or attach to a document.

  • Using the radio button, select how to save the print form:
    • Save to local folder.
      • Specify a path to the folder on your computer.
    • Attach to document.
  • Prior to saving or attaching a print form, you need to specify one or more formats for it using the check boxes.
  • To compress the print forms before saving, select the Archive attachments (.zip) check box.
  • Click Save. Your settings also will be saved.
