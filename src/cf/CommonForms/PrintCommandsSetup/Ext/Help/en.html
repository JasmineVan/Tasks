It is designed to set up the components of all print forms of the Print submenu in documents and lists of the application.

To open it, click the Print submenu in the Administration - Print forms, reports, and data processors section.

The list consists of two sections:

  • print commands are listed in the right part. Commands are grouped by the lists of applied objects,
  • the left part lists the external print forms that correspond to the print commands.

Edit print form compsition

  • The composition of the print forms of documents and lists can be determined using the check boxes: you can enable or disable separate commands and the Print submenu for the entire list or a document.
  • After setting the Print submenu, click Write and close to confirm changes.

Go to list

  • Click Go to list to open the selected list, for which commands are set.

External print forms

  • There are external print forms for some documents, and you can also enable and disable them as other commands from the Print submenu.
  • You can assign an external print form only from the Additional report and data processors list.
  • Click the link in the External print form column to go to external print form parameter settings.

Print forms and extensions

  • The developer can assign print forms in the extension.
  • In the Print submenu, commands are displayed with a mark from the extension, you can control them using check boxes.

Print submenu operations

  • The set of commands for the Print submenu is provided both in the documents and in the lists.
  • You can select several documents from the list and print them. In the case of a multiple choice in a list form or in a document journal, only those printed forms that are suitable for all selected documents remain available as part of the Print submenu.
  • The composition of the Print submenu in a number of documents, lists and journals automatically adjusts to the selected document: only those print forms are shown that can be generated for this document at the current values of attributes.
  • Some print forms of the documents in the list can be missing (depending on posted settings).
