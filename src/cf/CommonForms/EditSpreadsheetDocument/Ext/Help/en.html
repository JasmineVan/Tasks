Use this window to edit spreadsheet documents.

To edit a spreadsheet document (or a template embedded in a print form or a spreadsheet document), in the More actions menu, click Change template.

Important: In print form templates, changing area names and template parameters might lead to the loss of print form functionality. You can revert to the default print form settings in the Print form templates list.

Edit

  • You can use the standard formatting buttons to edit the font size, the typeface, and alignment. Other formatting options, such as background, font, and border color, are available in the More actions menu.
  • To merge cells, on the More actions menu, click Merge cells. To split cells, use the same command.
  • To add a comment, on the More actions menu, click Insert comment.
  • When you have finished editing, click Save and close.

Switching between view and edit modes

  • To switch between the modes, click Edit. In view mode, the template editing buttons are disabled.

Printing templates

  • To print a template, on the More actions menu, click Print.
  • To preview the template before printing, on the More actions menu, click Preview.
  • To edit the print settings, click More actions—Page setup.

Editing in external applications

  • To open a template in an external application, click More actions—Edit in external application.
