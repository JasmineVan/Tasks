This form is used to set up or validate inter-application connection parameters used for data synchronization.
Parameters of various connection kinds are set on the tabs.

The connection parameters are also displayed during data synchronization setup procedure in the Data synchronization creation wizard. The default parameters are displayed. They can be modified.

To start setting up the Connection parameters, use the related command in the data synchronization settings.

The following parameters are common for all connection methods:

  • Default connection - select a connection to be used as default during both automated and interactive data synchronization.
  • Test connection - click to validate the connection parameters and attempt connecting with these parameters.
  • Compress and archive sent data - select this check box to reduce the size of sent messages. For direct infobase connections, this check box is disabled.
    • Archive password - enter the password required to compress or decompress exchange messages. For successful data exchange, compression password in the source application and decompression password in the target application must be identical.

Direct connection:

  • Use the radio button to select an infobase type:
    • Correspondent File mode infobase
      • Select Infobase directory.
    • Correspondent Client/server mode infobase
      • Enter Server name;
      • Enter Infobase name on server.
  • Select the OS authentication check box if necessary (this check box is cleared by default). In this case, the following fields should remain blank.
  • Enter User name;
  • Enter Password.

Internet

  • Enter the application URL.
  • Enter Username and Password.
  • Select the Remember password check box if necessary.
  • To configure proxy server parameters, click the Internet access parameters link.

Network folder

In the Folder field , Select path to a local or network folder.

FTP resource

  • Specify Folder path and data transmission Port.
  • Enter Username and Password.
  • Specify Maximum file size if necessary to reduce the size of the sent message.
  • Select the Passive connection check box if necessary.
  • To configure proxy server parameters, click the Internet access parameters link.

Email messages

  • In the Email account field, select an email account from the list or create a new account.
  • Specify Acceptable email attachment size if necessary to reduce the size of the sent message.
