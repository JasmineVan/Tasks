In the application you can enter reminders by specifying the time and subject, about which the program will display a list of reminders at the specified time.

The subject of reminders can be various catalogs, documents and other applied objects. For example, a reminder can be entered for a specific partner, goods movement document, etc., in order to remember to clarify or fill in any data.

To open it, click Organizer - Remind in distinct applied objects. In this case, you can enter only one reminder for one object.

You can enter a reminder connected with some personal event or task in the My reminders list by the Create command. Such reminder can be useful not to forget to congratulate colleagues on their birthdays on time. Also you can use reminders together with notes as an electronic replacement for "stickers on the edges of the monitor".

Entering a reminder

  • Edit or write text of a reminder.
  • If the reminder is connected with a document, catalog or another applied object, then the Subject field contains a link to the program object, its description will be inserted in the reminder text by default.
    • Using the link in the SubjectSSL field, you can view the program object for which the reminder was created.
  • Select one of the following options in the Remind field:
    • At the specified time is used to specify the exact date and time of the reminder. Enter the date manually or select it from the calendar clicking the button.
    • In a certain period of time. To do this, select a value from the list to set the period of time in which the reminder will activate. You can also use the keyboard to specify an arbitrary number of hours, for example, in 10 hours. The application automatically calculates the date and time of the reminder and displays it in brackets next to the Remind field.
    • Relative to subjectSSL is used to specify the time of the reminder in relation to the date in the reminder subject, for example, relative to the execution date in the task. This value of the Remind field is displayed and suggested by the application by default, when the item has at least one field with a date in the future. Select from the list how early before the Date you need a reminder to be displayed. If such a date in the subjectSSL changes, the reminder time changes automatically. If the date in the subjectSSL is reset for some reason, then the reminder period will be considered as approached and the reminder will be displayed ahead of schedule.
    • Periodically is used for periodic reminders, for example, you can select from the list the on Mondays at 9 a.m. value, or enter a different schedule. To enter your schedule, select the value from the list according to the specified schedule and make the necessary settings.

Cancelling a reminder

  • To cancel a reminder, click Delete.
  • The application displays a message, where it requires confirmation. To confirm, click Delete. To cancel, click Do not delete.
