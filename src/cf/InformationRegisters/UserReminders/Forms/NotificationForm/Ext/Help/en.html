It is designed to display activated reminders.

At the time specified in the reminder, the application opens the Reminders list. The list contains all newly created and deferred reminders, which notification time is up.

Each reminder displays its text and time, for how much it was overdue. If a reminder is connected to date of some subject, for example, with the due date for a task, then the reminder indicates how much time is left before the event starts. In the list, you can perform the following actions:

  • Go to the catalog item or another program object to which the reminder is connected (click Open or double click the item). If the reminder is not connected to the program object, then the reminder will open itself.
  • To cancel the selected reminders and delete them from the list, click Stop. Periodic reminder after clicking Stop continues to operate according to the configured schedule. You can completely stop this reminder in the My Reminders list.
  • You can defer all reminders in the list for a period of time, specified in the Remind in field by clicking Defer all. To fill in the field, select value from the list or use the keyboard, specifying an arbitrary time value.
