It Is designed to work with all the reminders operating in the program at the moment. Each user can see only their reminders.

The list includes the following columns:

  • Reminder time is the time when reminder activates.
  • Details - is the text of a reminder.
  • Event time is the time to which the reminder was assigned at the moment of its creation, or the time in the subject of the reminder, if the Reminder Time is relative.
  • Source is a reference to the applied object, to which the reminder is connected. To view the applied object, click the link.

Entering a reminder

  • You can enter a reminder about a personal event or task that is not associated with any particular applied object.
  • Select Create and fill in the required fields.
  • Such reminders will have Not specified in the Source column.

Viewing a reminder

  • To view or edit a reminder, select the needed line in the list, click Edit.

Cancelling a reminder

  • To view or edit a reminder, select the needed line in the list, click Delete.
  • The application displays a message to confirm. Click Delete. Click Do not delete to cancel deletion.
