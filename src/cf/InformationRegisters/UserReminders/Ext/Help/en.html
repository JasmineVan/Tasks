In the application you can enter reminders by specifying the time and subject, about which the program will display a list of reminders at a specified time.

The subject of reminders can be various catalogs, documents and other applied objects. For example, a reminder can be entered for a specific partner, goods movement document, etc., in order to remember to clarify or fill in any data.

To open it, click Organizer - Remind in distinct applied objects. In this case, you can enter only one reminder for one object.

You can enter a reminder connected with some personal event or task in the My reminders list by the Create command. Such reminder can be useful not to forget to congratulate colleagues on their birthdays on time. Also you can use reminders together with notes as an electronic replacement for "stickers on the edges of the monitor".

Reminder list:

  • At the time specified in the reminder, the application opens the Reminders list. The list contains all newly created and deferred reminders, which notification time is up. Each reminder displays its text and time, for how much it was overdue. If a reminder is connected to date of some subject, for example, with the due date for a task, then the reminder indicates how much time is left before the event starts.
  • From the list, you can go to the catalog item or the document to which the reminder is connected (click Open or double-click the item). If the reminder is not connected to the program object, then the reminder will open itself.
  • To cancel the selected reminders and delete them from the list, click Stop. Periodic reminder after clicking Stop continues to operate according to the configured schedule. You can completely stop this reminder in the My Reminders list.
  • You can defer all reminders in the list for a period of time, specified in the Remind in field by clicking Defer all. To fill in the Remind in field, select value from the list or use the keyboard, specifying an arbitrary time value.
