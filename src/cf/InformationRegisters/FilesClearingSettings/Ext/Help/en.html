The tool is designed to clean up space on the disk used by unnecessary files (obsolete, or added by mistake).

The application allows you to reduce the amount of data used to store unnecessary files. Unnecessary files are files that are no longer needed, for example, after a certain period of time.

The application provides you with a tool to configure rules to delete unnecessary files according to the file properties and properties and types of their owners. You can also delete these files by using a scheduled job.

Obsolete file versions can also be deleted.

To open it, click Configure file cleanup in the Administration - File operation settings section.

For example, you can configure cleanup of attachments with low importance for the Incoming email message documents created more than a year ago.

Configuring cleanup

  • Cleanup can be made manually, or automatically according to the schedule. You can set custom rules to clean up files. There are several buttons to configure cleanup of unnecessary files.
  • Click Configure cleanup to Add or Delete settings. For example, you can add file cleanup settings for a new file folder.
  • Click the required button to Set an action performed by the application with unnecessary files:
    • Clean up files and versions
    • Clean up files
    • Clean up versions
    • Do not clean up
  • Click Set period to select a period beginning from which files will be cleaned up:
    • Over one month
    • Over six months
    • Over one year
  • You can also configure your own file cleanup rules. To do so, select According to rule in the Cleanup period column.
    • In the Rule column, the Change button becomes available. Click the button to configure a more selective rule to filter unnecessary files.
      • For example, you can configure cleanup of files and file versions created before beginning of this year for the Accounts folder.
      • In the Configure file cleanup window, click Add, and then set a new filter for the file owner.
      • If the object has an attribute of the Date type, you can configure file cleanup rules according to this attribute. In this case, click Add, select Add condition by date, and then enter a date to filter.
    • After the selection is confirmed, the Change link becomes active. You can use it to change entered filter conditions.

The "Unused file size" report

  • To get a report on the volume of data on the disk used by unnecessary files, click the Unused file size link.

Cleaning up files manually

  • By clicking Clean up, you can always delete unnecessary files manually according to the configured cleanup rules.

Configuring automatic file cleanup

  • Select the check box to Automatically clean up unused files on schedule by using a scheduled job.
  • After that, the Configure schedule link becomes active. Configure the job schedule.
