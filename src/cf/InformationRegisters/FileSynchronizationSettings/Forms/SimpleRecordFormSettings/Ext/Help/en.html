The form is designed to synchronize files for each owner—document, folder, and other application objects.

You can open it from the Attached files list of the application objects by clicking More – Synchronize with cloud.

Configuring file synchronization

  • Select the Synchronize files with cloud service check box.
  • In the list, select a cloud service account to synchronize files with. If there is no necessary account in the list, you can create it.
  • Click Write and close to confirm.
