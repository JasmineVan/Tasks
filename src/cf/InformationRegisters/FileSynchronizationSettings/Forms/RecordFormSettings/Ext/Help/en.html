The form is designed to synchronize files for each owner—document, folder, and other application objects.

To open the form, click the required link in the File synchronization settings list.

  • The Synchronize files with cloud service check box is selected by default.
  • In the field below, select a cloud service account to synchronize files with. If there is no necessary account in the list, you can enter a new one.

What to synchronize

  • In the What to synchronize section, you can select one of the values:
    • All files attached to (you can select an application object). In this case, filter conditions are not required.
    • Files (specify the selected list) by synchronization rules. Set as default.
      • There is no need to fill in file filter conditions if you want to synchronize all files.
      • If necessary, add a new filter: click Add, and then select Add new item. You can also press Ins on your keyboard.
        • In the Field column, select a field from the list.
        • Select a Comparison kind.
        • Select or type a Value.
  • Add a Short rule description.

Condition by date

  • To set filters quickly, you can use a standard condition with a date related to the object – file owner.
  • Click Add, and then select Add condition by date.
  • Specify a number of days against the date after which files are cleaned up.
  • Select a Date from the list: one of the dates specified in the object (file owner). For example, document sent date, etc.

Filter grouping

  • Select several filters, and then click Group conditions. Select grouping condition: And, Or, or No.
  • By default, several filters are grouped by And.
