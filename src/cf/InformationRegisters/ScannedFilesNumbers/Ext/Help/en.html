The list if designed to view numbering of scanned documents and their owners.

To view the list, click Scanning setup - Open scanned file numbers on the File operations tab in the personal application settings.

The application automacially assigns a number to the scanned documents. In the list, you can view file movements and those that were removed.

The list includes the following columns:

  • Owner. Application object. If a number is used in several application objects, you can see it in the list.
  • Number. Number of a scanned file.

Resetting scanned file numbers

  • During scanning, you might need to reset numbering of scanned files (start from zero) for a particular file owner (file folder, or list item). For example, if several images were scanned as a test, and after that documents must be scanned and assigned correct numbers.
  • To do so, select a file owner, and then set the value number to 0.
