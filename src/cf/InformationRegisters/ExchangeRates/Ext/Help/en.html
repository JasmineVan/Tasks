This list stores currency exchange rates.

Currency rates can be updated manually (see Exchange rate import) or downloaded from the 1C website automatically by a scheduled job.

You can open this list from the Currencies navigation panel.

If currency rate download option is unavailable, you can input the currency rates manually.

The list includes the following columns:

  • Date.
  • Currency. An alphabetic code of a currency compliant with international standards.
  • Exchange rate.
  • Multiplier.

Entering currency rates manually

  • Click Create,
  • select a Currency from the list identical to the Currencies list,
  • enter the Date for the rate or select it in the calendar ,
  • enter the Exchange rate,
  • enter the Multiplier (the default value is 1). If an exchange rate for a currency in relation to a domestic currency is very low (accuracy is less than four digits in a fractional part), specify the quantity of quoted units. For instance, if the exchange rate of a Vietnamese dong (VND) is RUB 16.1574 for VND 10 000, you need to set the Multipler field to 10 000, and the Exchange rate field to 16.1574.
