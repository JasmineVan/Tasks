This list is used for print template management. The list includes templates (blank forms) for all print forms available in the application.

To open the list, execute the Print form templates command located in the Administration - Print forms, reports and data processors section.

The list includes the following columns:

  • - modified templates are marked with this icon;
  • Print form template is full name of print form; name of application used to create the template. An icon next to the description specified application used to create the template.
  • Template owner is an applied object to which the print form is linked. The templates are grouped by applied object.

Filters

  • For your convenience, extensive lists of templates can be filtered.
  • You can easily locate a template by its full or partial name. Type in the template name (full or partial) into the left field.
  • You can also filter the templates via the Show field:
    • All templates,
    • Standard templates (without changes),
    • Used templates with changes,
    • Unused templates with changes,
    • All templates with changes.

Viewing templates

  • To view a template, click Open. Or double-click the template. Then, select the template opening mode if necessary. Templates in MXL format are opened in read-only mode.

Edit template

  • Print forms are generated on the basis of standard templates distributed with the application. However, standards templates can be edited if necessary.
  • To edit a template, click Change. Or double-click the template. Then, select the template opening mode if necessary.
  • Templates in spreadsheet document format can be edited directly in the application. Templates in office document formats (Microsoft Word, OpenOffice.org Writer) are edited in the corresponding software, which needs to be installed separately.
  • If the required editor software is not installed, any attempt to edit a template in related format will display a message that reminds you to install the software and offers a link to the installation webpage. Click Continue.
  • Web client may require additional setup before you can start working.
  • If the required editor woftware is installed on your computer, the selected template will be opened for editing.
  • Make your changes to the template, save the template, and close the editor software. To save the template and exit the editing mode in the application, click End editing.
  • The templates with saved changes will be marked with the icon in the list, meaning that the template was modified and will be now used for printing by default.

Switching between standard and user templates

  • After a print form template was edited, you can freely switch between the standard and edited versions of the template.
  • To switch between template versions, click
    • Use custom (user) template;
    • Use standard template. In this case the icon will become inactive, which means that a custom template is available.

Warning

  • In certain cases, a template can be edited in such a way that prevents correct print form generation. Generally, this is caused by deleting or renaming template areas and parameters.
  • If a print form generation fails after template editing, you can attempt to roll back the harmful edits and repair the template.
  • If a damaged template cannot be repaired in this way, you need to Delete changed template. The standard template will be used instead.

Action on print form selection

  • To specify the action to be performed by double-clicking a template, execute the More actions - Set action on print form template choice command. Specify the software to be used to open the template. If you want the application to memorize your selection for future use, select the corresponding check box.
