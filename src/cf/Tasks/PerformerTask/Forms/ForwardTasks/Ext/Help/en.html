You can forward a task or a group of tasks to another user via personal or role addressing.

  • Select an option by using the radio button:
    • For assignee. Personal addressing requires that you explicitly select an assignee from the list of application users.
    • For role performers. Role addressing requires that you specify a role, and, if necessary, the main and additional addressing objects.
  • Add a Comment if you want to explain your reason for forwarding the task.
  • Click Forward. The task will be added to the personal task list of the selected assignee.
