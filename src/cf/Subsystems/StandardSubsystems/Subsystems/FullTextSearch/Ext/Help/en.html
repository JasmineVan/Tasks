this configuration allows performing a full-text search, looking for a specified string in any object attributes of any application object. If there are attached files containing text information, the search is also executed by them.

Full-text search provides such features as: latinization (conversion of non-Latin characters to Latin), substitution (using Latin characters instead of identically-looking non-Latin letters), and fuzzy search (finding strings approximately matching the search string).

To enable and set up the full-text search feature, use the system parameters setup form located in the Setup and administration section.
