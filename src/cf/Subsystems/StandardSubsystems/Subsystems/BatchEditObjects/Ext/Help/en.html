Use batch object modification to change attributes, additional attributes, or additional information (hereinafter: attributes) of selected objects of a specified type in one action. Batch object modification does not require additional roles from a user. If a user can edit an object itself, batch object modification is also available for this user. To go to the batch modification wizard, select a set of objects in a list using multiple selection and click Change selected items in the command bar or in the list context menu.
