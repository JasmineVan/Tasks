Users and external users can be synchronized with the infobase users list that can be accessed in Designer. It allows to view more data on users and to create references to them from other infobase objects.

 

Important: It is recommended to manage infobase user lists and user access rights only in 1C:Enterprise. Use Designer only to create users with full rights—system administrators. When a user with full rights starts the infobase in 1C:Enterprise mode, corresponding items will be automatically generated in the Users catalog.
