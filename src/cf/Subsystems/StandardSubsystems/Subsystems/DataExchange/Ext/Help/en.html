This application supports performing the data synchronization between infobases by data exchange rules.

The exchange rules are created in the Data conversion application, version 2.1.

Data synchronization can be monodirectional or bidirectional. It can be performed automatically by schedule or manually by user request. To synchronize data, various communication channels can be used: network directory, email, or FTP.

To set up data synchronization, use the Data synchronization command located on navigation panel in the Administration section.
