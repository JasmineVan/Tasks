In the application, you can store files of custom format. Unlike storing files on a hard drive, files stored directly in the application can be linked with infobase data. For example, you can store detailed goods description in a file attached to the "Products and services" catalog.

To manage files, in the Setup and administration section, configure application parameters. You can also find user settings related to file management in the My settings form of this section.
