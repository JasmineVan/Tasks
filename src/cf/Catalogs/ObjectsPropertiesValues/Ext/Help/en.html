The list contains values of additional attributes and info.

In the application, you can use additional attributes or info of several types. The data type is selected upon setting additional attributes or info. When you select the Additional value data type, the application suggests that you enter values into the list. For each additional attribute or info of this type, a specific value list is used. With this data type, you can describe various properties of application objects.

The Additional value data type is commonly used, information is entered by selecting values from a list, which speeds up data input process and reduces the number of mistakes.

Both standard and grouped value lists are used.

Selecting values

  • Select an item.
  • Click Select.
  • You can also double-click the item.

Entering values

  • To enter values, click Create, and then fill in the required fields.

Grouping values

  • If there are a lot of values, you can group them in the list. To do this, enter a number of groups. Groups can be independent or hierarchical.
  • To include group in the list, click Create group, and then fill in the required fields.
  • Drag values into groups.
