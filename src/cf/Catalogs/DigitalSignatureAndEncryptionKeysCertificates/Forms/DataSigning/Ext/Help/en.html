Use this form to digitally sign files, electronic documents, and other data sets.

Digital signature verifies that an electronic document was not altered in any way since the moment it was signed. Electronic document integrity is verified by matching the digital signature with the document and by checking validity of the certificate. Integrity control of an electronic document can be performed at any moment of time.

Data signing

  • To view the object to be signed, click the hyperlink at the top of the window. The hyperlink text displays the name of the object.
  • Multiple users can sign the same data set at the same time. In this event, a unique signature is created for each user.
  • Select Certificate from the list if no certificate is selected. If the certificate is listed as installed on the computer during the next application startup, it is not necessary to select it again.
  • When choosing a certificate from a drop-down list, the list only displays certificates that have the User field blank or equal to the name of the current session user and have the Application field filled. If you enable filter by a specific company before opening the Signing window, the drop-down list only displays certificates that have the above company specified in the Company field. If you select a specific certificate list before opening the Signing window, only this list is displayed.
  • If you select personal certificates (installed on the local computer and on server) by clicking Show all, all certificates are displayed, regardless of values of User and Application fields. If you enable filter by a specific company before opening the Signing window, the list only displays certificates that have the above company specified in the Company field. Certificates that are not installed (only stored in the certificate storage managed by local computer or server OS) are not displayed in the list.
  • Enter the private key Password that was specified during certificate creation or installation.
  • To skip entering the password again during the next sessions, select the Remember password check box.
  • In some situations, the Comment to signature field is available for entering additional information about the signature.
  • To apply the signature, click Sign.
  • The data set will be signed using the certificate you specified.
  • The created digital signature is stored in the application. The signed object cannot be edited. When displayed in lists, it is marked with icon.
