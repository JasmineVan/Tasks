Use this list to select a digital signature certificate.

The list includes the following columns:

  • Presentation - full name of the certificate
  • Company - the company that issued the certificate
  • User - the user that owns the digital signature

Selecting a certificate

  • Click a certificate and then click Select. You can also double-click the item.

Adding a certificate

  • Click Add and select a certificate source:
    • from certificates installed on computers - choose a certificate from list
    • from file - select path to a file A new window will open. Fill the required fields in it.

Viewing certificate data

  • To view the certificate data, select the Change command in the context menu or More menu.
