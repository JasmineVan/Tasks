Use this form to decrypt encrypted data.

The following fields are available:

  • The hyperlink at the top of the window displays information about data to be decrypted.
  • The Certificate field is used to select an installed certificate to use for data decryption.
    • If the drop-down list of certificates is empty, you need to install a certificate with a private key on this computer.
      Make sure that you install a certificate that was used to encrypt the data you want to decrypt (for example, if the encrypted data is in a file, check the file profile on the Decryption allowed tab).
    • If the list of certificates used for encryption is empty but you know the name of the certificate that was used for encryption, click Show all in the drop-down list and then select this certificate.

Data decryption

  • In the Password field, enter the password for the selected certificate.
  • To skip entering the password again during the next sessions, select the Remember password check box.
  • Click Decrypt.
