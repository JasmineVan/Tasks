Use this form to select certificates for data encryption.

Data encryption

  • To view the object to be signed, click the hyperlink at the top of the window. The hyperlink text displays the name of the object.
  • Personal certificate - required for decryption. Mandatory.
    • select a personal certificate from list.
  • Certificates of other users allowed to decrypt data - list of other certificates
    that can be used for decryption.
    • To prepare a selection list, click Select (to select one or several certificates from list) or Delete.
  • You can use multiple certificates during data encryption. For example, it may be necessary when encrypting a data set for several users.
  • Click Encrypt to confirm.
