Information about digital signature certificate.

The following certificate information is displayed at the top of the form:

  • Owner
  • Issued by
  • Expiration date
  • Purpose

Certificate owner information:

  • Signature owner data (are populated automatically) in the Owner field:
    • Full name
    • Company
    • Job title
  • Presentation - full name of the certificate (filled automatically)
    • Company - the company the certificate is issued to (manually selected from list)
    • User - the certificate owner (manually selected from list)
    • Added by- the user that added the certificate to the list (selected manually)

Private key settings

If necessary, you can store the password not in 1C:Enterprise application, but in digital signature application.
    • Application - choose digital signature and encryption application, where the password will be stored;
    • Entering and saving the password in application - enable the check box if it is necessary to store the password in digital signature application.

Saving to file

  • To save the certificate to a file, click Save to file and specify the destination folder.

Automatically filled attributes

  • To show automatically filled attributes, select the Show automatically filled attributes check box in the More menu. This will display the automatically filled details of the signature owner.

Revoked certificates

  • To revoke a certificate, click More - Certificate revoked. Be advised that this action cannot be undone.
