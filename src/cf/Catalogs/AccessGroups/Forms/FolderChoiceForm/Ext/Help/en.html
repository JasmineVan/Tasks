This list is used to select folders for access groups.

To facilitate management of multiple access group profiles with similar rights restrictions and avoid confusion, you can create folders to simplify the search for the required access group and to understand the access right restrictions in general.

The access groups can be organized by a variety of criteria.

Selecting an access group folder

  • Select a folder and click Select. Or you can left double-click to open a report option.

Creating access group folders

  • Click Create group and type in the folder name
