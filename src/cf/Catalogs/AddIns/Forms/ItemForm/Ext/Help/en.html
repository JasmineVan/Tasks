Information about add-in.

To open the add-in information profile, go to the Add-ins list and double-click an add-in, click Add from file, or execute the More – Change command.

All information required for add-in installation is stored in the add-in file and automatically displayed in the add-in profile fields. The following fields are mandatory:

  • ID. Internal ID of the add-in object.
  • Version. Add-in version number. Used to determine whether add-in update is required.
  • Description. Full add-in description. Displayed during add-in installation.
  • Update from 1C:ITS portal. Select this check box to enable auto updating the add-in from the 1C:ITS website.

If any of the mandatory fields are not filled automatically, you need to fill them in manually; otherwise, you will not be able to save the profile.

Add-in statuses

Add-in status is set using the tumbler:

  • Used. This status is set by default during the add-in installation.
  • Disabled. To temporarily disable an add-in for any reason (for example, to make a test version of the add-in unavailable to users), select this tumbler position. This will not delete the add-in but the users will not be able to access it.

Updating add-ins from 1C:ITS portal

  • To update an add-in from the 1C:ITS portal, click . This option is available for add-ins originally downloaded from the 1C:ITS portal.

Updating add-ins

  • To update an add-in, click Update from file. A security warning will be displayed.
  • If the add-in is obtained from a reliable source and is guaranteed to contain no viruses or other harmful content, click Continue.
  • Specify a path to the add-in file and click Open.
  • The add-in profile information will be updated.

Saving add-ins to files

  • Add-ins can be saved to files for backup purposes.
  • Click Save as. Specify a file path on the computer. Click Save.
