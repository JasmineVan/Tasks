This list is intended for add-in management.

Add-ins are a category of program modules that support:

  • Peripheral equipment management
  • Data exchange with banks and document workflow
  • Clipboard operations, scanning
  • Barcode printing, text extraction, word declension, and more

Add-ins can be imported from files. Multiple versions of the same add-in can be installed at the same time, if necessary. This feature could be convenient, for example, in a situation when a variety of hardware and drivers is used across the client computers.

The administrator can control the list of add-ins that are used at workplaces. User are not allowed to install third-party add-ins withour explicit approval of the administrator.

All currently installed add-ins are displayed in the list. In column , add-ins receiving updates from the 1C:ITS portal are marked with flags.

Installing add-ins

  • Click Add from file.
  • A security warning will be displayed. If the add-in is obtained from a reliable source and is guaranteed to contain no viruses or other harmful content, click Continue.
  • Specify a path to the add-in file and click Open.
  • All information required for add-in installation is stored in the add-in file and automatically displayed in the add-in profile fields.

Updating add-ins from 1C:ITS portal

  • To update an add-in from the 1C:ITS portal, click .
  • This option is available for add-ins originally downloaded from the 1C:ITS portal. To receive add-in updates, you need to register at the 1C:ITS portal.

Updating add-ins from files

  • To update an add-in, click Update from file. A security warning will be displayed.
  • If the add-in is obtained from a reliable source and is guaranteed to contain no viruses or other harmful content, click Continue.
  • Specify a path to the add-in file and click Open.
  • The add-in profile information will be updated.

Saving add-ins to files

  • Add-ins can be saved to files for backup purposes.
  • Execute the More – Save as command, or right-click the add-in and click Save as.
  • Specify a path to the file on your computer. Click Save.

Filters in list

Use the Show field to filter the add-in list to display only these add-ins:

  • All
  • Used
  • Disabled
