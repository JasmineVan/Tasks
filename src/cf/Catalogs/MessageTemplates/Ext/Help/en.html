With this catalog, you can manage message templates. Templates can help you save time preparing messages that differ in minor details. As most emails and text messages are the same for all clients, message templates significantly facilitate message exchange.

To manage the templates, click Organizer – Notes, reminders, questionnaires, message templates, and then click Message templates.

List filters

  • The template list supports filters, which make it easier to manage.
  • With the Template for field you can filter the following templates:
    • Email and SMS
    • Email
    • SMS
  • With the Assignment field you can filter the following templates:
    • All
    • Common. Template for any message.
    • Or you can assign template for certain document type.

Creating new templates

  • Select Create template. Then, select the template type from the drop-down list:
    • Email
    • SMS
