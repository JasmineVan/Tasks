Use this form to view and edit user group properties.

To open it, in the Users list, click the button.

You can combine users into groups by a variety of parameters—for instance, by position title or according to the company structure. This facilitates access rights setup for multiple users with similar job duties.

Creating user groups

  • Enter group Description.
  • To make a new group subordinate to an existing group, in the Parent group field, select the parent group.
    • When editing the list of subordinate group members, keep in mind that these users belong to parent groups. Therefore you must monitor changes in user access rights when transferring users from group to group or excluding from a group.
  • On the Group members tab, you can add users to a group.
    • To add multiple users, click Select. Then select users in the list and add them to the group.
    • To add a single user to a group, click More—Add.
  • You can add a detailed description of the group on the Comment tab.

Specifying access rights of users groups

  • On the group navigation panel, click Access rights.
  • Click Add to group and select an access group.
  • If the explicit assignment of roles is available, you can assign the roles both to a single user and a user group.
