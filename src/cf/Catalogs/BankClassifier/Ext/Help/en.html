The All-Russian Bank Classifier (the bank classifier; the classifier) stores bank data, which is used to fill in bank details for such program objects as billing statements and documents.

The classifier can be filled in automatically using the All-Russian Bank Classifier. This option makes entering data significantly faster and easier. The user only needs to select the bank from the classifier, and all bank data will be imported and displayed instantly, which minimizes mistakes. More to this, the program automatically updates the classifier.

The banks that ceased activities are displayed as inactive.

To open it, execute the Banks command in the Catalog section.

The classifier displays:

  • Bank name, which is a full name of a bank;
  • Branch ID, which is a bank's branch ID;
  • Correspondent account, which is a correspondent account;
  • City, which is a city a bank is situated in;
  • An Address field contains an address of a bank.

Please note that addresses and phone numbers of banks are imported only from the 1C:ETS hard drive.

To enter additional information, execute the More - Edit the form command:

  • The program accounts for inactive banks and automatically marks the Bank ceased activities check box for such banks. This check box can also be marked manually.
  • Bank group. The default value is the state where a bank is located.
  • SWIFT Branch ID;
  • Addresses and Phone numbers of banks can be imported only from the 1C:ETS hard drive.

Grouping banks

  • By default, banks are grouped by states and the list displays only states. To view a list of banks, select a state.
  • You can add subgroups to the list. Click Create a group, enter its name, and drag any banks in it with a mouse.

Import bank classifier

  • Every time the program is launched, it checks the bank classifier for updates. If it has not been updated for 30 days or more, the program displays a notification offering to update the bank classifier. Follow the Update the bank classifier link.

  • Click the Import the classifier button to update bank data.

  • To have the classifier updated automatically on schedule, create the "Bank classifier import" scheduled job.
  • Not all information is imported to a bank card. The Address and Phone numbers fields remain empty if the classifier is imported from the 1C website.
  • When re-importing the classifier, only the altered data is being overwritten. Please note that if you alter a BIC number, the new bank record will be created. If you edit other fields, your changes will be overwritten (except for the Address and Phone numbers fields in case the classifier is re-imported from the 1C website).

Creating a bank

  • Click Create to create a bank. Fill in the required parameters.
