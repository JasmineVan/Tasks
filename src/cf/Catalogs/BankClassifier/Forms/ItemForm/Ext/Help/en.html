Bank properties:

  • A Name is a full name of a bank.
  • In group. The default value is the state where the bank is located. To create a new group, click Show all, select a group from the list, and click Create.
  • A Bank ceased activities check box indicates inactive banks. The program automatically marks the check box for such banks. This check box can also be marked manually.
  • A Branch ID field contains a bank's branch ID.
  • SWIFT BranchID.
  • A Correspondent account field contains a correspondent account.
  • TIN.
  • A City field indicates a city a bank is situated in.
  • An Address field contains an address of a bank.
  • A Phone number field, as well as addresses, can only be imported from the 1C:ETS hard drive or entered manually.
