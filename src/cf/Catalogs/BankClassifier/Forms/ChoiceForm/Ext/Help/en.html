This form allows to select bank data to be used for bank details forms in various program objects.

The list includes the following columns:

  • Name is a full name of a bank;
  • Branch ID, which is a bank's branch ID;
  • Correspondent account, which is a correspondent account;
  • City, which is a city a bank is situated in;
  • An Address field contains an address of a bank.

The banks which cease their activities are hidden by default. To display such banks in the list, execute the More - Show inactive banks command.

To update data on banks, execute the Import the classifier command.

Selecting a bank

  • Select a bank.
  • Click Select.
  • You can also double-click the item.
