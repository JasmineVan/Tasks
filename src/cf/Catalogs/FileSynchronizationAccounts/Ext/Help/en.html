The catalog is designed to store file synchronization accounts.

Entering new accounts

  • To enter a new account, click Create.
  • Enter Description.
  • In the Service field, select a cloud service to synchronize files of this account.
  • Specify Username and Password. This is your access credentials received during registration on the selected cloud service.
  • In the Root folder field, choose a folder to store all files for this account. You should create this folder on the cloud service.
  • In the File author, use radio buttons to specify a file author.
    • This account is used by default.
    • If only one user can edit files in the cloud, it is recommended that you specify this user as an author of new and edited files.
      • In this case, set the radio button to User , select a user type by clicking , and then click OK. After that, select a user from the list.

Checking accounts

  • Click Check settings in the account window to check whether account data is specified correctly. If the check passes, you can see information on free space in the cloud service and volume taken by files.
