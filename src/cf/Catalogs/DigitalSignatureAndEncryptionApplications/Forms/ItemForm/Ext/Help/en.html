Information about digital signature and encryption application.

To open this form, click Add in the Digital signature and encryption settings list on the Applications tab.

Filling application information

  • In the Presentation field, choose an application from the list. The frequently used applications are listed here.
      • All other fields will be filled automatically.
    • If you want to use an application not in the list, choose <Other application>.
      • Then, you will need to fill the other fields manually.
  • All fields are mandatory.
  • Application name - full name of the application
  • Application type - number referring to the type of the digital signature provider
  • Signature algorithm - selected from the list offered by the digital signature provider
  • Hashing algorithm - selected from the list offered by the digital signature provider
  • Encryption algorithm - selected from the list offered by the digital signature provider
