This form is used to configure the accounting checks.

To open the check form, go to the Accounting checks list.

All checks are enabled by default; some of the checks are preconfigured to run automatically in background.

Generally, there is no need to reconfigure the checks manually. The following check settings are available for manual modification:

Comment field contains additional information about the check.

Usage

  • If the check is not needed for the moment, clear the Usage check box to disable it.

Issue severity

  • Issue severity can take the following values:
    • Information
    • Warning
    • Error (default)
    • Useful tip

Auto execution

  • To have a check running automatically, select the Runs in background on schedule check box.
  • Use the radio buttons to select the auto check parameters:
    • On general schedule (default)
    • On individual schedule Selecting the individual schedule enables the Set up individual schedule hyperlink.

Running checks manually

  • A check can be run manually at any moment. To run a check, click Check.

Additional settings

  • On the More tab, you can specify the following check parameters:
    • Check start time. You can specify the check start time for documents and other timestamped objects. For example, this might be useful to avoid running checks for closed or archived time periods. By default, this field is left blank, meaning that all periods are checked.
    • Issue limit. The maximum number of issues that can be identified per check. For example, you can specify that a mandatory attribute filling check running for a long list will stop after identifying a set number of issues.

Reverting to default settings

  • To revert to the default settings, click More – Set standard settings.
