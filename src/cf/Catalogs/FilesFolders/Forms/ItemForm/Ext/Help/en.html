Viewing data on a files folder

Data on a folder

  • Fill in Description to a folder. This field is mandatory;
  • One folder can be a part of another folder. Select a folder of a higher hierarchy from the list in the In folder field;
  • The Full path field will be filled in automatically;
  • The Description field is used to enter and store detailed information about a folder;
  • In the Person responsible field, select a user from the list; this user will be responsible for the folder;
  • The Creation data field automatically stores date and time of a folder's creation.
  • You can select a Working directory for a folder to store this folder's files after editing them separately in the selected working directory.

Rights to access folders

  • Folders can be assigned access rights of different level. To start editing access rights for a folder, click Manage rights.

Quick access to files list

  • To go to the Files list, click the Open folders and files list.
