A folders list is hierarchical. It is used for selecting files and working with them.

All files are stored in a structure of folders that is convenient to keep. For instance,

  • according to your enterprise's organizational structure;
  • by a file type (such as Contracts, Orders, or Memorandums);
  • by authority level (Common or Confidential).

The list has predefined folders that cannot be deleted. For instance, File templates.

Managing authority levels for accessing folders

  • Folders can be attributed with access rights of different level. Click Manage rights in a list or a folder card.

Creating a new folder

  • In the Files list, right-click Create on the context menu to create a new folder.

Editing a folder

  • To edit a folder, select it and click Edit on the More actions menu. Only users with sufficient access rights can perform this action.

Select a folder

  • Select a folder and click Select. You can also double-click the item.
