Fill in the key operation profile to proceed with application performance tests.

Creating a new profile

  • To create a new profile, click the Create button in the profile list.
  • Fill in the Description field.
  • You can fill in the profile using the following ways:
    • Load an existing profile;
    • Fill in a profile;
    • Add key operations.

Loading an existing profile

  • Click the Load profile button. To use a previously saved profile or a profile from a file, use the related button.
  • Specify a path to the profile in a local directory. The specified profile will be loaded. The Description field will be filled in from the previously saved profile.

Filling in a profile

  • Click the Fill in button.
  • All key operations from the key operation list will be transferred automatically.
  • The Target time and Priority fields will be filled in with default values (1.00 for the target time and 5 for the priority).

Entering key operations

  • Click Add. Select a key operation from the list.
  • Specify the Target time (for example, 1.00) and Priority (for example, 5) fields.

Saving a profile

  • Click the Save profile button. Specify a path to save the file in a local directory.
