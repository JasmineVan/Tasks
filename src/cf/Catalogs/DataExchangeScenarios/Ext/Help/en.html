This list is used to store data synchronization scenarios.

You can access the list from Data synchronization section. You can also open the list by clicking the appropriate command in the More menu of the Data synchronization list.

The list includes the following columns:

  • - import (getting data);
  • - export (sending data);
  • Description - full scenario description;

Scenarios that run automatically on schedule are marked in bold in the list.

You can add extra fields, such as Data version, Comment, and more, via the More - Customize form command.

Setting up scenarios

  • Click Add and fill in the required fields.

    Data synchronization scenario setup procedure is performed in the following steps:

    • Set a data synchronization scenario schedule.
    • Select infobases participating in the data synchronization scenario.
    • For each infobase, select connection method and current action (sending or receiving data).
  • After the data synchronization scenario setup is complete, the data synchronization setup record in the list will be marked with . Data synchronization is performed on schedule, as a scheduled job.

Enabling data export and import

  • To manage data export and import, use the Enable/Disable dropdown menu.
  • If export or import is enabled, a check box will appear in the appropriate column.
  • The scenario can be used to export data or to import data separately, but it is not recommended to use this option. Full data synchronization must be performed in a single scenario.

Enabling scheduled execution

  • To Enable/Disable data synchronization schedule, click . If the schedule is enabled, the scenario is marked in bold in the list.
