Use this list to view metadata object IDs. The IDs are required for application activity and are not user-editable.

The list is populated automatically based on application metadata during the first application run and each subsequent application update. They are automatically updated during each application version update (including added, deleted, and renamed metadata objects).

It displays all application objects grouped by type. The Synonym object field is displayed by default. Use the More actions — Change form command to add the Full name and ID fields.
