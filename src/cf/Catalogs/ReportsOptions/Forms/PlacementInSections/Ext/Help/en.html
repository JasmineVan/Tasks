This object allows you to set up placing one or more report options in the application sections.

To open it, in the Reports list, click Place to sections.

  • The Report options field lists options selected for placing.
    • To add report options for placing, use Select button.
    • In the List of values, you can Add a single report option or click Select to open the report options list, select multiple report options from the list, and then click Select.
    • To confirm your choice and go back to placing report options, click OK. If the list does not suit your needs, click Cancel to go back to setting up the report options placing.
  • The Section, group column shows the sections and command interface groups tree.
    • Switching check boxes adds report options to sections and command interface groups or removes these options from sections and groups.
    • To cancel your choice and start placing again, use More - Clear check boxes command.
  • You can also set the report option Importance by clicking Select:
    • you can mark as Important and therefore highlight the report option in bold in the report panel;
    • you can place the report option to the report panel in See also group.
  • Click Place to confirm.
