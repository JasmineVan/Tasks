Use this object to store report options and their location settings in the command interface.

Report option information

  • Fill in the report option Name to be displayed in the application lists and report panel.
  • Fill in Description, which is detailed information on the report option. It is displayed as a tooltip in report panels and report options lists.
  • Select Available radio button to specify users that can see this report option:
    • Author only - the report option is visible only to the author you can Select from the user list. If necessary, you can specify another author for the report, for example, in case the person responsible for the report has changed.
    • All users - the report is available to all users.
      • Visible in report panels by default - select this check box to make the report visible to all users in the report panel.

Report option layout in panels

Report options placing can be changed by report option authors and administrators.

  • To place a report option to the application sections or groups, in the Section, group column select one or more check boxes. When placing to a section, the report option will be displayed on the top of the section report panel and will not be placed to groups.
  • In the Importance column, select a method to highlight the report option in a group:
    • Important - the report option will be marked in bold.
    • See also - the report option will be displayed in a separate group at the bottom of the report panel.

Creating reports

  • Click Open, specify the report parameters, and click Run report.

Reset settings

  • If the settings are incorrect and cause inconvenience when dealing with a report, you can reset the settings. Execute the following commands:
  • More - Reset placing settings - reset placing settings for this report option;
  • More - Reset user settings - reset user settings for this report option in report panels.
