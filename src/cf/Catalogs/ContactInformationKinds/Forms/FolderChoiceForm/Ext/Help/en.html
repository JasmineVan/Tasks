The list is used to select contact information kinds.

To open the list, click Select in the new contact information field input dialog.

Contact information kinds are predefined groups each of which matches lists or documents containing contact information. Items in each group define content of contact information of these application objects. Group components cannot be modified. A list of groups is a hierarchical tree.

Selecting contact information kind

  • Select a group, and then click Select.
