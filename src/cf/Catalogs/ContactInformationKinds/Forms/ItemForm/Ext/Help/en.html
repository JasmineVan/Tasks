Properties of contact information field.

The application is supplied with predefined contact information fields for some documents and lists. You can only change settings of predefined contact information fields, other attributes are filled in by default and cannot be edited.

Contact information types

The application allows you to store the following contact information types:

  • Address. Fields with this contact information type are used to store an address as an arbitrary string or as an address template. In this field, you can fill in an address using an input template by using an address classifier. If the address is filled in using the address classifier, the postal code of the address is populated automatically from the classifier once you finish entering the address. Predefined physical and legal addresses are used in document print forms.
  • Phone. A field with this contact information type is intended to store phone numbers as an arbitrary string or as phone number input template. Predefined phone is used in document print forms.
  • Email. A field with this contact information type is intended to store email addresses. The attribute is used to send emails from the application.
  • Web page. A field with this contact information type is intended to store web page URLs. Allows you to insert URLs to store them in the infobase and navigate.
  • Fax. A field with this contact information type is intended to store fax numbers as an arbitrary string or as fax number input template. It is used in document print forms.
  • Other. A field with this contact information type is intended to store an arbitrary text of contact information, for example, you can put information about how to get to the company. The field length limit is 1024 characters.
  • Skype. A field with this contact information type is intended to store the Skype username.

Adding a contact information field to list or document

Fill in the required information:

  • Select Contact information type from the list.
  • Enter Description to represent this contact information field in the list or document.
  • Enter Tooltip text to be displayed when the cursor is over the contact information field.
  • Select Group if it is not selected by default. The group corresponds to the document or the application list, to which a new contact information field is added.

Settings of contact information fields

  • Select the Mandatory check box to ensure that the contact information field is not left blank when entering documents or lists. You will not be able to save data if the field is blank.
  • The Edit in dialog only setting is used for the following contact information types: Address, Phone, and Fax. With this setting, you can enter new addresses, phone numbers, and faxes or edit the existing ones only using an input template or by selecting from the address classifier. If the check box is selected, you can enter an address in free format. You might need it if the address data is specified incorrectly in some documents but the conrrect data is required for printing. Such address will be marked by the application as entered in a free format and used with restrictions. Once you find out the right address, you can correct it using the template.
  • The Prohibit entering incorrect address setting is used for the following fields: Address, Phone, and Email address. With this setting, you can prohibit entering addresses, phone numbers, or email addresses that are filled in not according to the set rules.
  • Select the Allow multiple value entry to enable entering multiple values. For example, a company can have several emails or phone numbers.

International address format

  • To create an international address (for example, for foreign currency payments), select the International address format check box. In this case, the Additional address settings link is not available.

    Additional settings of the Address field

    By clicking the appropriate link, you can go to the additional address settings:

    • Select the Only national address check box to prohibit selecting any foreign countries during address input. Use this feature to facilitate information entry if your company employees use only national addresses. The following settings become available:
      • Prohibit entering incorrect address. If this check box is selected, you will not be able to save the address data until the address is validated. We recommend that you only use this option when input templates and address classifier are implemented. To enable this option, select the Edit in dialog box only check box. Address is checked against FIAS.
      • Specify Municipal territory classifier. Select this check box to automatically populate the Municipal territory classifier field. You can edit the field if necessary.
    • Include country in presentation. Select this check box if you want the country name to be displayed when automatically generating the full address in the Address field.
    • If the option of storing the address change history is enabled, the Store change history status is displayed in the field.

    Phone and Fax fields settings

    • To enable Phone with an extension, select the matching check box.
    • For some types of phones (for example, a mobile phone), there are no extensions.

    Email address field settings

    • For the fields of the Email address type, the entered email address data can be validated automatically. Select the Forbid invalid address entries check box to ensure that only valid email addresses are entered.

    Other field settings

    • With the radio button, you can specify Field kind in order to place the field in the object window more accurately:
      • Multiline wide enables multiple strings.
      • One-line wide. A long string.
      • One-line thin. If the Other contact information kind is used to enter a short text, the input field size can be reduced.
