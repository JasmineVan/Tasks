Use this list to manage external user groups.

External user groups can be convenient when a large number of external users work with the application:

  • To group large numbers of external users (this simplifies external user selection).
  • To quickly configure access rights settings for all external users in a group.

You can create groups as a flat list (all groups are independent), or as a hierarchical list. In a hierarchical list, members of a subgroup also belong to parent groups.

The predefined group All external users contains all external users.

You assign roles not only to users, but also to user groups.

Grouping external users

External user groups can based on their relation to application objects, for example:

  • Counterparties
    • Trusted partners
    • Major customers
    • Major suppliers
    • Poll responders
  • Individuals

This approach facilitates the setup of access rights.

Creating groups

  • Click Create and fill in the fields.
