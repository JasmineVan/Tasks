Use this list to map a user to an operating system user.

Then the user will be able to sign in to the application using their operating system credentials.

The list consists of two parts:

  • The Domains list displays the names of domains visible from the computer.
  • The Users list displays the users registered in the selected domain.

Selecting a user

  • Select a domain.
  • Select a user and then click Select, or double-click the user.
