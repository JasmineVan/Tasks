This form is used to view additional attribute and info sets.

In the form, you can see Description of the set and the catalog group this set relates to.

You can also see lists of additional attributes and information included in the set. The lists are read-only and cannot be changed in any way. To change list items, use Additional attributes or Additional information commands available in Administration - Common settings.
