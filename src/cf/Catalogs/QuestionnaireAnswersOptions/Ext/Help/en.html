Keep a list of options of responses to questions of types Select one response or several responses from the provided options.

There are questionnaire questions of type Select one response from the provided options or Select several responses from the provided options. In such cases, a respondent selects either one response or several responses from the list using radio buttons.

Specify possible options of responses to the question.

Enter an option of a response to the question

  • Enter one of the response options in the Description field. The field is required.
  • If the Requires open ended response check box is selected in an option of a response to a question of the Select several responses from the provided options type, the respondent will be prompted to give a detailed response in the text field next to the response option, hence commenting on their choice.
