It is used to enter an additional attribute or info value.

In the application, you can use additional attributes or info of several types. The data type is selected upon setting additional attributes or info. When you select the Additional value (hierarchy) data type, the application suggests that you enter values. For each additional attribute or info of this type, a specific value list is used.

Entering values

  • Specify Description of an additional attribute or info value.
  • Using the Included in group field, you can subordinate the new item to another one in the list. To create a new parent item, click Create or press F8. To specify an existing item, click Show all or press F4, and then select an item from the list.
  • If the Use weight coefficients for values check box is selected for an additional attribute or information, enter a numeric value in the Weight coefficient field.
    • This field can later be used for analysis in reports. You need to think through the usage logic if this weight coefficient yourself. For example, you can specify weight for the Reliability property value as follows: For values from one to three, specify weight equal to zero, and for values four and five, specify weight equal to one. This technique is useful for fast selection of reliable customers with positive weight coefficients.
