The list contains values of additional attributes and info.

In the application, you can use additional attributes or info of several types. The data type is selected upon setting additional attributes or info. When you select the Additional value (hierarchy) data type, the application suggests that you enter values into the list. For each additional attribute or info of this type, a specific value list is used.

The Additional value (hierarchy) data type is commonly used, information is entered by selecting values from a list, which speeds up data input process and reduces the number of mistakes. With this data type, you can describe various properties of application objects.

In the list, you can use values hierarchy. Items of such list have identical properties, but they can have subordinate items. The hierarchy can be very complex.

You can select any item in the hierarchy as a value, while in the grouped list of additional values, selecting a group description as a value is not permitted.

Selecting values

  • Select an item.
  • Click Select.
  • You can also double-click the item.

Entering values

  • To enter values, click Create. Fill in the required fields.
