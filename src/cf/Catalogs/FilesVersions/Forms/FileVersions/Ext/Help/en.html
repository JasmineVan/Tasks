The form is designed to store information on file versions.

To open the form, click File versions on the navigation panel of the file profile.

In the list, using the navigation panel buttons you can:

  • View. Open a file version by using an application for viewing files. You can also do it by double-clicking a file.
  • Open profile. View file version information.
  • Make active. Set the selected file version as the current one. The command is available only for files that are not used.
  • Show difference. Select two file versions and compare them. Version comparison is supported for the following file formats: HTML, TXT, RTF, DOC, DOCX, and ODT.
  • You can save a file version to your computer by clicking Save as in the More menu. Specify a path to the file on the computer to save the file version.
