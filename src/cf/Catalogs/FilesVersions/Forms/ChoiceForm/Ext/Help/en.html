This list is used to select a file version.

To open the list, select a Parent version in a file card.

The list includes the following columns:

  • Version number. The program numbers versions automatically when it is being changed. The latest version has a highest numeric value.
  • Author. A user who added the file version. The file type is displayed in this column as a picture.
  • Creation data. A date and time when the file version was added to the infobase.
  • Size (bytes). A file version size in bytes.

File version choice

  • Select a version and click Select. You can also double-click the item. You can select only one file version.
