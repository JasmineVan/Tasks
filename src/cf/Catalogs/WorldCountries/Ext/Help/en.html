Use this catalog to store information on world countries, customs unions, states, or territories. The catalog data is used, for example, when selecting a country for contact information address or a country of origin for goods.

To open it, click World countries in the Catalogs section. You can choose to store only the list of frequently used countries in this catalog.

You can fill it in manually or import data from the All-Russian Classifier of Countries of the World (ARCC). To open the classifier, click More actions.

The list displays the following information:

  • The Description and Full description fields store a short and full official description for a world country. Short country descriptions are used by default for country presentation or printout. If the full description is not in the list, it matches the short description of the country.
  • The Code field contains a 3-digit numeric code, which can be used to identify countries according to ARCC. For countries, customs unions, and other structures not directly related to the country classifier (such as European Union), we recommend that you enter "--" into this field.
  • Fields Alpha-2 code and Alpha-3 code contain two-digit (for alpha-2) and three-digit (for alpha-3) codes used to identify independent states, dependent territories, and major administrative areas within states in accordance with GOST 7.67 and ISO 3166
  • EEU member. If a country is an EEU member, the check box is selected in the matching column.

We recommend that you only add countries from the classifier to the catalog when necessary. If information on a country has been updated recently but the country classifier does not yet reflect this update, you can edit the country data directly in the catalog.

Adding country from the classifier

  • Click Add from classifier.

Entering manually

In specific cases, it may be necessary to enter a name not available in the country classifier (for example, a name for a local region or an international community, such as European Union).

  • We recommend that you refrain from manually entering country names, as manual input increases error risk.
  • To add a new country to the country list, click More actions – Add manually.
  • Fill in the required fields. Instead of numeric codes, the "--" value is proposed automatically, because this value is commonly used in print forms for countries not present in the classifier (for example, see Letter of the Ministry of Finance of the Russian Federation dated 10.02.12 No. 03-07-09/06).
