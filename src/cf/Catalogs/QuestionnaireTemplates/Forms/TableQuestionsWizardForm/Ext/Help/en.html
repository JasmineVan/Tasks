Use this window to enter and edit table questions of the questionnaire.

To open it, click Add table question in the questionnaire template.

Selecting a table question type

  • On the first page of the wizard, write the question Wording, for example, "Specify your blood relatives".
  • In the Tooltip display method field, specify how the tooltip will be shown in the questionnaire text:
    • If you specify Tooltip, the tooltip text will always be visible.
    • If you specify Question mark right, the tooltip text will be hidden. You can see it by clicking the question mark.
  • In the Tooltip field, write a brief explanation of the question so that the respondent can answer it easier.

  • Select a table question type using the radio button. The following types of table questions are available:

    • Composite table question
    • Table question with predefined answers in rows
    • Table question with predefined answers in columns
    • Table question with predefined answers in rows and columns.
  • On the right side of the window, you can see a picture illustrating an example of the selected table question type.
  • Click Next to select questions.
  • To work with questions, right click the commands in the More actions menu or the context menu.
  • If you prepare questions of the Select one response from the provided options or Select several responses from the provided options type in advance, it will be possible to use predefined responses in table questions. You will also be able to add response options quickly.

Composite table question

  • For a composite table question, specify question components. The specified questions will be displayed as headers of table columns.
  • The respondent answering the questionnaire questions will be able to add any number of rows to the table while responding.
  • By clicking Add, select the required number of questions from the list.
  • Click Next to view the resulting table.

Table question with predefined answers in rows

  • For this table question type, specify question components on the second page of the wizard.
  • Click Add and select at least two questions from the list.
  • The specified questions will be displayed as headers of table columns in the respondent questionnaire.
  • A number of rows in the table is fixed. The respondent will not be able to add or remove rows from the table while responding. The number of table rows will be equal to the number of predefined answers.
  • For the first question, on the third page of the wizard, specify predefined answers to be displayed in the rows of the first table column. Select Add. To add predefined answers, use one of the following methods:
    • Write response options for the first question manually.
    • If a question of the Select one response from the provided options or Select several responses from the provided options type is selected, you can click Fill in response options to quickly list possible response options for the first entered question.
  • Click Next to view the resulting table.

Table question with predefined answers in columns

  • This table question type is an option of the previous type.
  • For this table question type, specify question components. The specified questions will be displayed in rows of the first table column. Click Add and enter at least two questions.
  • On the third page of the wizard, the questionnaire author specifies predefined answers to the first question for the table columns. To do so, click Add. To add predefined answers, use one of the following methods:
    • Write response options for the first question manually.
    • If a question of the Select one response from the provided options or Select several responses from the provided options type is selected, you can click Fill in response options to quickly list all options by selecting them from the first entered question.
  • These predefined answers will be displayed as headers of table columns. The respondent will be unable to add or remove rows from the table when responding. The number of table rows will be equal to the number of questions reduced by one question (responses to the first question are column headers, and the first question in the table is not displayed).
  • Click Next to view the resulting table.

Table question with predefined answers in rows and columns

  • For this table question type, specify three questions. Click Add and select questions from the list.
  • On the third page of the wizard, specify predefined answers to the first two questions.
    • Predefined answers to the first question will be displayed as headers of table rows.
    • Predefined answers to the second question will be displayed as column headers.
  • The respondent will be required to specify answers to the third question in the table cells.
  • Click Add. To add predefined options of responses to the first two questions, use one of the following methods:
    • Write response options manually.
    • If questions of the Select one response or several responses from the provided options type were created in advance, you can click Fill in response options to quickly list all responses.
  • The respondent will be unable to add or remove rows from the table when responding. The number of table rows will be equal to the number of predefined answers to the first question.

    Viewing the resulting table question

    • On the last page of the wizard, you can see the table to be displayed in the respondent questionnaire. If you need to change the components of questions or their placement in the table, you can go back to the previous steps by clicking Back.
    • Click Finish to finish editing.
