Use this window to select the questionnaires templates.

It opens when you select a questionnaire template from the Survey purpose document.

The list includes the template Description and its Code, which is assigned automatically.

Selecting a questionnaire template

  • Select a questionnaire template and click Select.
  • You can also double-click the item.

Entering a questionnaire template

Select Create and fill in the required fields.

Template grouping

  • You can group templates. To do this, enter a number of groups.
  • Click Create group and type Description. You can subordinate a group to one of the existing groups. For this, specify a group from the list in the Parent field.
  • Drag templates to relevant groups or enter templates in groups.
