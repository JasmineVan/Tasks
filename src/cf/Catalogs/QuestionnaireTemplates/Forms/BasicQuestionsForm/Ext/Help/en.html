Use this window to enter a simple question in the questionnaire template.

To open it, click Add simple question in the questionnaire template.

Entering a simple question in the questionnaire template

  • In the Elementary question field, select a question from the previously prepared list of questions for questionnaire. Once the question is selected, its description appears in this field. You can create a question in the selection list if there is no question you need.
  • Once the questionnaire question is selected, the Wording field is automatically populated with a question wording text. Here you can change the question wording to be shown to respondents in questionnaires of this template.
  • If the Required check box is selected, the respondent will be unable to complete the questionnaire and answer this question.
  • In the Tooltip display method field, specify how the tooltip will be shown in the questionnaire text:
    • If you specify Tooltip, the tooltip text will always be visible.
    • If you specify Question mark right, the tooltip text will be hidden. You can see it by clicking the question mark.
  • In the Tooltip field, write a brief explanation of the question so that the respondent can answer it easier.
  • Write Notes about the question. This clarifying information will be displayed at the bottom of the questionnaire template.
