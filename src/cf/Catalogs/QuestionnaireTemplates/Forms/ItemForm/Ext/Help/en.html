Use this window to prepare a questionnaire template.

Questionnaires are prepared in two stages. First, you need to prepare questions to be included in a questionnaire. Then you need to create a questionnaire template using the prepared questions. Later, this template will be used for automatic generation of a questionnaire when the respondent starts filling it in.

Specify the template Description. The Description field is displayed in the template list.

A questionnaire template includes two tabs, namely Headers and Questions.

Headers

  • Specify Header to be displayed in the questionnaire when it is being filled in by a respondent.
  • A questionnaire can include several sections but two sections, Introduction and Conclusion, are required and present in any questionnaire.
  • Write Introduction. In this section, you can welcome the respondent and explain how to fill in the questionnaire. The text will be shown to the respondent on starting the questionnaire.
  • Write Conclusion. In this section, you can thank the respondent for filling in the questionnaire. The text will be shown to the respondent on completing the questionnaire.

Questions

  • Any questionnaire includes at least one section with questions.
  • Generate a list of questionnaire questions. The questionnaire list includes the following columns:
    • Question, where the description of sections and questionnaire questions is displayed.
    • Required, where Yes is displayed if the question has the Required check box selected. No is displayed if the check box is cleared.
    • Notes, where is displayed if additional information on the question is specified. To view the notes at the bottom of the list, select a question.
  • Click Add to add a new item to the questionnaire (you can also right-click the context menu). Select Item type:
    • Section
      • Specify Section name. In the Notes field, write clarifying information on the section to be displayed at the bottom of the template or the questionnaire. Add at least one section. To add a section, you can also press Ctrl+F9.
    • Simple question
    • Table question
    • Question with condition.
  • If a questionnaire includes a large numbers of questions, it is recommended that you group the questions by several sections so that it is easier for the respondent to use the questionnaire. It is recommended that you add not more than 10–15 questions to a section for the best respondent experience.
  • Click Move up and Move down to set the order of the questions in the section and the sections in the questionnaire. You can manage the order by dragging sections and questions. You can drag questions from one section to another.
  • If you delete a section, questions of this section are also deleted.

View questionnaire

  • When you select a table question in questionnaire template sections, you can see its appearance at the bottom of the template. To get a picture of how the respondent will see the whole questionnaire while responding, click Filling form.

Save questionnaire template

  • If you plan to continue working on the template, click Write and close.

Template edit prohibition

  • Once the list of questions in the questionnaire template is generated, you can prohibit its further editing. To do so, click End editing.
  • If at least one questionnaire is generated from a template, the questionnaire template automatically becomes read-only. For proper analysis of responses to questionnaires in reports, a template must remain unchanged because data is analyzed for all questionnaires populated from this template.

Simple question

  • Click Add - Simple question to pick a question from the Questions for questionnaire list for the template. You can also press Shift+F6.
  • Fill in the following fields to create Questionnaire template question:
    • In the Elementary question field, select a question from the previously prepared list of questionnaire questions. The question description is shown in this field. You can create a question in the selection list if there is no question you need.
    • The Wording field is populated automatically with a question wording text once a questionnaire question is selected. Here you can change the question wording to be shown to respondents in questionnaires of this template.
    • If the Required check box is selected, the respondent will be unable to complete the questionnaire and answer this question.
    • In the Notes field, write clarifying information about the question to be displayed at the bottom of the questionnaire.

Table question

  • It is easy to enter similar structured data in tables. For example, it is better to place information on family members in a table. The columns will include questions about family members, family members will be listed in the rows in order, and their characteristics will be placed in the table question order.
  • Click Add - Table question to open Table question wizard. Follow the wizard instructions to enter a table question.

Question with condition

  • There are cases where there is no point in asking the respondent a question if no condition is fulfilled. For example, there is no point in asking the respondent how ofter they use a car service or a car wash if the respondent does not have a car.
  • In this case, on creating a questionnaire template, you can create a question with condition, for example, about whether the respondent has a car and add subordinate questions, for example, about a car service and a car wash.
  • If the answer to the question with condition is positive, the respondent will be asked to answer subordinate questions as well. Otherwise, subordinate questions will be unavailable.
  • To create questions with condition, click Add - Question with condition or press Shift+F8. Only questions of the Boolean type (Yes/No) can be questions with condition.
  • You need to add one or several subordinate questions to each question with condition by clicking Add - Simple question or Add - Table question. Subordinate questions are displayed under their parent conditional question.
