Information on the access group profiles folder.

There can be many access group profiles with a similar purpose, so it is possible to create folders (groups) to simplify the search for the required profile and to understand the principles of their generation.

Creating access group profile folders

  • Enter Description;
  • You can make a folder subordinate to a Parent group (folder). Select a folder from the list and click Select. Or double-click the parent folder.
