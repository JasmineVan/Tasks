This list is used to select folders for access groups profiles.

To facilitate management of multiple access group profiles with similar rights restrictions and avoid confusion, you can consolidate similar access group profiles into folders.

The access group profiles can be organized by a variety of criteria.

Selecting an access group profile folder

  • Select a folder and click Select. Or you can left double-click to open a report option.

Creating access group profile folders

  • Click Create group and type in the folder name
