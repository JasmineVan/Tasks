The list is located on the application desktop and designed to work with notes.

You can keep notes in the application. Notes are arbitrary text information, for which it is impossible to find any suitable catalog or document, and no one else could have access to it. Notes are available only to their author.

Notes cannot contain information, for which the application provides the appropriate lists or documents.

Notes can be marked with color.

Some notes (the most important, not marked for deletion, that have the Show on desktop check box selected) can be displayed in this list.

A part of notes in the list is entered in relation to applied objects. Navigation panel of such objects has the My notes command to view the list of notes related to them. In lists such objects are marked with the sign. In this case, the Subject field of a note is populated with a reference to applied object.

Entering a note

  • Click Create. Fill in the required fields. An entered note will exist by itself, without connection to any subject.

View all notes

  • Execute the corresponding command to view the All notes list.
