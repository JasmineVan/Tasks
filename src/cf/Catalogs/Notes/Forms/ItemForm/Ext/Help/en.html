You can put to the note an arbitrary text information, for which it is impossible to find any suitable catalog or document, and no one else could have access to it.

Notes are available in the card of applied objects or on the desktop, that is why you can quickly and for the short time write some information and then move it to a more suitable catalog. For example, you can use a note as a temporary place to quickly record an address during a phone conversation with a client, in order to then move this address to the Counterparties catalog without being in a hurry.

You can also use notes to store information that stays relevant for a long period of time (for example, you can use notes as a notebook in which you can record a birthday of a coworker or a customer who is important for you, and you can enter a reminder about this note not to forget to wish them a happy birthday).

You can mark notes with color, put them on the desktop, and combine into groups.

Entering a note

  • When entering a note about any applied object, it is automatically populated with Subject - reference, by which you can open this object.
  • When you enter a note in the My Notes list on the application desktop or in the All Notes list, the Subject field is populated with "not specified".
  • Write a note text. There are standard formatting commands to edit a note text. Some of them that are frequently used, are displayed in the form of buttons for convenience. For the rest of them right-click to see the context menu or go to the More menu.
  • You can arrange a note text in two ways:
    • You can enter almost all text not highlighting the title, then almost all text will be displayed in note lists.
    • Or you can enter a short title in the first note row, and this text will be displayed in note lists. Then press Enter and enter the rest of the text.
  • The most important notes can be displayed in the My notes list on the application desktop. Select the Show on desktop check box. (It is selected by default).
  • In order to distinguish notes, you can mark them with icons of different colors. You can sort them by color in the All notes list. Select Color.
  • You can group notes, for example, by subjects. Select Notes group from the list by clicking Show all. You can enter a new group in the list of groups.
  • The bottom of the note displays the note recording status. If a note has not been saved yet, the field has the Not written value. The application populates the Written field with date and time of the last note saving.

Reminder

  • In order not to forget about the event mentioned in the note, click - Remind. Enter a reminder to a note.
