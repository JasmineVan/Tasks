The list is designed for operations with notes. The list contains all notes that you have created in the application.

To open it, click All notes.

You can keep notes in the application. Notes are arbitrary text information, for which it is impossible to find any suitable catalog or document, and no one else could have access to it. Notes are available only to their author.

Notes cannot contain information, for which the application provides the appropriate lists or documents.

You can mark notes with color, put them on the desktop, and combine into groups. Notes that are displayed on the desktop are marked in bold in the lists.

The list includes the following columns:

  • Subject - a note title.
  • Subject - a document or other applied object, about which a note was entered.
  • Change date - date and time of writing a note.

Entering a note

  • A note can be entered for a document or other applied object. Click - Create note in the applied object window. In this case the Subject field will be populated automatically.
  • Click Create in the My notes or All notes desktop lists. In this case, the application automatically selects "Not specified" in the Subject field.
  • Fill in the required fields.

Grouping notes

Notes can be grouped, for example, by subjects or topics.

  • Click Create group and fill in the fields.
  • When the required number of groups is created, drag notes to the corresponding groups.

Filters

  • If there are many notes in the list, you can filter them by fields Color and Subject.
  • Choose required values. You can use two filters simultaneously.

Viewing notes marked for deletion

  • Select the Show deleted check box to display notes marked for deletion. Cannot show notes that have been permanently deleted by the administrator.

Reminder

  • You can enter a reminder to a note. Click - Remind.
