All notes entered for this applied object are displayed in this list.

To open it, click My notes on the navigation panel from the applied object card.

Objects, to which notes are related, are marked with the sign in the lists.

Also you can place notes to the application desktop (there is a check box in the note window), enter them on the application desktop without connection to any subject, or use them as a notebook or instead of "stickers on the edge of the monitor".

You can also view all notes that you have entered by clicking the corresponding command.

Notes located on the application desktop are marked in bold in the list.

The list includes the following columns:

  • Subject - a note title
  • Author. This field is for administrator only, because all other users see only their own notes. Only administrator can view notes of all authors.

Entering a note about an applied object

  • Click - Create note in the object card.
  • Click Create in the list of notes related to the object.
  • Fill in the required fields.

Reminder

  • To enter a reminder about a note, click - Remind.

Viewing notes of other authors (for administrator only)

  • Select the Show other users' notes check box to view all notes on this object.

Viewing notes marked for deletion

  • Select the Show deleted check box to display notes marked for deletion. Notes that were deleted from the application by the administrator cannot be shown.
