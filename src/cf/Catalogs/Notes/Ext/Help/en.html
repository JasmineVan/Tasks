It is designed to manage notes.

The Notes list belongs to internal data for the administrator. It is useful for controlling all notes in the application entered by all its users.

To work with your notes and enter them, use the All notes or My notes lists on the desktop. Groups entered in the All notes list are also displayed in this list.

You can keep notes in the application. Notes are arbitrary text information, for which it is impossible to find any suitable catalog or document, and no one else could have access to it. Notes, as well as reminders, are good as digital replacement of "stickers on the edges of the monitor".

Using note, you can quickly write down some information for a short time, and then move it to a more suitable catalog (for example, you can use the note as a temporary place to quickly record the address during a telephone conversation with a client, in order to then slowly move this address to the Counterparties directory). You can also use notes to store information that is relevant for a long time (for example, as a notebook to which you can record the birthday of a client who is important for you).

You can mark notes with color, put them on the desktop, and combine into groups. Notes are available only to their author.

Notes can be entered both with reference to any application data (for example, you can create notes by a certain partner, goods movement document, and so on) and independently (for example, you can use them as a notebook).

Notes cannot contain information, for which the application provides the appropriate catalogs or documents. For example, it is not recommended to place into notes contact information, information about tasks, and so on.

The list includes the following columns:

  • Subject - a note title.
  • Author- a user who created the note.
  • Subject - full description of a subject, for which a note is created.
  • Content - a note text.
  • For desktop. The check box is selected if a note is on the desktop.
  • Mark - a note color.
  • Change date - date and time of writing a note.
  • Subject - content of the Subject reference in a note.

Entering a note

  • A note can be entered for a document or other applied object. Click - Create note. In this case the Subject field will be populated automatically.
  • In the My notes or All notes desktop lists click Create and fill the required fields. In this case, the application automatically selects "Not specified" in the Subject field.

Grouping notes

Notes can be grouped, for example, by subjects or topics.

  • Click Create group and fill in the required fields.
  • When the required number of groups is created, drag notes to the corresponding groups.

Reminder

  • You can enter a reminder to a note. Click - Remind.
