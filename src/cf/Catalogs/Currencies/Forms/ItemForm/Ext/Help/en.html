This form describes properties of a currency.

Currencies are widely used in any enterprise's activities: all documents related to monetary issues (billing statements, transfer orders, contracts, consignment notes) have a Currency field, explicitly or by default. The Currency field value can be filled in differently, depending on the enterprise's activity area: Russian ruble, another currency, or a conditional unit.

General currency information

When adding a currency from the national currency classifier, the following fields are filled in automatically:
  • Currency name. The full name of a currency.
  • Numeric code. A unique numeric code accepted as an international standard. It is used to identify currencies downloaded from the internet or from the 1C website.
  • Alphabetic code. An alphabetic code accepted as an international standard; serves as the main presentation of a currency.
  • If the radio button in the Currency exchange rate field is set to Import from the internet, the currency exchange rates are downloaded from the 1C website.

About Currency exchange rate radio button

  • You should only set the radio button to Import from the internet position when:
    • Adding a currency that supports rate import from the internet, in order to use it as a conditional unit and specify its exchange rate manually.
    • Checking whether a currency rate is up-to-date (because this check requires setting the radio button to "Import from the internet").
  • For a local currency, the Currency exchange rate radio button must be set to the Input manually position, because the exchange rate of any currency in relation to itself is always 1. For example, when working with a Russian enterprise, set the Currency exchange rate radio button for RUB to Input manually.

Interdependencies between currency rates

  • This feature can be useful when, for example, you need to calculate an internal currency rate for your company.
  • Set the Currency exchange rate radio button to the Calculate based on another exchange rate position.
  • Select a source currency from the list, or type it in. A source currency is used to calculate exchange rates for dependent currencies.
  • Specify the Increase by, % coefficient, which is a ratio between exchange rates of the source currency and the dependent currency.

Calculating exchange rates by formula

  • You can specify a currency whose rate depends on multiple currencies.
  • To do this, set the Currency exchange rate radio button to the Calculate by formula position.
  • Specify the exchange rate calculation method (for instance, (USD+EUR)/2).
  • Possible separators for fractional part of numbers in the formula are fullstop and comma, for example:
    • EUR*1.025
    • EUR*1,025
  • But if you have to use functions with two or more parameters, separate parameters with commas and fractional parts with fullstops, for example:
    • Round(EUR*1.025,2)
  • For more information about the rules for constructing such expressions, see the Expression language of data composition system help section.

Viewing exchange rates

  • To convert any currency to a local currency, use its exchange rate.
  • On the currency navigation panel, click Currency exchange rates. For your viewing convenience, the date interval is adjustable.

Entering exchange rates manually

  • Exchange rates for most currencies can be imported from the 1C website. If this option is not available for a currency, the Currency exchange rate radio button is automatically set to Input manually.
  • To specify an exchange rate manually:
    • On the currency navigation panel, click Currency exchange rates.
    • Click Create and type in an exchange rate for a date (the current date is set by default).
