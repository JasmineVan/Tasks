This list is used to manage currencies used in the application. The required currencies can be picked from the national currency classifier (recommended) or entered manually.

You can access the currency list from the Catalogs section.

The list includes the following columns:

  • Currency name. The full name of a currency.
  • Numeric code. A unique numeric code accepted as an international standard; used to identify currencies downloaded from the internet.
  • Alphabetic code. An alphabetic code accepted as an international standard; serves as the main presentation of a currency.
  • Rate. Exchange rate value for a day it is addressed.
  • Multipler. This parameter is set to 1 by default and not displayed. For currencies with very low exchange rates (accuracy is less than four digits in a fractional part), this field displays the amount of quoted units. For instance, the exchange rate of a Vietnamese đồng (VND) is RUB 16.1574 for VND 10,000. This field includes three subcolumns:
    • RUB. per. A logical connective.
    • Multiplier. An exchange rate multiplier.
    • Brief name. The alphabetic code of a currency.

Adding currencies from the national currency classifier

  • Click Create - From classifier (recommended for an accurate currency input).
  • The following fields of a new currency are filled in automatically from the classifier: Name, Alphabetic code, Numeric code, Currency rate radio button position (set to Import from the internet if it can be imported from the 1C website).

Adding currencies manually

  • Click Create - New currency or press Ins.
  • Fill in the currency information.

Importing exchange rates

  • To import currency exchange rates from the 1C website, click Import exchange rates.
  • To import exchange rates from the 1C website automatically, run the Import currency exchange rates scheduled task.
