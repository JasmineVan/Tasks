This form is designed to pick up files.

The list consists of two sections: to the left, you will see the list of folders, and to the right - the list of files in the selected folder. The list includes the following columns:

  • Description. Name and detailed description of the file.
  • Author. User who added the file to the application.
  • Date created. Date and time the file was entered to the application.

Select several files in the list, and then click Select.
