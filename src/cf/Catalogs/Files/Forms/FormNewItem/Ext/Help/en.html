The form is designed to select a method of creating a new file.

The form appears by pressing Insert, or right-clicking Add in the context menu from the file list, or attached files list.

Methods of creating new files

Select an option by using the radio button:

  • From template. A new file is created based on the selected template.
  • From file on disk. A new file is created from the file located on a hard or network drive.
  • From scanner. A new file is created based on a graphic file received from scanner. This option is available if the scanner is connected to your computer. To import files from scanner, you should configure scan settings.

Click Create to add a file.
