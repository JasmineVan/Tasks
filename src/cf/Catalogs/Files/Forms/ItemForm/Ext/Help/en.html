This form is used to view information on an attached file.

The form opens from the list of attached files by clicking More actions - Open card or by pressing F2.

File information

  • Description. Full name of the file. This field is filled in automatically when the file is attached and the file card is created. It can be changed manually.
  • Details. Specify detailed file information.
  • Author. User who added the file to the infobase.
  • Created. Date when the file is added to the infobase.
  • Edited by. User who last edited the file.
  • Modified. Date when the file is modified (local time).
  • Placement. Description and date of the document the file is attached to. You can view the document.
  • Size. File size.
  • Extension. File extension (format).

Editing attached files

  • In thin client, spreadsheet document files (.mxl format) are edited in embedded spreadsheet document editor. In web client, you need to download an application for editing spreadsheet documents. For more information about editing spreadsheet documents in web client, see this page.
  • To edit files in other formats, you need to have appropriate applications installed on your computer. For example, to edit files in Microsoft Word format, you need to install Microsoft Word.
  • To open a file in an external application, click View. The file opens for viewing or editing depending on if the file is locked for editing by another user.
  • To open a file in edit mode and lock it, click More actions - Edit. Locked files are unavailable for editing to other users. In the list, they are marked in green.
  • To unlock the file for editing and save it in the application, click More actions - Finish editing. After editing is finished, save the file, and then close the external application.
  • To unlock the file for editing without saving changes in the application, click More - Cancel editing. This command is useful to discard file editing results, for example, if some unwanted changes or mistakes are made in the file.

Saving files

  • To save a file to the specified directory on the hard drive or a network drive, click More actions - Save as.
  • To open the file directory in Windows Explorer, click More actions - Open file directory in Windows Explorer.

Updating files

  • To update an attached file from the original file stored on the hard drive (the source of the attached file), click More - Update from file on hard drive.

Digital signature and encryption

If you use a digital signature, the More actions - Digital signature and encryption drop-down list is available. In this drop-down list, you can use one of the following commands:

  • Sign to create a digital signature for a file. Before creating a digital signature, the application suggests that you select a digital signature certificate from the existing ones on your computer. You can also select a certificate upon customizing personal application settings. Digitally signed files are not available for editing. In lists, they are marked with icon .
  • Add digital signature from file to add a digital signature to a file. For example, this command is useful if the digital signature file was received later than the signed file.
  • Save with digital signature to save a digital signature to a file on the hard drive. For example, this command is useful if you need to deliver the signed file to another user.
  • Encrypt to encrypt a file. Unlike file signing procedure, you can use multiple certificates for file encryption, for example, if you need to encrypt a file for several users. In lists, encrypted files are marked with icon .

  • Decrypt to decrypt a file. To decrypt a file, you need to enter the private key password.

If you use a digital signature and encryption, the following tabs are available:

Digital signatures

This tab contains information on digital signature of the file:

  • Signature author. User who signed the file.
  • Signature date. Date when the file is signed.
  • Comment. Additional information on the digital signature.
  • Status, Check date. Signature genuineness and date it is checked.

Using the navigation panel buttons you can:

  • Sign a file.
  • Open the window with main properties of the digital signature certificate related to the selected file: issued by, issued to, purpose, and expiration date of the certificate based on which the signature is created.
  • Check whether the signed file was changed by someone using one or all digital signature certificates. To do this, select one or several digital signatures. If a digital signature of the stored file does not match a particular digital signature certificate, after checking, such digital signature is marked in red in the list. For such digital signature, you will see Incorrect in the Status column.
  • Check all digital signatures.
  • Save a digital signature to file on the hard drive.
  • Delete a digital signature from a file.
