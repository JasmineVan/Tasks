The form is designed to select a file template. Templates are selected from the File templates folder.

There is a predefined File templates folder in the application. This folder contains files used as a template to create new files. For example, you can put a prepared file with contract text to this folder, and then use it to create contracts for certain counterparties.

You can open the folder from the Files or Attached files lists by clicking Create - From template.

The list consists of two sections: on the left, there is a File templates folder; on the right, there is a list of file templates stored in this folder. The list includes the following columns:

  • . Shows whether the file has a digital signature, or it is encrypted.
  • Name. File format and name. File description is also provided.
  • Author. User who created the file.
  • Date created. Date and time the file is entered to the application.

Creating file from template

  • Select a template in the list, and then click Select. You can also double-click the item.

  • The application automatically creates a file profile and displays it on the screen. Fill in required fields, and then click Write and close.

Adding file templates

  • To add a new template to the File templates folder, click Add file, and then select one of the options by using radio buttons:
  • From file on disk. Choose a file on your computer.
  • From scanner. Follow the Scanning wizard steps.
  • The application automatically creates a file profile, and then displays it on the screen. Add a template description. The Folder field is filled in with File templates by default.

Viewing file information

  • To view the file profile, click More - Open card.
