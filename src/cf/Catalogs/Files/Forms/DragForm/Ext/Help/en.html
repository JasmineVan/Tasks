This form is designed to import files and folders to the file list by using the Drag-and-Drop method.

Importing files and folders by using Drag-and-Drop method

  • To import one or several files (folders), you can use the Drag-and-Drop method. Drag a file (folder) or several files (folders) from the Windows Explorer to a folder of the application files list.
  • If you drag a single file, it is immediately placed to the file list, and the file card is created.
  • If you drag several files or folders, fill in the required information:
    • Files dragged with a mouse are displayed as a list. Folders dragged with a mouse are displayed as a tree. You can expand each folder to see the files that it contains.
      • Select the required check boxes to specify files and folders to be imported to the application.
    • The application automatically fills in the Folder field with the folder in the file list to which the files will be imported. You can select another folder from the list if necessary.
    • To save disk space, select the Delete files after adding check box (it is cleared by default).
    • Add Description for created files.
    • Select the Store versions check box (recommended). It is selected by default.
    • You can Select encoding from the list for text files.
  • To confirm, click Import files.
