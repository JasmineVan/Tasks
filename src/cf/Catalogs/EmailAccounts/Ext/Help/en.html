This list is used for management of email accounts used to send or receive email messages.

To open the list, click the relevant link in Administration - Organizer.

The list includes the following columns:

  • Account Name
  • User name. Name of a message sender. It can differ from the user name registered in the application.
  • Account email address in the name@server.net format.

If you want to send and receive email messages automatically, you need to specify the password for this account. Moreover, you need to set up the system email account in order to ensure the application works correctly.

System email account

  • A predefined system email account is available in the application. It is used by default to send a variety of notification messages related to the application. For example, it can be used to notify the administrator that the application update procedure has completed successfully. Before you can start using this functionality, you need to set up the system email account.
  • To set up the system email account, click the relevant link in Administration - Organizer. You can also Change it in the list.

Creating email accounts

  • Before saving the email account settings, they will be validated (a test email message will be sent). If validation fails the account settings will not be saved.
  • Click Create and fill in the required fields:
    • Specify an account email address in the name@server.net format.
    • Enter the account Password.
    • Specify Account usage options by selecting check boxes:
      • For sending email.
      • For receiving email.
    • Specify Account is for:
      • All.
      • Me only.
    • Specify Sender name (your name, or name of your company.)
    • Use the radio button to select account creation method:
      • Detect connection parameters automatically (recommended.)
        • Click Create. Account will be set up and validated automatically. Follow the application instructions. However, if the automatic setting failed, you can try to set up the parameters manually.
      • Set up connection parameters manually.

    • Click Next to fill in other parameters. For many mail servers, recommended settings are filled in automatically. You will only need to validate and save the account settings.

Email account setting

  • Click Change and modify the settings accordingly.

Validating email accounts

  • Select an email account from list and click More - Validate account to validate the email account parameters. At this, the application sends a standard message and checks connection to incoming mail server (if the account is used for receiving email.)
  • If password is not specified in the account settings, you will be prompted to enter it. Enter password if necessary.
